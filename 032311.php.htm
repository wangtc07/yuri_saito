2016/0518Wed斉藤優里マスカラも買えたよ

 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
毎日、毎日ちょっずつだけ
化粧の仕方を変えて楽しんでます♡
目元にキラキラたっぷりつけて
漫画の主人公みたいにしてみたり
リップを薄めにつけて
ナチュラルな日も作ってみたり
私服だと雰囲気を変えられるから
楽しいんです\( ¨̮ )/♡
 
 
 
 
それとね最近気になってた
Lancomeのグロス、
ジューシーシェイカー買えたの( ´･ᴗ･` )
これから夏に向けて大活用です♡
 
 
 
 
つけてみたら、ブログに載せてみるね◎
もう少し待っててね〜
 
 
 
 
 
 
 
 
 
 
 
そして東京ビックサイトでの
握手会ありがとうごさいました
個握なのに「握手会デビュー！」
ですってお話して下さる方が
多くてとっても嬉しく感じてます
 
 
 
それにね、おに魂きっかけです！
っていう方も凄く多くて
本当におに魂ずっーーーとやらせて
頂いて本当に、有難いことなんだなって
ひしひし感じてます◎
 
 
 
最近は乃木坂のメンバーでも
沢山のメンバーがラジオパーソナリティ
として活躍していて嬉しい限りです
ラジオからの窓口がどんどん広がるね◎
でも、負けじと頑張らなきゃですね
 
 
 
 
 
 
それではお洋服のご紹介
 
 
 
前回に引き続きRosarymoonの
ワンピース首元が少しあいてて
一目惚れしちゃったんだ( ˙˘˙ )
 
普段花柄ってあんまり着ないんだけど
ついつい可愛くて...♡♡♡
 
 
 
 
 
次の握手会の洋服も
可愛いの着たいな(♡˙ᵕ˙♡)
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
先日のおに魂には
とにかく明るい安村さんが
来てくださいました♡♡♡
 
 
 
映像とか流れてるわけでは
ないのにつねに裸状態には
びっくりしました！！！
 
何よりもこの写真のインパクトが凄い!!!!
 
私が恥ずかしくなってしまいました...笑い
 
 
 
 
 
また絶対遊びにきて
面白いお話を聞かせて
もらいたいなって思いました
 
後、その時も裸がいいな...笑い
 
 
 
 
 
 
 
 
 
今日はじょしらくのチームくを
まいまいと一緒に見に行ってきたの
未来のお話で前回と少し繋がったり
あーあれ懐かしいなって思う
部分もあったりと楽しめました♡
 
 
ちーちゃんのとある所で
まいまいと爆笑してしまいました...笑い
 
 
じょしらく見た方は分かるかな？
 
 
 
 
 
 
 
 
のほほんとした可愛らしい
舞台他にはないんじゃないかな？♡
本当に楽しめました( ´･ᴗ･` )
 
 
 
 
 
 
 
 
 
 
 
それでは、またね
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
