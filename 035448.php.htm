2016/1114Mon斉藤優里珍しくね？♡

 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
最近時代の変化か年齢の変化か
どっちだか分からないけど
前髪をセンター分けにするのに
はまってるんです.....（笑）
 
 
 
 
実はこの１枚目の写真もセンター
分けだったりするんだよ♡♡♡♡
 
 
 
 
昨日の握手会でもさり気なくしてみたりと...
ちょいちょい、いつもの前髪と
センター分けを織り交ぜてってるんだ！！！
 
 
 
 
 
 
 
ということで前回の握手会も
沢山の方にお越し頂きました
ありがとうございます٩( 'ω' )و
色んなお話が出来てライブのことや
最近の悩み事 相談など私で良ければ
いつでも力になるのでお話
しにきてくださいね？？
 
 
 
 
 
 
1 2 3 4部と全て髪型を変更して
みたけど色んな人の好みが
わかったりして面白かったな〜
 
好みって人それぞれなんですね♡
 
 
 
 
 
 
3部ではめちゃめちゃ珍しく
久しぶりのツインテールしてみたり( ˊᵕˋ )
 
 
 
 
 
 
 
こんな感じで楽しく毎回やっているので
良ければ遊びに来てくださいね〜？
 
 
 
 
 
そして何より一番嬉しかったのが
「サヨナラの意味」がミリオン！！！
 
 
本当に沢山の方から「おめでとう！」
という言葉を頂きましたが、
私達が本当にありがとうという気持ちを
もっともっと皆さんに
伝えていかなきゃです(TT)
 
 
 
 
 
いつも側にいて私達を応援してくれて
そして支えてくれて
ありがとうございます♡
 
 
 
私も、もっと貢献出来るように
1つずつ目の前のことに
取り組んでいきたいです
 
本当にありがとう！！！！
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そしてこの前のラジオのゲストは
乃木坂ちゃんでーーーーす！！！
 
 
 
 
 
楽しい放送をありがとうʚ♡ɞ
やっぱり乃木坂からゲストに
来てくれるとアットホーム感が
更にまして心も体もうきうきしちゃうな〜
 
 
 
それでは今日のおに魂も
沢山の方に聴いてもらいたいな
私もいっぱい喋っちゃうよ♡♡
 
 
 
 
 
 
告知です！！！
