2017/0420Thu斉藤優里彼女感

 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
現在発売中のBLT graphは
チェックして頂けましたか？
なかなかそれ以来ブログ更新
出来ずにオフショットとかも
載せられずでごめんなちゃい(TT)
 
 
 
 
 
 
ですが、今日は沢山載せちゃいたいと思います♡
皆お待たせ？かな？（笑）
 
 
 
 
 
 
テーマはね「彼女感」ということで
春夏秋冬の季節を私と過ごせている
気分になれるように撮って頂きました
 
↑同棲しているらしいです♡（笑）


 
 
 
 
 
↑豆乳鍋美味しくてカメラのこと
忘れながらひたすら食べてました（笑）
あんまり豆乳鍋とか選ばなかったんだけど
次からチャレンジしようと思いました♡
 
 
 
 
 
↑ジェンガ！！！私強いよ本当に（笑）



 
 
↑夏休みで浮かれている（笑）
夏は浮かれるタイプ！！！！
ふぁみたんなら分かってるか（笑）
イベントは全力で楽しむタイプ！！！







 
という感じで本当に楽しくて
るんるんしながら撮影してました
その楽しさとか伝わってたらいいな？
 
 
 
 
 
 
 
 
 
 
 
そして最近友達と遊ぶといえば色んなカフェに
行くことになってます（笑）
この前も友達とカフェに
行ってきました♡♡♡♡♡♡♡
今回は「urth cafe｣
 
 
 
ずっとね苺のが飲みたくて飲みたくて
1回チャレンジしに行った時には
お店がcloseしてて諦めたんだけど
どうしても食べたくて再挑戦！！！
 
 
 
 
やっと念願のものを食べれました☆
「ストロベリーボンボン」
 
 
 
 
 
見た目よりめちゃくちゃボリューム感が
あって凄いお腹が満足になるの
でもね、この中に苺1パックぐらい
入ってるんじゃないかってレベルで
贅沢なんだ、絶対女の子は
好きだと思うんだよね♡♡♡♡
 
 
 
 
 
 
その後は「お花見」しに行って
苺のシャンパン\ ♪♪ /
 
 
 
 
この日は苺三昧の1日でした
そして同時にピンクに囲まれた
1日でもありました♡♡♡♡♡
 
 
あのねその日はお花見しに行くのもあって
ピンクの洋服を着ていったんだ
桜と写真撮っときに雰囲気から
可愛くなれるようにって
 
 
 
 
 
 
 
夜ご飯は私のお家で鍋パーティー！！！
BLT graphで食べた豆乳鍋が美味しくて
その味にしたかったけど元が売ってなくて
出来ませんでした(TT)
季節外れっちゃ外れだもんね.......
 
 
 
その日はとってもいい休日を過ごせました
 
 
 
高校からの付き合いの子もいて
乃木坂のライブみにきてくれたり
舞台にも来てくれたり本当に
良い友達を持ったなって実感しています♡
 
 
 
 
 
 
一緒に過ごしてくれてありがとう( ¨̮ )
 
 
 
 
 
 
 
 
それではまたブログ更新する時まで
まったね〜◎












たんたんたん ゆったんたん( •ㅈ• )
