2016/0219Fri斉藤優里急いでブログ書きま した

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
明日の4:00〜はじまる、46時間TV！
に備えなきゃなと思いつつ、
流石に長丁場なので...
全部見てもらいたいですが
お仕事などの都合上で
全部見れないって方はぜひ！
このコーナーを見てもらいたいです♡♡♡
 
 
出演するとこをふぁみたんに
見てもらいたくてブログを
焦りながら書いてます( ´›ω‹｀)
 
 
 
 
 
 
 
20日朝4:00〜からはじまり！
どんどんとコーナー名載せちゃいますね

 
オープンニング、ボウリング、乃木声リレー、クイズ王、2016ラッキー3福人は誰だ！、
THE歌王、乃木坂電視台、ライブ、のぎ天、
乃木のの、カウトダウン、
エンディングです♡♡♡♡
 
合計12コーナーになります
 
 
 
 
ふぁみたん良かったら見てね〜
あ！これ良かったらじゃなくて
絶対見てね〜だ！笑い
 
 
 
 
 
 
 
 
 
1つ言うと人狼ゲーム出たかった。泣
羨ましい羨ましい...٩( ö̆ ) وむ
凄い楽しそうじゃん、普段から楽屋で
やってるしさ何でよんで
くれなかったんだよ٩( ö̆ ) وむ
いいもん、裏で空いてるメンバー探して
やるもん絶対やるもん٩( ö̆ ) وむ
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そして昨日のおに魂は感謝祭ということで
ペイジさをとサリオちゃんのいる木曜日
にもお邪魔してきました\( ¨̮ )/♡
 
 
 
 
 
 
ビリビリマシーンが壊れてしまったので
急遽センブリ茶に罰ゲームが変更！
正直、私的には申し訳ないけど
ビリビリマシーンよりも
センブリ茶のが罰ゲームだけど
何倍もましです...( ´;o;` )
 
 
 
 
 
勿論、センブリ茶もめちゃ不味だけど
ビリビリなるくらないなら
センブリ茶なら何百杯でも
飲めるわ〜うりゃ〜って感じです◎
うりゃうりゃセンブリかかってこい！よし！
 
 
 
 
 
よし、それでは書きたい近々の情報は
書けたので明日に向けてお風呂に
ゆっかりつかっておやすみしたいと
思います(♡˙ᵕ˙♡)
 
 
 
 
 
 
そこまで急がなくても良いけど
こちらのチェックも絶対お願いします
 
2/25 ラーメンウォーカーTV2 
12:00〜伊藤かりん
12:15〜斉藤優里
12:30〜新内眞衣
 
 
 
ラーメン実は4回目の出演になります
うわーいめっちゃ嬉しい(ˊᵕˋ)
乃木坂ちゃんの中で最多出場
狙ってるのでこれからもまた、
たなたんやまるたんスタッフさんに
お会い出来たらいいな〜♡♡♡
ちなみに、最多はかりんちゃんらしい。む
ライバルだ！む←勝手にライバル。
 
 
 
 
 
お風呂までpinkにしちゃいました◎
やっぱりpinkが大好き♡
明日も頑張ろうーせーのっで！おー！











たんたんたん ゆったんたん( •ㅈ• )
