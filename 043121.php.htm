2018/0201Thu斉藤優里love

 
 
 
 
 
そうです！！そうなんです！！
この写真を見れば皆ピンっと来たはず（笑）
TWICEさんのshowcaseに行ってきました♡♡
大好きなまちゅと♡♡♡
 
 
 
 
 
その前にご飯を食べようと思い
何か食べたいものあるか聞いたら
返ってきた答えが「スンドュブ！」（笑）
「韓国料理が食べたいな〜」とか
ご飯の系統を答えてくれるのを期待してたら
まさかの個人料理名で来たから
思わず笑っちゃた（笑）
 
 
前もね、まちゅとご飯行った時に
「スンドュブ！！」って答えられたから
きっとあの人毎日食べたいんだと思う（笑）
 
 
 
 
で、韓国料理屋さんついて食べたいもの
注文したんだけど...ここらがまたおかしい（笑）
 
 
・サムギョプサル2人前
・ヤムニョムチキン
・スンドュブ
・チヂミ
 
 
っていう2人じゃ食べ切れないほどの量を
注文しました（笑）
結局、ヤムニョムチキンはかなり
量があったからお持ち帰りしました（笑）
 
 
 
 
 
 
 
そんな楽しいご飯を終えていざ！
showcaseへ♡♡♡
 
 
 
 
お洋服もグッズのマンツーマンを
お揃いで着てCANDY BONG持って
ずっとノリノリでした( ˙O˙ )
 
 
 
 
何より、MCの時に皆が一生懸命だけど
たどたどしく日本語を喋ってくれるのが
とっても嬉しかったです
その姿がまた可愛いの〜
MC中とかは2人でずっと「可愛い〜」
 
しか言ってなかった気がする（笑）
 
 
私個人的に、聴けると思ってなかった
曲が何曲かあったのでそれがとっても
嬉しかったな〜( ˙O˙ )
 
※今日も公演があるので書いちゃうとネタバレに
なっちゃうから内緒にします（笑）







本当に昨日は幸せな時間を過ごせました
可愛くて可愛くて、沢山パワーを
貰いましたこれからもずっと応援します♡
今日も公演あると思うので
頑張ってください☆☆☆
 
 
そして、何より行かれる方は楽しんでください☆
感想とかONCEの方とお喋り出来たらいいな
 
最後になりましたがジヒョさん
お誕生日おめでとうございます♡♡♡
素敵な1年にしてください！！！！！
 
 
 
 
 
ずっとずっと大好き〜♡♡♡♡
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
