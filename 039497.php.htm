2017/0628Wed斉藤優里にゃんにゃん

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
今日は色んなことがありましたね
１つ１つ書いていくね？
まず１つ目はじゃらんのCM発表会に
出席してきました♡♡♡♡♡♡♡♡
 
 
 
 
 
 
実際に箱根の旅館を借りて撮影したり
したんですよ( ´･ᴗ･` )
実は私もはじめて出来上がったCMを
記者会見で見たのですがとっても可愛い
仕上がりになっていて本当にお気に入りです
 
なんなら、今までのCMの中で
1番可愛いんじゃないかって
皆のこと大好きだから思っちゃう♡
 
 
 
 
 
私は、にゃらんと一緒のシーンが
なかったから少しだけ心残り（笑）
また今度機会があったら一緒に撮影
したいななんて思います
 
 
 
本当に沢山のシーンやカットがぎっしり
詰まったCMです、これを見て今年の夏は
じゃらんと私達と一緒に夏を満喫して下さいね！
 
 
 
 
 
 
7/3(月)〜全国放送です、よろしくね？♡
 
 
 
 
 
↓休憩中には皆で寝転がってみたり（笑）
色合いもふいに撮ったのにめちゃめちゃ
綺麗になってて( ˙˘˙ )最高
 
 
 
 
 
 
 
 
そして先日インフルエンサーの
全握がラストでした！！！！！！
しかもラストのペアはてぃーちゃん♡
 
 
 
 
久しぶりのてぃーちゃん
だったのでとっても嬉しかったし
なんならめちゃめちゃはじけた（笑）
だってさ、仲良しのスイカメンバーの
てぃーちゃんだよやばいだろう♡♡♡
 
 
はじけたっていう言い方はあれだけど
凄いテンション上がっちゃて
沢山の方に喋りまくっちゃいました！！
話したい会話が出来なかった方
もし居たらごめんね（笑）
 
 
 
 
それでは、またてぃーちゃんとなれること
楽しみにしてます( ´›ω‹｀)
ひなめろの皆さんもわちゃわちゃ
してくれて本当にありがとう
 
 
 
 
 
 
 
そしてそんな最後のお話は「テレ東音楽祭」に
インフルエンサーとスカイダイビングで
出演してきました〜！！！！！
 
インフルエンサーはヒム子さんと一緒に♡
ヒム子さんはTVで見てたことが
あったのでまさかのコラボが
出来るとは思ってなかったですし
凄い振りをしっかり覚えて下さって
本当に忙しい中ありがとうございました！！！
 
 
 
 
私達も覚えるの大変だったから
絶対やばいと思うの(TT)
筋肉痛になってないかな？
 
 
 
 
 
ヒム子さんのポーズ、ななとまいやんが
一緒にやるとこ本気で可愛かった（笑）
なんなら一緒にやりたかった（笑）
 
 
 
バナナマンさん本当に大好き♡♡♡







ブログ最後まで読んでくれて
本当にありがとう♡♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
