2016/1106Sun斉藤優里「red♡」がテーマ

 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
昨日は富士急で乃木の「の」の
公開収録そして今日は名古屋での
握手会ということで最近は
ファンの皆さんとの交流が
とっても多くてうきうきです♡♡
 
 
 
 
 
 
富士急イベントでは新曲のブランコを
フルバージョンで披露させてもらったり、
久しぶりの13金を歌わせてもらったりと
楽しいセットリストになっていました◎
 
 
 
イベントスペースの外でも
見て下さった方が沢山いて
何回かやらせて頂いてますが、
回数を重ねるごとに興味をもって
立ち止まって下さる方もいらっしゃて
嬉しい限りです！！！！！！！！
 
 
 
 
 
 
 
 
そんな今日はイベントの感想を
直接伝えて下さる方も多かった
握手会( ´･ᴗ･` )
 
 
 
3部4部
 
 
 
 
 
サイドがリボンになっているので
横から見ると少しだけ腕が、
見えるようなデザインになっています
 
 
 
 
 
今日のテーマは「red」と
決めていたので実は5部も赤いお洋服！
 
でも時間がなくて写真が
撮れなかったの(´·_·｀)
ごめんちゃい〜
 
 
 
 
 
 
 
 
今日も皆さんとわちゃわちゃしたり
素敵なお話が出来て楽しい１日に
なりました◎
後ね、全完売おめでとうって
言ってくださる方も多くて
本当に親身に一緒に考えて
くれてるんだな〜って実感しちゃって
本当にありがたかったです！！
 
 
 
 
 
 
そして個握の服装、他の日も
着る衣装決めたので
良ければ参考にしてね？
 
12/18...クリスマスコスプレ
3/4........ ポリスのコスプレ
 
 
 
 
 
 
 
 
 
ブログにこの前書いたんだけど絶賛ブームの
ウォーキングデッドはシーズン7の2まで
見終えましたよ◎
 
衝撃的なことがありすぎて
ショックでした( тт )
 
 
 
 
 
でもこれ以上書いちゃうと見てない方の
ネタバレになっちゃうので内緒
ということで書きません（笑）
 
ただ...ただ...悲しいです（TT）
 
 
 
 
 
 
 
 
私ね世の中がウォーカーまみれになっても
絶対生き残れると思うの
ある程度、学んだからね（笑）
 
もし、世界がウォーカーだらけに
なったら私に任して！！！（笑）（笑）
 
 
 
 
 
 
 
 
 
それでは、ちょっと短いですが
この辺で終わりたいなと思います\ ♪♪ /
またブログ書くから待っててね？
 
 
 
♡明日のおに魂も20:00〜
よろしくお願いします！
ゲストは小百合と愛未が
来てくれるよ楽しい１日に
なりそうだ〜( ˙˘˙ )
 
 
 
 
 
 
 
今日もTWICE聴きながら
ブログ書いてました（笑）
はぁ〜ツウィちゃん♡♡♡//
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
