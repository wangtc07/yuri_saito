2016/0310Thu斉藤優里お部屋のインテリアも

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
ふぁみたんはこの前のGuam
ぶろぐ見てくれて、沢山の方が
「行きたいよ♡」って言ってくださって
なんだか書いて良かったなって
思いました〜\( ¨̮ )/♡
 
 
 
コメントして下さった
皆さんありがとうございます
 
 
 
 
それとね、もう一つコメントに
多かったのが「ゆったんとGuam行きたい( ´･ᴗ･` )」
って書いてくれたことがはぴはぴです◎
私も、もう1回足を運びたい
海外ランキングに入ってくるので
その時にふぁみたんと
会えたらいいな♡♡♡にゃんちゃんなの。








今日はそんなGuamの写真をまた
載せようかと思ったんですが、
それよりも3/9発売のBOMBさんで
私とちーちゃんのWサイトウの
グラビアが掲載されています(ˊᵕˋ)
 
 
 
 
ちーちゃんと2人ではお仕事自体が
はじめてなんじゃないかな？って
思ったけど、
とっても楽しい撮影になりました!!
 
 
オフショットはこんな感じ◎

 
 
衣装も凄く可愛くて大好き♡♡♡





雑誌には残念ながら掲載
されなかったけど途中で
トランプをしているシーンも
あったんだよ( ´›ω‹｀)
 
 
 
ババ抜きしてたんだけど
残り2人合わせて3枚になっても
全然カードが揃わないから
確認してみたら、なんと
トランプ自体が揃ってなかったんです٩( ö̆ ) و
 
なんてこった、パンナコッタ
 
 
 
 
 
撮影とかって撮ってもらえれば
もらえるほどもっと色んな作品を
作ってもらいたいな〜って
欲が出てきちゃいます( ´;o;` )
 
 
 
 
 
 
 
また、よばれるように
頑張らなきゃですね！
 
 
 
 
 
 
 
 
最近、撮影終了後にお花を頂いたので
お部屋に飾ってみました( ´･ᴗ･` )
私の家には花瓶がなかったので
メイソンジャーを代用してみたのですが
結構、可愛くてお気に入りです♡♡♡
 
 
お花癒される〜♡♡♡
花瓶よりメイソンジャーのが
おしゃれ感がグッ！と出る気が
するのは気のせいかな？






明日、急遽になりましたが
柏店のオートバックスさんに
真洋と2人でお邪魔したいと
思いますのでお時間ある方はぜひ...


せっかくなので、私も
沢山の方とわいわいしたいので
待ってますよ？
ねーねー、にゃんちゃんなの？

 
それでは今日からまた冷え込むらしいので
あたたかくして寝てくださいね？
モコモコに包まれるの〜
 
 



あ！北野曰く、雪降る予想らしいです〜













たんたんたん ゆったんたん( •ㅈ• )
