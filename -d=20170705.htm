2017/0705Wed斉藤優里誰にも、とめられなーい☆☆☆

 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
7月1、2日からはじまりました
真夏の全国ツアー2017
神宮からというのは今までなかった
初の試み、そしてもう一つ期生ライブ
というのもはじめてでした( ´･ᴗ･` )
 
 
 
 
overtureが鳴ってるのにもかからわず
裏でじっとしてモニターを見てるなんて
今までなかった経験、又見ていて何だか
そわそわしてしまう気持ち色んな感情が
混ざりあってたの☆☆☆
 
 
不安だったのは頭から出演出来ない
ということでライブっていう気持ちが
ふわふわしたままステージに立ってしまう
んじゃないかってことでした
だって、そんなのはじめてだから
はじめてって誰でも不安になることよね...
 
 
 
 
いざ、はじまってみるとそんな不安は
1mmも感じませんでした（笑）
皆さんの暖かいコールやタオルの量みたら
もう凄く凄く嬉しくて〜
やっぱりふぁみたんは
いつでも傍にいてくれる
素敵な存在です♡♡♡♡
 
 
 
 
そしてまだまだステージに立ってから
浅いのにも関わらず堂々とパフォーマンス
していた3期生は凄くかっこよかったです！
私がまだ入って間もない時には
そんなちゃんと出来なかったですし
ましてや表情なんてかたかったりして...
 
裏で見てた時大絶賛だったんだよ( ˊᵕˋ )
 
 
 
 
2期生も期生の中では最小人数にも
関わらず凄く迫力のあるパフォーマンスでした
特に「きっかけ」を歌ってる姿は本当に
かっこよくて目が離せませんでした！！！！
 
これからもよろしくね？
 
 
 
 
 
今回の期生ライブという形式
どうなるかドキドキわくわく
でしたがそれぞれの魅力がたっぷり
出せたセットリストではなかったでしょうか？
来てくれる方が楽しいと思ってくれる
そんなライブを私達はこれからも
していきたいです( ´･ᴗ･` )
 
 
 
 
 
 
 
 
 
 
 
2日目にはテレ東音楽祭でコラボした
ヒム子さん！設楽さん！も遊びに
来て下さり本当に良い1日になりましたね
 
 
イヤモニから「キャーー」って声が
聞こえた時最悪の事態を
想定してしまいました（笑）
 
したらまさかのヒム子さん（笑）
 
 
 
 
 
いやーもう本当にびっくり（笑）
わざわざ忙しいのに駆けつけて下さいました
本当に嬉しいの♡♡♡♡♡♡♡♡♡♡♡♡♡
今度はね明るい曲でコラボしたいの
絶対ヒム子さんなら似合うと思うの（笑）
 
 
↑※もう次コラボすること勝手に決めちゃってるけど（笑）




真夏の全国ツアー2017 finalも決まり
アンダラ アンダーアルバムも発表に
なりますます盛り上がっていきそうな
乃木坂ちゃんに注目しててね？









*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
そして先日放送された乃木中の中で
お弁当GPという企画がありました
見事、その中で優勝を頂いちゃいました☆
 
 
めちゃめちゃ嬉しかった
頑張って時間をかけて色んなもの作って
良かったな( ´･ᴗ･` )




↓私のお弁当はこんな感じでした（笑）
 
 
 
シェアして食べたいから沢山作ったし
色んなおかずもいれたんだ！！！！
あんまり自炊してもブログには載せないし
モバメばっかに載せちゃうんだ（笑）
これからもお料理頑張ろう☆☆☆☆







仕事前にパイレーツ・オブ・カリビアン
見てきたよ( ˙O˙ )
パイレーツならターナー君推し♡♡♡♡
ネタバレしちゃだめだからあれだけど
衝撃の展開でしたびっくり（笑）
ふぁみたんにも絶対見てもらいたいな
お話しーたーいーのー！！！！！！！
 
 
 
シーが11日からパイレーツのショーとか
やるから絶対見に行かなきゃって
意気込んでるよ！！！！！！！
かりんちゃんと絶対抽選に
心折れないようにって！！！（笑）
 
 
 
 
 
 
それじゃあ、また更新する時まで
ばいばい今日も元気( 　˙³˙　 )
だって一日の中で楽しいこと1つは
見つけるようにしてるんだ！！！！
メンバーとお喋り出来たらもう楽しい
色んなこと話すのが好き！！！！！
ちなみに昨日は怖い話してた（笑）
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
