2018/0123Tue斉藤優里ひつまぶし

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
皆さん昨日の大雪大丈夫でしたか？
テレビを見ていたらだいぶ、
交通網が乱れていたのできっと
会社とか早く終わっても帰宅するのに
時間がかかるって方が多かったですよね？
 
 
 
 
お疲れ様でした(TT)
 
 
 
 
そして雪が降ってからも今日とか
路面凍結していたり雪が積もりすぎて
段差が見えなかったりと思わぬ所が
危険だったりするから皆さん気をつけてね
くれぐれも、怪我したりとかしないでね
 
 
 
 
 
 
 
握手会で名古屋に行ってたんだけど
嬉しいことがあったんだ〜
あのね好きな食べ物ランキング3位
には絶対入ってくる「鰻」をやっと
名古屋で食べることが出来ました♡♡♡
 
 
 
※名古屋名物 ひつまぶし♡
本当に大好きでランチでも鰻食べるの
必ず自分で自分のことお祝いする時も鰻（笑）






マネージャーさんに何食べてもいいよ〜
って言われていたので一番高い鰻にしました（笑）
2尾入ってたんだよ、たまらんっ！！！
 
その他にも、うまき頼んだり
土瓶蒸しも本当は食べたかったんだけど
お腹いっぱいになりすぎて
食べれませんでした（笑）
 
 
今度行った時は食べなきゃ♡♡♡♡♡
 
 
 
 
 
 
 
皆は名古屋で何か食べた？（笑）


 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
先日のTHE 魂のゲストは
お会いするのは2回目の「シソンヌ」さん
 
 
 
若月と舞台を共演していた部分もあり
とってもお話がしやすかったです☆
またぜひ遊びに来てください！！！
 
 
 
 
そして昨日のゲストさんは
「久住小春」さん♡♡♡♡
 
 
 
大雪の中来て下さり本当に
ありがとうございました
久住さん本当に可愛くて更に
ストイックな部分も知れて
びっくりしました( ´･ᴗ･` )
 
舞台、応援しています☆☆☆
 
 
 
 
 
 
 
名古屋握手終わりのらじらー！も
皆さん疲れてるのに聞いてくれたりして
ありがとうございます☆
私は喋るの好きだから凄く楽しめました
今度はスタジオに行ってお話したいな〜
しゃゆ 中田さん 藤森さん
ありがとうございました(ˊᵕˋ)
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんん( •ㅈ• )
