2016/0330Wed斉藤優里てんちょおおおおお おおおーーーー

 
 
 
 
 
 
 
 
 
 
ハルジオンが咲く頃が発売にあって
だんだんと恒例化？に
なってきた個人PV\( ¨̮ )/♡
 
 
 
 
 
 
 
 
 
発売前には予告編動画がYouTubeに
上がってたり期待感を煽る動画
ばっかりでしたね♡♡♡♡♡♡
私は今回、「斉ボーグ優里」という
タイトルそのままの片腕が
マシンガンになって敵とバトルするという...
 
 
 
 
 
妄想の激しい女の子の役でした...
世間から言われる
中二病みたいな役でした＼(^ω^)／
 
 
 
 
 
 
撮影前は凄い恥ずかしかったけど
振り切れば振り切るほど
楽しかったです＼(^ω^)／
 
 
だって22歳にもなって
架空の敵と戦うと思ってなかったし
それに、
「バーン！バーン！ヒュイーーーーーーン」
なんて叫ぶと思いませんでした＼(^ω^)／
 
 
 
 
 
 
 
これって個人PVだからこそ
遊べるんですよね〜
楽しかったな、ZUMIさん×荒船さん
とっても仲良くなれた気がしました！
ZUMIさんがペア個人PVで
みり愛と絢音ちゃんのを撮っていて
どんな方なのか撮影前にみり愛に
聞いてみたら、
 
 
 
「んー優里さんなら絶対
友達になれるタイプだとと思う＼(^ω^)／」
 
 
 
と言われたのが印象的...
友達になれるタイプ...
 
 
 
 
 
 
 
 
 
 
友達にはなれませんでしたが 笑い
和気あいあいとした現場で
とってもお仕事しやすかったです♡
ZUMIさん荒船さん
ありがとうございました
 
 
 
 
↑顔にすすつけたり
2月なのにも関わらず河川敷で
ノースリーブショートパンツになったり...
今思い出しただけでも寒いいいいいいい






 
 
 
受け身をとったりする撮影では
下にマットをひいて本格的に
アクション指導の方におそわり
転がったりしましたよ＼(^ω^)／
 
 
 
 
 
じっーーーーーーーと座ったり
室内にいるよりも体を動かしたり
走ってたりするほうが好きだから
あ！でもゲームも好きだな...
 
 
 
 
あれ？インドアなんだかアウトドア
なんだかよく分からなくなっちゃたけど
また撮影できたらいいな♡
 
 
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そして先日おに魂のゲストに
原駅ステージA ＆ ふわふわ の
最年長コンビが来てくださいました
 
 
最年長コンビといっても
原駅ステージAの馬場夏美さんは18歳
ふらふわの赤坂星南さんは20歳
 
 
 
原駅ステージA & ふわふわが
それほどフレッシュってことが
凄い伝わってきました♡♡♡♡
 
チームが全部で4チームあって
全部サイトで見させて頂いたのですが
どのチームが個性が光っていて
素敵でした( ´･ᴗ･` )
 
 
 
 
 
 
 
むしろむしろ可愛くて凄い
お友達になりたかったな...
またぜひ今度は全チーム
全員で来てほいな〜♡笑い
わちゃわちゃしたラジオも
楽しいよね？
 
 
ん？おに魂が異様に
わちゃわちゃしてるのか...
 
 
 
 
 
 
それではまた来週のおに魂が楽しみだ
わいわいわちゃわちゃ
がやがやどんどんぱちぱち
ぴかぴかぴか にゃんちゃんなの。
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2016/0323Wed斉藤優里濃厚なチョコが食べ たい

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
永島の卒業ライブが終わって
しまいましたね...
凄い沢山泣いた2日間だったけど
こんなに気持ちがすっきりして
見送れたライブもはじめてだと思います。
 
 
 
何だろう、本当にライブでは
自分では上手く行かなかった
部分も正直ありました。
だけど、終わりが良ければすべて良し！
って言葉があるように永島の言葉や
ファンの皆さんの愛情や雰囲気
全てがあたたかくて気持ちよくて...
本当に達成感とすっきり感のある
まさに！青春！みたいなライブが
出来ました。
 
 
 
 
 
 
 
一緒にいた4年半は悲しいことも
悔しいことも嬉しいことも
楽しいことも全てを共有出来た
わけじゃないけど少しでも、一緒にいて
気の使わない人、楽な人になれてたかな？
 
そういう存在に慣れていたらいいなって
思うのと同時に私にとって永島は
そういう存在であったのも事実です。
 
 
 
 
 
 
 
 
相談してくれなかったのは
水くさいなって思ってしまったけど
武道館ライブの前日に永島の口から直接
卒業の話を聞けました。
 
 
最後まで話すか悩んでたって言ってたけど
話してくれて私は本当に嬉しかった。
こうやって言ってしまうと他のメンバー、
ファンの皆さんから反感をかう
かもしれないけど、
そういう存在になれてたかと思うと
本当に本当に嬉しかった。
 
 
話を聞いた直後は永島に
私達は謝られたけど、
そんなことなくて逆に
言ってくれてありがとうって
感謝の気持ちでいっぱいでした。
 
 
 
 
 
 
 
 
今まで本当にありがとう
これからは自分のことだけを考えて
前を歩いていってね！
ずっとずっと応援しているよ！
 
 
 
 
 
 
 
 
 
それとね初日の公演にSKE48の
真木子と青木詩織さんが
見に来てくれたよ( ´›ω‹｀)
 
 
 
 
真木子はアンダラ通算何回目？
っていうぐらい見に来てくれてて
アンダラのあの雰囲気が好きって
言ってくれて本当に嬉しいな♡
 
 
 
青木詩織さんはじめましてだったけど
凄いフレンドリーに沢山話してくれました
また遊びに来て欲しいな
 
 
 
 
 
 
 
 
それではこれからも永島聖羅と
乃木坂46の応援をよろしく
お願いします！
ファンの皆さんあっての私達です。
いつも本当に応援ありがとうございます。
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2016/03/23 18:30｜個別ページ｜コメント(274)

2016/0312Sat斉藤優里福島県お邪魔しました

 
 
 
 
今日は「震災から５年"明日へ"コンサート」
に出演してきました。
そして、歌わせてもらった楽曲は
「悲しみの忘れ方」
 
 
 
 
 
福島県になかなかお邪魔する
タイミングがなかったので
今日足を運べて、福島県民の方や
東北地方の方にお会い出来て
良かったです\( ¨̮ )/♡
 
 
 
 
帰りに「来てくれてありがとう！」
というボードを持って下さった方が
いたり、あたたかい方達が沢山
いらっしゃいました。
 
こちらこそ、寒い中待っててくれたり
ありがとうございました。
風邪ひかないでくださいね？
 
 
 
 
 
 
「まつり」 「世界に一つだけの花」で
降ってきた花びら達が頭の上に
沢山積もったので記念にぱしゃり♡
 
 
 
 
 
 
私と誰の後ろ姿か分かるかな？(ˊᵕˋ)
 
 
 
 
 
 
 
 
 
 
 
 
 
正解は、ななでした♡♡♡
このぶろぐを書いてる途中に
まだ髪から花びら達が
おちてきました( ´;o;` )
 
 
 
 
 
まさか、まだ残ってるかも...
 
 
 
 
 
 
 
久しぶりに選抜メンバーと一緒に
お仕事出来て楽しかったのは勿論
歌番組にも出演出来たのがなお、
嬉しさ倍増でした( ´･ᴗ･` )
 
 
 
こうやって、沢山いるアイドルや
アーティストの中から選んでもらえて
当たり前のように出演していますが、
本当にこれって凄いことだし
恵まれていると思います。
もっと日々感謝しなきゃいけないですね
 
 
 
 
 
 
 
そして、このコンサートを通じて
改めて昨日という日がとても
大切な日であって忘れてはいけない日
ということを実感しました。
 
 
 
これからも乃木坂46として
1人の人間として、微力ながらですが
私に出来ることをしていきたいと思います。
 
最後になりましたが、
震災により亡くなられた方々に
御冥福をお祈り申し上げます。
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
