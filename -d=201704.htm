こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
ずっと前にこの写真撮ってたんだけど
貼れずにいたよ〜
服装とかさよく見るとちょっと前の
握手会の時とかだもんな（笑）
 
 
3期生の珠美ちゃんと蓮加ちゃん♡♡♡
写真撮るのはじめてだったしなんなら
3期生と撮るの自体はじめてだ（笑）
 
 
 
 
 
 
まだまだなかなか深くお喋りとかは
出来てないから1人1人と仲良しに
なりたいな〜皆可愛いの( ´･ᴗ･` )
小さい子可愛い好きーー♡♡♡♡
 
 
 
 
 
 
そして先日の個握もありがとうございました
最近ますますはじめまして！の方が
増えてきて本当に嬉しいです( ¨̮ )
 
 
 
もっと色んな方と色んなお喋りしたいの
コミュニケーションとるのめっちゃ好きよ☆☆
知らない人を知るのが好き
この人はこういう考え方なんだとか
何でそこに辿り着いたんだろう？とか
面白いよね人間って（笑）
 
 
 
仲良くなると言葉遣いとか
ちょっと悪くなっちゃうんだけど
そこはご愛嬌で♡（笑）
 
 
 
↑三つ編み..これ実はただの三つ編みじゃないの（笑）
私が持っている方はかずみんが編んでくれました♡

メイクさんに「髪の毛変えてくださーい！」
って頼んでたら丁度セットしてた
かずみんとメイクさんが一緒に返事してくれて...（笑）


きっとノリで返事してくれたんだろうなって
思ってたらまさかのまさか！！！！
私の髪の毛本当にセットしてくれました♡
ありがとう、かずみん♡





そして今回もラブリーちゃんなお花
全部全部大切で可愛いの
 
 
 
韓国のファンの方からもお花を頂いて...
大好きなTWICEの新曲
knockknockという文字まで
入っていたんです嬉しい〜(TT)♡♡♡♡♡
 
 
 
 
 
また握手会で皆さんに会えるの楽しみにしてます
アルバムの握手会もはじまって
きっとこちらは歌衣装だと思うので
また新鮮かななんて思ってます( ¨̮ )
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そして先日のTHE 魂(ソウル)には
上野優華さんが遊びに来てくださいました！
 
 
 
 
とっても可愛い19歳！！！
自称人見知りと言ってましたが全然
そんなことなくてお喋り大好きなの
かと思わせるくらいのコミュ力でした☆
 
 
また絶対遊びに来てほしいな〜
 
 
 
 
♡告知♡

2017/0420Thu斉藤優里彼女感

 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
現在発売中のBLT graphは

出来ずにオフショットとかも
載せられずでごめんなちゃい(TT)

皆お待たせ？かな？（笑）
 
 
 
 
 
 
テーマはね「彼女感」ということで
春夏秋冬の季節を私と過ごせている
気分になれるように撮って頂きました
 
↑同棲しているらしいです♡（笑）


 
 
 
 
 
↑豆乳鍋美味しくてカメラのこと
忘れながらひたすら食べてました（笑）

次からチャレンジしようと思いました♡
 
 
 
 
 
↑ジェンガ！！！私強いよ本当に（笑）



 
 
↑夏休みで浮かれている（笑）
夏は浮かれるタイプ！！！！
ふぁみたんなら分かってるか（笑）
イベントは全力で楽しむタイプ！！！







 
という感じで本当に楽しくて
るんるんしながら撮影してました
その楽しさとか伝わってたらいいな？
 
 
 
 
 
 
 
 
 
 
 
そして最近友達と遊ぶといえば色んなカフェに
行くことになってます（笑）
この前も友達とカフェに
行ってきました♡♡♡♡♡♡♡
今回は「urth cafe｣
 
 
 
ずっとね苺のが飲みたくて飲みたくて
1回チャレンジしに行った時には
お店がcloseしてて諦めたんだけど
どうしても食べたくて再挑戦！！！
 
 
 
 
やっと念願のものを食べれました☆
「ストロベリーボンボン」
 
 
 
 
 
見た目よりめちゃくちゃボリューム感が
あって凄いお腹が満足になるの
でもね、この中に苺1パックぐらい
入ってるんじゃないかってレベルで
贅沢なんだ、絶対女の子は
好きだと思うんだよね♡♡♡♡
 
 
 
 
 
 
その後は「お花見」しに行って
苺のシャンパン\ ♪♪ /
 
 
 
 
この日は苺三昧の1日でした
そして同時にピンクに囲まれた
1日でもありました♡♡♡♡♡
 
 
あのねその日はお花見しに行くのもあって
ピンクの洋服を着ていったんだ
桜と写真撮っときに雰囲気から
可愛くなれるようにって
 
 
 
 
 
 
 
夜ご飯は私のお家で鍋パーティー！！！
BLT graphで食べた豆乳鍋が美味しくて
その味にしたかったけど元が売ってなくて
出来ませんでした(TT)
季節外れっちゃ外れだもんね.......
 
 
 
その日はとってもいい休日を過ごせました
 
 
 
高校からの付き合いの子もいて
乃木坂のライブみにきてくれたり
舞台にも来てくれたり本当に
良い友達を持ったなって実感しています♡
 
 
 
 
 
 
一緒に過ごしてくれてありがとう( ¨̮ )
 
 
 
 
 
 
 
 
それではまたブログ更新する時まで
まったね〜◎
