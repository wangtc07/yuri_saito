2015/1011Sun斉藤優里ガスちゃんじゃない ゆっちゃん

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
現在発売中のMARQUEE
川後Pの魔法にプロデュースに
私もついによんでもらえました♡
 
 
今回の企画はHalloweenということ
だったのですが、てぃーぽよに
「何で今回は私だったの？」って
素朴な質問をなげかけてみたら
てぃーぽよから一言....
 
 
 
 
 
「だって１番Halloweenっぽいじゃん！」
 
 
って、思わず聞いた瞬間あまりにも
短くてびっくりしたけど
でもでも、とっても嬉しいんだ(o´ω`o)
てぃーぽよちゃんよんでくれてありがとう♡
 
 
 
 
それではここでオフショット
載せちゃいたいと思います〜
 
 
 
 
 
 
ちなみにね私のお気に入りは
ブログの一番上に載せた
目隠ししてる写真だよ〜
この雰囲気が可愛くて♡♡♡
 
 
 
 
プロップもとっても可愛くて
プロップだけでも写真を何枚も
撮ってしまいました⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
遠藤さんありがとうございます
 
 
 
 
あーまたMARQUEE出演したいな♡
 
 
 
 
 
 
 
そしてすべての犬は天国へ行くも
残すところ明日の2公演と
なってしまいました〜
 
残り1日ってなると
寂しいです..........泣
 
 
 
 
 
 
昨日も犬メンでご飯に行ってたりと
凄く居心地がいいメンバーが本当に
大好きです、
ああーーー終わって欲しくないいい(´;ω;`)
明日も頑張るぞ〜
 
 
 
 
 
ガスちゃんではなく久しぶりの
ゆっちゃんでばいばい
 
 
 
 
明日のおに魂はかずみんと真夏が
私の代わりに出演してくれます٩( 'ω' )و
ありがたき〜
私も舞台が終わったらかけつけるので
よろしくお願いしますっ！
 
 
そしてこの前はペイジさん。
今日のお昼はサリオちゃん。
夜公演はきっくんが見に来てくれたの♡♡♡
 
 
嬉しい嬉しい、ありがとうございます♡
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
