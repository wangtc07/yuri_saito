2018/0123Tue斉藤優里ひつまぶし

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
皆さん昨日の大雪大丈夫でしたか？
テレビを見ていたらだいぶ、
交通網が乱れていたのできっと
会社とか早く終わっても帰宅するのに
時間がかかるって方が多かったですよね？
 
 
 
 
お疲れ様でした(TT)
 
 
 
 
そして雪が降ってからも今日とか
路面凍結していたり雪が積もりすぎて
段差が見えなかったりと思わぬ所が
危険だったりするから皆さん気をつけてね
くれぐれも、怪我したりとかしないでね
 
 
 
 
 
 
 
握手会で名古屋に行ってたんだけど
嬉しいことがあったんだ〜
あのね好きな食べ物ランキング3位
には絶対入ってくる「鰻」をやっと
名古屋で食べることが出来ました♡♡♡
 
 
 
※名古屋名物 ひつまぶし♡
本当に大好きでランチでも鰻食べるの
必ず自分で自分のことお祝いする時も鰻（笑）






マネージャーさんに何食べてもいいよ〜
って言われていたので一番高い鰻にしました（笑）
2尾入ってたんだよ、たまらんっ！！！
 
その他にも、うまき頼んだり
土瓶蒸しも本当は食べたかったんだけど
お腹いっぱいになりすぎて
食べれませんでした（笑）
 
 
今度行った時は食べなきゃ♡♡♡♡♡
 
 
 
 
 
 
 
皆は名古屋で何か食べた？（笑）


 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
先日のTHE 魂のゲストは
お会いするのは2回目の「シソンヌ」さん
 
 
 
若月と舞台を共演していた部分もあり
とってもお話がしやすかったです☆
またぜひ遊びに来てください！！！
 
 
 
 
そして昨日のゲストさんは
「久住小春」さん♡♡♡♡
 
 
 
大雪の中来て下さり本当に
ありがとうございました
久住さん本当に可愛くて更に
ストイックな部分も知れて
びっくりしました( ´･ᴗ･` )
 
舞台、応援しています☆☆☆
 
 
 
 
 
 
 
名古屋握手終わりのらじらー！も
皆さん疲れてるのに聞いてくれたりして
ありがとうございます☆
私は喋るの好きだから凄く楽しめました
今度はスタジオに行ってお話したいな〜
しゃゆ 中田さん 藤森さん
ありがとうございました(ˊᵕˋ)
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんん( •ㅈ• )

2018/0113Sat斉藤優里お年玉

 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
昨日のMステは見て頂けましたか？
有難いことに「いつかできるから今日できる」
をめちゃくちゃ披露させて頂いてます♡
それに加えてあがるJpopランキングにも
ランクインさせて頂き感謝感謝です♡
 
 
 
V中に高校生達の部活動コラボを
見れたりと楽しかったです
あのね、凄くキラキラしていて
皆かっこいいの( ˙˘˙ )
まさに、THE 青春って感じ！！！
私も戻りたくなった（笑）
 
 
 
勉強も勿論だけど部活動や
学園祭や体育祭などの行事も
全部全部もう1回やりたい（笑）
 
 
やり直したいってことはないけど
出来ることならもう一度って思うから
これからそういう行事が待っている方は
悔いのないように楽しんでもらいたいな♡
 
 
 
 
 
 
 
そして、お正月気分も抜けてきた頃
とてもびっくりなことがありました！
「五木ひろし」さんからお年玉を
頂いてしまいました！！！！！
 
 
 
 
 
何故？私が？って思った方も
沢山いらっしゃると思います（笑）
貰った私もその時はポカーン状態でした（笑）
 
 
 
THE 魂で同じ曜日のアーティストコーナーを
担当している「川上大輔」さん伝いに
頂いたんです( ´･ᴗ･` )
川上さんが、五木ひろしさんとご飯に
行っているときにTHE 魂のお話が出たみたいで...
 
 
...ということで私にも下さいました
本当に感謝しています☆☆☆
 
 
 
大事に大事に手帳に挟んで
つねに持ち歩いてます\( ¨̮ )/
本当にありがとうございました♡
 
 
もしお時間があればTHE 魂にも
遊びに来てください♡
待ってます〜！！！！！
 
 
 
 
やっぱりお年玉って何歳になっても
嬉しいものだね♡♡♡♡♡










たんたんたん ゆったんたん

2018/0112Fri斉藤優里阿妹茶樓

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
台湾旅行第2弾のブログになります（笑）
その次の日の日は「九份」に行ってきました
千と千尋の神隠しの舞台となった場所！！！
 
 
あのね、台湾来る前に千と千尋の神隠し
復習しとけば良かったなって少しだけ
後悔しちゃった（笑）
より一層、そっちのが世界観とか
楽しめたのかなって(TT)
 
 
 
 
 
 
 
 
 
 
でも、日本にはない建物の形とかが
沢山見れてとても新鮮でした
湯婆婆の屋敷の「阿妹茶樓」でも
お茶もしてきました☆
 
 
 
作法を日本語で教えてくれたの！
分かりやすかった！それに
香りを嗅ぐためのコップがあるなんて
知らなかったの色々、学べたよ！
 
 
 
 
 
そして何よりお茶が美味しくて美味しくて
仕方ないのよ、お茶菓子も最高♡♡♡♡
甘い梅干しが初体験だったから
少しびびってたんだけど
全然、美味しかったな( ˙˘˙ )
 
お茶の作法を教えてくれるお兄さんが
日本語凄く上手で、丁寧に教えてくれたんだ！
感謝☆☆☆
 
 
 
 
 
↑階段の真ん中に座ってたわんちゃん♡
全然人が周りを歩いてても気にせずでした（笑）


 
 
 
 
とっても楽しくて休みを満喫出来ました
また遊びに行ってみたいな
かき氷食べ忘れちゃったから
今度行く時は忘れずにだな〜
 
 
 
 
 
そして本日のMステに出演します
「いつかできるから今日できる」
ぜひチェックしてみて下さい！！！
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2018/0110Wed斉藤優里台湾謝謝

 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
1/2からお休みを頂いたので
友達と台湾旅行に行ってきました 
はじめての、台湾♡♡♡
 
 
寄りたい場所を事前に調べてたんだけど
凄く行ってみたい所が多くて大変だった（笑）
夜市に行って小籠包、タピオカ飲んだり...
終電までそこで遊んでたよ（笑）
 
 
 
台北から新幹線？みたいなもので1時間
離れた台中にも移動してカフェ「I'm talato」
黄おじいちゃんが家屋にペイントした「彩虹眷村｣
にも遊びに行ってきたの( ¨̮ )
 
 
 
 
 
びっくりしたのが地面にもペイントされてること！
こういう場所であまりここまでペイント
されているのを見たことがなかったから感動☆
何より、色使いがなかなか日本では
見ることが出来なくて素晴らしい☆
 
 
 
 
ここでは、友達と交代に写真を撮っているのを
みかねた「現地のおじいちゃん」が私達の写真を
めちゃくちゃ撮ってくれたんです(ˊᵕˋ)
しかもね、ポーズとかも教えてくれて
本当に良い思い出の写真が出来ました（笑）
おじいちゃんはきっと私のことなんて
1人の観光客としか思わないかも
しれないけど私達にとっては本当に
有難い存在です♡♡♡♡♡♡♡♡♡
 
 
おじいちゃんぶろぐ見てくれたらいいな（笑）
 
 
 
 
そして、台湾を調べていた時に
めちゃくちゃ行きたいーって
ここは外せないー！って
心のInstagramが騒いだスポットが
こちらのカフェ「I'm talato」♡♡♡♡
 
 
 
 
 
これさ、え？どういう状況なの？
ってなるでしょなるでしょう？
カフェに隣接された撮影スポットなんです！！！
テーマはプールがイメージ？なのかな？
プールサイドのようなタイル貼りの地面
だったりがより一層可愛いアイスを
ひきたててる♡♡♡♡♡♡♡♡♡
 
 
 
 
 
アイスに溺れてきた♡♡♡
しかもね可愛い可愛いピンクとホワイトの配色♡♡♡
 
 
 
あーもう大好き〜幸せ♡♡♡




といった行きたかった場所に
行けた大満足の1日ですが...
台湾といえばここは行くよね〜みたいな
マストの場所は次のブログに書しますね！
 
 
 
 
本当は1つのぶろぐでも良いかな
って思ったんだけど写真の都合上で...（笑）
やっぱりさ？ほら？沢山載せたいじゃん？（笑）
 
 
友達がねしっかりしたカメラを
持ってて撮ってくれてたんだけど
大体1日で240枚くらい撮影してたの（笑）
やばいよね？送ってもらった時保存するのも
大変だった（笑）
でも、それほど残したいものが
沢山ある素敵な台湾！！！！！






それじゃあ、また次のブログ楽しみにしててね？
 
 
 
 
 
 
 
こいつ、愛くるしい♡♡♡











たんたんたん ゆったんたん( •ㅈ• )

2018/0109Tue斉藤優里現在の充電 61%

 
 
 
 
2018年あけましておめでとうございます☆
 
 
 
 
2017年は私にとっては物凄く良い
スタートからはじまり、
悔しい思いも正直ありました
そしてグループとしてもとても大きく
成長出来た1年でした
決して、嬉しい思いや楽しい思いだけでは
ここまでよかったと思える1年には
ならなかったと思います
悔しいことや辛いこと、
そういう感情もあるからこそ
一層感情の起伏があって嬉しいことは
更に嬉しく輝いて感じることが出来るのです
だから、悔しい感情も嫌いじゃない...
むしろありがとうなんて思えたり...
 
 
 
 
レコード大賞を頂けたり
東京ドームで公演が出来たり
自分達だけではなし遂げられなかった
ことも沢山ありました
スタッフさん ファンの皆さん
本当に去年は素敵な年にして
頂いてありがとうございました
 
 
 
 
 
 
まだまだ手にしてないもの
見たことのない世界があります 
今年はそれを1つでも集めたり、
そんな場所に足を踏み入れてみたり
色んな冒険が出来たらいいな
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
そして何と嬉しいことでしょう
2018年1発目のゲストは
「乃木坂46」でした（笑）
 
 
 
 
 
蘭世 絢音ちゃん 遊びに来てくれてありがとう
そして1/10発売の
「僕だけの君〜under super best」を
よろしくお願いします( ´･ᴗ･` )
 
 
 
個人的には新しい曲が4曲も
入ってるのが最高だし何より
それぞれその曲のセンターが
違うのが尚更良い！！！！！
 
 
 
そして、今まで衣装についてピックアップ
されたことがなかったけど今回のブックレット
にはそれぞれの楽曲衣装が掲載されているので
この時はこんなの着てたんだな〜なんて
思って懐かしんだり.............
乃木坂の衣装って本当に凄く可愛く
作って頂いているので細かな所まで
写真で見れるのって本当に良いよね！！！
それだけでも、ずっと眺めてられるし☆
 
 
 
 
 
 
それじゃあ、また更新する時まで( ˙˘˙ )
 
 
※ちょっとだけ時期ずれちゃった？（笑）
