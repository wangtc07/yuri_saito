2015/1025Sun斉藤優里うふふなお仕事♡

 
 
 
 
 
 
 
こんにちたーん( •ㅈ• )
 
 
 
 
 
 
 
モバメで昨日「うふふ♡なお仕事」に
いってきますってお話をしたんだけど
覚えてるかな？実はね、
新宿芸術天国2015のお仕事でした〜
 
 
 
 
 
 
そこの
  • ●Shinjuku GALA Parade
  • ●新宿スタイルコレクション
  • にANNA SUIのお洋服を着て
  • 乃木坂のメンバーと出演させて
  • 頂きました♡♡♡
 
 
 
 
 
 
 
 
私としては人生初ランウェイで
まさかANNA SUIさんとお仕事も出来て
興味のあったことをさせてもらって
本当に光栄でした。
 
 
 
 
 
 
初ランウェイ！初ランウェイ！
 
 
 
 
 
楽しんでね！ってね沢山のスタッフさんに
言われてたけど練習では和気あいあいと
してても、やっぱり緊張しちゃうし
前を歩かせて頂く責任を感じながら
一歩一歩踏みしめました(´°ω°`)
 
 
 
 
 
 
楽しいことや新鮮なことをすると
時間が早く立ったように感じることが
ありますが、まさに昨日がそうでした。
 
 
 
 
 
 
ランウェイ中は何にも覚えてなくて....
私の記憶はポジションに着いた時と
元の位置に戻った時のことしか
覚えて....ない......!!
 
 
 
 
でもそれ程楽しめたってことですよね
よし、ポジティブに考えよう〜笑い
 
 
 
 
 
 
 
私が着た衣装は秋物の方でした
 
 
 
ちなみにサユリストと帽子が
色違いだったから一緒に
撮っちゃいました♡♡♡♡
 
 
 
 
 
 
メイクも今回はしてもらったので
眉毛がいつもよりキリッてしてるの....笑い
 
 
 
 
 
 
 
 
 
 
 
ANNA SUIさんからお財布を頂いたので
早速活用してます(ˊ•̤ω•̤ˋ)
星座モチーフで私は蟹。
 
他にないデザインだからお気に入りです✩
 
 
 
本当これ可愛い可愛すぎる....♡
 
 
 
 
 
 
 
 
 
 
 
 
それでは今からもむふふ♡なお仕事♡
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2015/1015Thu斉藤優里これからドラマ見るの

 
  
 
 
 
 
 
後ろからずっと見つめてる。
私達もほっぺたむにむにされたい。
挙句の果てにさゆりんは
カメラ目線してるし。笑い
 
 
 
でもこの後、私達はむにむにされることは
なかったのでした。笑い
ちゃんちゃん....
 
 
 
 
 
 
はい、何事もなかったかのように
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
「すべての犬は天国へ行く」
全て公演が無事終了しました♡
何回も来てくださった方
遠くから来てくださった方
そしていつも近くで応援してくれている
ふぁみたんありがとうございました。
 
 
 
 
 
私にとって人生二度目の舞台。
とってもとっても楽しかったです！
って言いたいのですが
自分が未熟、過ぎて正直
悔しいことばかりでした。
勿論、舞台に立てている時は
楽しい気持ちで溢れているのですが
私の役柄は容姿はとっても
尖っているヤンキー娘。
でも見た目とは正反対で中身は
とっても心優しくてビビリな娘。
周りの村人達に強くあたるものの、
人がいい裏面がでてしまう
という役柄だったのですが
使い分けがとても難しかったです。
 
 
 
 
 
 
 
私の声が元から特徴的なのもあるせいか、
ドスの声がうまく出せなかったり....
声を低く出す意識を持つことから大切
ということを教えて頂いたり....
立ち方から何から何まで、
勇ましくみせれるようにと....
 
 
 
 
 
 
 
舞台ではふぁみたんにどういう風に
私が映ったかは分かりませんが
その人によって捉え方は様々ですし。
 
 
 
 
 
 
やっぱり思い返すと悔しいこと
ばかりだったけどそんな、
悔しいことがあるからこそ
楽しい。っていう気持ちにも
させてくれるのが舞台の
面白い所だと感じます。
やっぱり楽しい。舞台って楽しいです。
そう感じさせてくれたのは
乃木坂の犬メンバーだったから、
素敵な共演者の皆さんだったから、
舞台を作ってくれた裏方さんだったから
そして堤監督だったからそう思えます。
 
 
 
 
 
 
 
 
 
また同じメンバーで舞台に立つことは
ないかもしれないけど一期一会は
大切なので出会えたことに感謝します。
 
 
 
 
 
 
 
 
また舞台に立つ機会がもらえたら
本当に嬉しいな〜
今度はもうちょっと女の子らしい役柄を
演じたいな....笑い
 
 
 
 
 
 
 
それでは舞台中会話を交わすことが
多かったお二人との写真を載せたいと
思います⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
 
カミーラさん役の猫背椿さん
ちなみにマリネ役の若月と劇中では
同じ年齢設定なんでよ。笑い
アヒル口してくれています♡♡♡
 
猫さんには沢山お世話になりました
有難い言葉を沢山頂いて
本当に感謝しています。
猫さん本当に大好きです
 
 
 
 
 
 
 
そしてグルーバッハ夫人役の山下さん
 
 
休憩中に一緒に脳トレしたり
本当に面白かったです
山下さん沢山いじってごめんなさい。笑い
でもそれ程大好きでしたし
楽しかったです( •ㅈ• )
 
 
 
 
 
はい、シャキーン。決めた。最後決めたよ。
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
