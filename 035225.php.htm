2016/1028Fri斉藤優里ピース大好き

 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
なかなかブログ書くの遅くて
ごめんなさい！！！！！！
書いてない間に墓女千秋楽
個握や選抜発表だったり
沢山のイベントがありましたね
 
 
何から書いていいのやら〜（笑）
 
 
 
 
 
 
あ！そうだ実はまだブログで
16thの制服載せてなくて...
今更ながら載せてみました（笑）
他のメンバーがどんどんと
載せているなか（笑）
 
1番最後だったかな？（笑）
 
 
 
 
今回の制服めちゃめちゃ可愛くて
大好きなんです\ ♪♪ /
足元まで本当は見てほしいんだ
ローファーがねいつもとはちょっぴり
違くてお洒落なの♡♡♡♡♡♡♡♡
 
※番組とかでもし見れそうな機会があれば
ぜひ足元見てみてね（笑）
握手会とかのミニライブとかでも
チェック出来るかもね◎






それではちょっとだけ
振り返ってみますね（笑）
 
まずは、墓女から！！！！
 
 
 
 
 
 
一言で表すならとっても成し遂げたという
達成感？が強い舞台でした
 
 
 
 
でも、やっぱり舞台に苦手意識を
稽古中は感じちゃいました（笑）
何でこんなに苦手だし得意じゃないとか
自分で分かってるのに、オーディションが
あると受けちゃうんだろう？
って考えてたらね、きっと皆で作り上げた
「達成感」「満足感」っていうのを
はっきり味わいたい
からなのかなって思いました
 
 
 
 
 
普段しているお仕事は何が正解で
何が不正解なのか分からないけど
舞台はお客さんから拍手をもらえたら
それは正解に近いんじゃないかと......
その拍手をもらうには
沢山の努力や積み重ねが
あってのものだし、
観てくださった方がつまらなかったや、
普通だったと感じればそれなりですし
受け手の方の言葉にせずとも感想に、
値するものだと思います
 
 
 
 
 
だから大きな拍手、歓声を頂いた時には
本当に嬉しかったです
 
 
 
 
 
 
口では嫌嫌言ってても結局、
千秋楽を終えると楽しかったことや
お客さんから頂いた拍手が
気持ち良く感じるんです
舞台立ててよかったって...........
 
 
 
 
 
 
 
 
私はこれからもチャンスを
頂けるのであれば、それを
逃さずに色んな自分を
見てもらいたいですし
もっといろんなことにチャレンジ
してステップアップしていきたいです◎
 
 
 
墓女の舞台に立てて本当に良かったです
色んなことを経験させて頂いた環境に
本当に感謝しています
 
 
 
 
 
 
 
そして、ファンの皆さん
いつも応援してくれて
ありがとうございます！！！！
 
 
 
 
 
 
 
 
 
 
 
 
 
そしてこの前の個握では
レーンが朝からワイワイガヤガヤ
賑わっていて本当に嬉しかったな〜
 
 
 
朝はねいつも寂しいことが
多かったからさ(´._.｀)（笑）
 
 
 
 
 
 
1部2部☆☆☆
 
 
 
 
このシャツ下までの写真
撮れてないけどワンピースなの
程よいゆるっと感もありつつ
実は生地はサテンという
女の子らしさも忘れずに♡♡♡
 
秋の渋めの色味love♡♡♡♡♡
 
 
 
 
 
 
3部4部
 
 
 
 
普段着ない色味、髪の毛も
分け目をかえて少し大人っぽく
してもらいました！！！！
 
ちょっと気の強そうな女の子♡
 
 
 
 
 
 
 
 
 
明日の個握ではHalloweenが
近いということで仮装しちゃいますよ♡
何になるかはまだ内緒（笑）
 
 
 
 
でもヒントを出すと
誰かさんとおそろいですよ\( ¨̮ )/♡
しかもね、その誰かさんの中の1人の
お手製です（笑）感謝♡♡♡♡♡♡♡
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そしてこの前のおに魂には
「ビッケブランカ」さんが
遊びに来てくださいました◎
 
 
 
 
 
とってもお話がしやすく、
曲もとっても素敵で
私も携帯の中にすぐ入れちゃいました（笑）
ピアノの才能が素晴らしくて
自分にはないものだったので
真剣にお話を聞いてしまいました！！！
 
 
 
また絶対来てくださいね？
 
 
 
 
 
 
 
 
 
 
そして16thの選抜発表と同時に
ななみんの卒業発表もされましたね
 
 
 
 
私達も聞いた時突然のことだったので
唖然としてしまいました(´._.｀)
 
 
 
 
でも、ななみんの決めたことだから
私は背中をおしてあげることしか
出来ないけどそういうのって
沢山悩んだ末のことだから
短い時間、同じ番組に出ることや
同じお仕事など少ないかもしれないけど
これからその分、その貴重な時間
目一杯楽しんでいきたいな◎
 
 
 
 
 
 
 
 
 
色んなこと書いてたら結構
長いブログになっちゃた(´._.｀)
自分の選抜発表についてはまた
しっかり読んでもらいたいし
考えて書きたいので次回以降に
書きますね(´._.｀)
ごめんなさい...(泣き)
 
 
 
 
 
 
とりあえず、明日の握手会も
楽しみたいのでもう寝ようと思います（笑）
気まぐれでごめんなさい♡♡♡♡♡♡
 
 
それじゃあ、おやすみなちゃい♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
