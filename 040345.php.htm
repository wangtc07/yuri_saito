2017/0819Sat斉藤優里うきうき

 
 
 
 
 
大阪公演お疲れ様でした〜
皆さん楽しめましたか？
 
私は物凄く楽しんじゃいました！！！
 
 
 
 
 
 
3期生とも仲良くなれたり
お喋りする機会も増えてきて
とっても嬉しい限りです( ˙˘˙ )
 
 
 
 
ライブ終わりに犬メンで
若月の部屋に集合して
ババ抜き・人狼・ワード人狼？
UNO(まっちゅんのお母さんに
急遽買ってきてもらったらしい（笑）)
をしました( ˙˘˙ )
 
めっちゃくちゃ盛り上がって
もう最高な夜だった（笑）
 
 
 
 
 
 
 
明け方までまいちゅんはずっと
ツインテールだったよ( ˙O˙ )
お腹痛くなるほど笑って
UNOは誰も上がれなくて
永遠にプレイするし、
ラスト1枚になった人のカードの色
全力で皆で当てに行くし、なんなら
ちょっと意地悪もするし本当に最高！！
盛り上がり方が尋常じゃなかったな
 
 
 
 
 
 
 
セルカ棒持ち歩いてるんだけど
かまえるとね皆いつも入ってきてくれるの
声かけると更にぎゅうぎゅうに（笑）
 
 
そんな写りたがりの皆が大好き♡
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
そして先日のTHE 魂ゲストは
「橋本裕太」さん☆☆☆
 
 
 
 
年齢も近いせいかとっても
お話もしやすく楽しい放送でした
甘い声素敵でした〜
また、ぜひ遊びに来てください( ˙˘˙ )
 
 
 
 
 
 
それではまたブログ
更新する時までばいばい♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
