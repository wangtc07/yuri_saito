2017/0506Sat斉藤優里ピピピ〜SIGNAL〜ピピピ

 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
大好きなTWICEが5/15にカムバします♡
本当に嬉しいですし、何よりその後に
日本でもリリースタイミングが
あるということで同じアイドルとしても
本当に尊敬しています♡♡♡♡♡♡♡♡
 
 
 
とっても活動が忙しくて大変だと思うのに
日本に来てくださったり応援させてもらってる
こちら側としては本当に有難いです( ˙˘˙ )
 
 
 
 
 
今回のシングルはまだまだどうなるか
分からないのですが日に日に
写真やカードの種類など発表されていて
本当に毎日見るのが楽しみで楽しみで
仕方ありません！！！！
 
今回のキーポイントにもなりそうな
「SIGNALポーズ」？本当に可愛くて
思わず真似してしまいました\ ♪♪ /
 
 
 
早くダンスや歌など聞きたいな♡♡♡
 
 
 
 
 
 
 
 
 
そしてこの前のGirlsAwardに乃木坂46の
ライブパフォーマンスとして
出演してきました♡♡♡♡♡
 
 
そこでは、インフルエンサー
命は美しい 裸足でsummer ガールズルールの
4曲も披露させて頂きました有難いですし
沢山の可愛い子ちゃん達とわちゃわちゃ
出来て本当に楽しいイベントでした
 
 
 
 
 
 
中でも、沢山のボードをステージの上から
見つけられて出来るだけそれには
答えられるタイミングでは、
答えたのですが出来なかった方
ごめんなさい(TT)
 
今度またどこかでお会い出来るの
凄く楽しみにしています〜
 
 
 
 
 
 
私もいつか皆みたいに可愛い洋服とか着て
ランウェイとか歩いてみたいな
 
 
 
 
 
 
 
そして今日の名古屋での全握では
真洋とのペアでした♡♡♡♡♡♡
GWも後半？ということで
凄くうきうきでした( ˙˘˙ )
 
 
 
 
Rotty夢の皆さんもとても優しくて
1日がhappyになりました
また次回も一緒に出来たらいいな〜
 
 
 
 
 
 
 
今日の夜ご飯は皆で食べに行ってきたよ
お腹いっぱい満腹( 　˙³˙　 )
皆で笑いながらご飯食べる時間が本当に
大好き♡♡♡♡♡♡♡♡♡♡♡
 
 
それではまた握手会でお話
しましょうね楽しみにしてます◎
 
 
 
 
↑かりんちゃん不在だけど93年組で
集まれたので写真撮っちゃた(ˊᵕˋ)
4人揃ってはまた今度一緒に撮ろうね♡
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
_
