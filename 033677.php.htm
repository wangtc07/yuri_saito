2016/0801Mon斉藤優里トリップトリップトリップ2

 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
前回のブログのコメントに
おすすめの旅行先について
ふぁみたんに聞いてみたら
なんと海外〜地方など、
沢山の名所が書いてありました！！！
 
 
 
 
凄い、嬉しいしこれから
検索かけてメモして
「行ってみたいリスト」に
追加します( ˙˘˙ )
 
 
 
 
 
私が「鰻好き」なのを知って下さる方が
鰻の美味しいお店を書いてくださったり、
時間があれば行けそうだから
ラッキーちゃん！！！！！！
 
 
 
スイカメンバー
(かりんちゃん 純奈 てぃーちゃん)
 
 
 
 
皆、鰻好きだから誘って行かなきゃ（笑）
 
 
 
 
 
 
 
 
 
そして、この前の旅行のお話の続き...
USJ☆☆☆☆☆☆☆☆☆☆☆☆☆
 
 
 
「フライングダイナソー」
かりんちゃんは絶叫系が苦手だったので
今回は辞退しちゃったので2人で挑戦！
 
降りる前の登っている時が
ジェットコースターの骨組みが
見当たらなくて本当に怖かった〜(泣き)
 
 
 
 
だってだって！！！！
 
下はただの湖？なんだよ(´·_·｀)(´·_·｀)
 
 
 
あれが更に怖さを引き立たせてたな！ムッ
 
 
 
 
 
 
下ったと思ったら、捻られたり
空が見えたりとんでもなかった（笑）
 
 
 
 
 
 
そして必ず皆この場所は
訪れるんじゃないかな？
ハリー・ポッター♡♡♡♡♡♡
 
 
 
 
 
２回も乗っちゃた乗っちゃた〜
フルーパウダーで移動するとこは
思わず「展望台」って叫んじゃった（笑）
 
 
 
凄い楽しくて本当に楽しかった
やっぱりいつも最後のシーンでは
手広げちゃうんだ
風が気持ちよくて〜
 
 
 
 
 
 
 
 
 
そして夜はサプライズで
誕生日をお祝いしてもらいました☆
とってもびっくり！！！！！！！
ホテルの部屋に帰ったら、
私が欲しがってた
スヌーピーの大きいぬいぐるみが
ベッドで寝てたんです( ˙˘˙ )♡♡
 
 
 
 
めちゃくちゃ、びっくりして！！！
 
 
 
 
 
一瞬、ベッドの前で棒立ちに
なっちゃいました（笑）
 
本当に大切なぬいぐるみ☆
 
 
 
そのスヌーピーには名前が
ついてるのですが何だと思いますか？（笑）
かりんちゃんがスイカメンバーからだから
「スイカ」と名付けました！！！！！！！
 
 
 
 
 
 
そんな感じの楽しい旅行でした
今度は純奈も一緒に来れるといいな〜
 
 
 
 
 
 
 
 
そして昨日はスイカメンバーで
フルーツパーティーをしました( 　˙³˙　 )
 
 
 
かりんちゃんはこの時もう、
お風呂に入っちゃったので
スイカで顔隠してる(´·_·｀)(´·_·｀)
 
 
 
 
他にも沢山のフルーツを買って
夜ご飯を食べていたのにも関わらず、
メロンひと玉 梨×2 桃×3
スイカ半分くらいをペロリと
食べちゃった（笑）
 
 
 
 
↑甘いものは別腹...
 
 
 
 
 
 
これからもスイカメンバー
(かりんちゃん 純奈 てぃーちゃん)
のことをよろしくお願いします◎
 
 
 
 
 
 
 
告知\( ¨̮ )/♡
 
 
 
・モデルプレスさんにはじめて1人で
インタビューして頂きました！！
動画もチェックして欲しいです
 
緊張して実は3回ぐらい
撮り直したりしてるんです（笑）
 
http://mdpr.jp/interview/detail/1601784
 
 
 
 
 
 
・8/5のぎ天のキャンプ会が
配信されちゃいます( ˙˘˙ )
久しぶりのキャンプ！！！
 
わちゃわちゃ色んなことをやらせて
もらえたので見応えあるかと思います
 
 
 
 
 
そして今日の20:00〜のおに魂！！！
今日も沢山好きなことや
リスナーさんのお便り沢山
お待ちしております( ´･ᴗ･` )
 
 
 
 
それでは、またね にゃんちゃんなの
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
