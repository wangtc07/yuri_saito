2016/0816Tue斉藤優里ゆっかいだな〜♪

 
 
 
 
 
 
最近はライブ三昧です\ ♪♪ /
 
 
私はとっても乃木坂のお仕事をしていて
好きだな楽しいなって思うもの
ランキングの中に必ずライブが
入ってくるので最近の充実した
感じがとっても大好きです( ˙˘˙ )
 
 
 
 
 
 
 
その大好きな気持ちがもっともっと
歌っていて踊っていて表現出来ていて
ふぁみたんや皆さんに伝わっていれば
いいな〜なんて思ってたりもします！！！
 
 
 
 
特にここ最近で嬉しかったのは
仙台のライブでMCを任して
もらえたことです\( ¨̮ )/♡
 
 
 
 
 
全体のライブではお喋りできる
メンバーが多い分なかなか回って
来なかったりするけど、
そういう機会をあたえてもらえること
自体に感謝ですね♡♡♡♡♡♡♡
 
 
 
 
これをきっかけにもっとチャレンジ
してみたいことも見えてきたり、
弱点となる部分も理解出来ました
これからもっと頑張らなきゃだね！！！
 
 
 
 
 
 
 
また地方での楽しみのひとつでもある
ご飯会！！！皆勤賞なんです！！！
さゆりんご軍団＋親衛隊＋スイカメンバー
 
※スイカメンバーとは...(かりんちゃん 純奈 
てぃーちゃん 私)の4人組のことである
 
 
 
 
 
 
 
そこで牛タンやお刺身なども美味しかったけど
好きな食べ物の一つでもある
鰻が食べれました♡♡♡♡♡♡♡♡♡♡♡
なんて幸せなことなんでしょう！！！！！
 
 
 
 
よいしょよいしょ...
 
 
 
 
 
ぱくっ♡♡♡
 
 
 
 
 
 
本当に美味しいご飯屋さんでした
また仙台に遊びに来たいなって
幸せな１日にもなりました◎
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そして先日のおに魂のゲストに
来てくださったおに魂ファミリーの
上間綾乃さん\ ♪♪ /
 
 
 
 
久しぶりのお姉さんにお会い出来て
とっても嬉しかったです！！！
ひとつひとつの言葉にとっても
重みがあって素敵な女性でした
 
私も上間さんみたいな女性に
なりたいです( ˙˘˙ )
 
 
 
 
 
 
 
本日のゲストはWHITE ASHさん\ ♪♪ /
 
 
 
 
 
もう数えてみると5回も
遊びに来て下さっています！！！
嬉しいですよね〜
 
毎回来て下さって本当に
ありがとうございます
 
山さんのフリースタイル俳句や
ホッペン本当に大好きで
いつも楽しくきいています♡♡♡
 
 
 
次回は彩さんもぜひですね（笑）
特技が楽しみだな（笑）
 
 
 
 
 
 
 
 
 
 
 
 
それでは、さっき帰ってきたので
急いでお風呂に入って明日の
準備しなければ( 　˙³˙　 )
 
 
 
皆さんも夏休み？だからって
夜更かしはだめですよ（笑）
 
たまになら良いよ（笑）
 
 
ちなみに私は夜行性だから
夜寝るのは遅いよ（笑）
 
 
 
 
 
 
 
 
 
 
それでは、おやすみなさい\ ♪♪ /
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
