2016/1128Mon斉藤優里夢のあるものが好き

 
 
 
 
 
こんにちたーーん◎
ちょっと前に紅白の発表
されましたね\( ¨̮ )/
 
まだ何にも決まってないけど
「乃木坂46」の名前があるだけで
本当に嬉しいです！！！！！！！
 
 
 
全員で今回も出演出来たらいいな〜
なんて思います
皆さんもぜひ大晦日はご飯食べながら
チェックしてくれたらなんて♡♡♡♡
 
 
 
 
 
 
 
 
そしてこの前の全握はなんと
仲良しのかりんちゃん！！！
 
 
来て下さる方には沢山の2人の
共通点があるとこに気づいて
2人のコンビ名を沢山もらいました（笑）
 
・ディズニーコンビ
・たいまつコンビ
・声が大きいコンビ
 
 
 
...などなど（笑）
 
 
 
いつの間にか沢山の名前をつけてもらうほど
特徴的なシーン？とかがあるみたいで
自分達では言われるまで
気づかなかった部分が
あったのでとっても
楽しかったです\ ♪♪ /
 
 
 
 
 
また、かりんちゃんとのペアを
ぜひ希望します（笑）
いつかなった時はよろしくね♡♡（笑）
 
 
 
 
 
 
 
 
 
あのね！！！ファンタスティックビースト
をつい最近見てきました◎
公開初日から見に行きたかったけど
行けなかったので（笑）
 
 
 
 
 
映画を見る時は、後ろブロックの
センター席が私のお気に入りなので
そこを予約して.......♡♡♡♡♡♡♡♡
(更にこだわりを言うと、字幕版ではなく吹き替え派
その理由は字幕だとその文章を目で追っちゃって
映像をちゃんと見れなくなってしまうから（笑）
映像をしっかりと見たい派なので)
 
 
 
 
全然合わないコンタクトで
行っちゃて映画の最中も目薬が
手放せなくて(TT)
 
 
 
 
 
なんかねなんかね、よく分からないけど
合わないって分かってたのにつけてって
ものの見事に撃沈(TT)
あーー悔やまれる(TT)
 
 
 
 
 
 
だから、もう1回見に行こうと思うんだ（笑）
ちゃんと集中して見たくてさ（笑）
 
 
この前は字幕版見たので次は吹き替えで...
(更にこだわりまで言うと（笑）
3Ｄもあんまりです（笑）
物語に集中したいので（笑）
ただ、動いたり沢山しそうだなって
言うものは見たりするよ！！（笑））




 
 
映画見てたらハリーポッターのBlu-ray BOX
欲しくなっちゃたから買うんだ( ¨̮ )
影響されちゃった♡♡♡♡♡♡♡♡
 

あーー私も魔法使いたい♡♡♡
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
先日のおに魂には「PASSPO☆」さんが
遊びに来てください下さいました☆☆
 
 
 
 
キャッチフレーズがめちゃめちゃ
可愛くてとっても素敵なグループでした
思わず一緒にやりたくなって
しまいました（笑）
 
 
またぜひ、お越し頂いた際には
一緒にやらせて下さい！！！
 
 
 
 
それでは今日のおに魂も
だんちゃんと一緒に頑張りますので
絶対チェックしてくださいね？
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
