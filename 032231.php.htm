2016/0513Fri斉藤優里それでは聴いて下さ

 
 
 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
今日は2016年最後の
13日の金曜日だよ〜
何か、日にちで考えると
2016年は半年以上残ってるのに
13日の金曜日っていう日付と曜日で
考えると最後だなんて面白いね◎
 
 
 
 
 
 
 
 
 
 
この曲は沢山乃木坂の曲がある中で
ライブでいつも披露させて頂いて、
本当に有難いなって思ってます。
 
 
 
 
 
それに昔からの曲でもあるので
皆さんが一緒に歌ってくれる
落ちサビ部分もあったり
本当にそこの部分を聴いているのが
大好きです。
皆さんにとってはライブの中での
1曲とか、盛り上がれる曲
と思っているかもしれませんが
私にとって凄く大切な曲です。
 
 
 
 
 
いつか、ふぁみたんだけでなく
乃木坂を応援して下さってる皆さんに
この曲は外せないよね！って
言ってもらえるようになりたいな
 
 
 
 
 
 
 
それではお待たせしました！
13金企画にうつりたいと思います♡
 
今回12日にも更新したから00:00まで
あんまり時間がなかったよね( ´;o;` )
ごめんなさい、反省
 
 
 
 
 
それでもいつものコメント数よりも
書き込んでくださる方が多くて
嬉しいかったです...♡♡にゃんちゃんなの
映画に対してのコメントや
握手会行くよ！なんてお話も
全て楽しかったな〜
 
それでは見事下13をGETした
皆さんにお返事書いてみたよ◎
 
 
 
 
 
 
 
 
 
 
 
 
 
 
本当に本当に参加して下っさた
皆さんありがとうございました(ˊᵕˋ)
 
次の13金は2017年の1月です...笑い
↑思ったよりも近かった♡
 
 
 
 
 
 
 
ちなみに前回のぶろぐで
観たい映画をメモしたりしている
って書いたんだけど次気になる映画は
 
 
・ミュージアム
・世界から猫が消えたなら
・ペット
 
 
 
ペットはバナナマンさん2人とも
声優さんをしているんだよ( ˙˘˙ )
可愛い♡♡♡♡♡♡♡♡♡
だからね観たいっていうのもあるけど
ワンちゃんのお話でコミカルな予告
だったから目を惹かれたのもあるの。
コミカルなのも大好きだし
命をかけたみたいなミステリーゲーム
的なのも大好き( ˙˘˙ )
 
 
あんまり恋愛ものは
観ないんだよな〜
何かそんなになんだよね...笑い
 
 
 
 
 
 
皆さんの映画の好みが
知りたい知りたい！
後ね、おすすめ作品！
鉄板でも少し変わったのでも
大歓迎！ぜひ教えてね♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
