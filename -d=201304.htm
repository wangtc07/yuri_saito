2013/0430Tue斉藤優里 過食気味になりつつあるのね

 
 
 
昨日は握手会で何人もの方に
『今日ブログ書くからみてね♡』
って言ったのにもかかわらず
ブログ更新してなくて
ごめんなさい(´;ω;`)
 
 
昨日は握手会が終わったら
おに魂でした♪
握手会の時にたくさんの方が
『今日も聞くからね！』と
言ってくれて本当に嬉しかったです！
 
 
ちゃんと聞いてくれたかな？♡
 
 
うっそーーん。
聞いてませんでした(´;ω;`)
 
 
 
とか絶対なしだからね♡
ちゃんと聞いたか聞いてないか
ブログのコメントに
書いてくださいね♡
たんたんは待ってるぞ。うんうん
 
 
そしてもう一つ嬉しいことが
握手会で『おに魂から来ました』
という方が何人も来て頂けて
驚きでした( •ㅈ• )
昨日来てくれた方本当に
ありがとうございました！
 
 
そして昨日の服装ですが
こんな感じだっ！てい
 
いつもよりラフかな♡
でもスカートでカジュアルに
しすぎずでいつものあたし。
 
 
 
最近ニューバランスの靴に
興味を持ち始めました♪
カラフルなやつが可愛くて...
 
 
でも色んな種類どれもこれも
可愛くてまよってしまう(´;ω;`)泣
 
 
らりんにどれがいいか。
聞いてみたら『これはあんまり..』
と言われてしまった(´;ω;`)おん
 
永島さーーーん。
 
 
 
ま、好みの問題ですからね(´;ω;`)おほ
 
 
 
 
それではまたね♡
みくものことはまた
別記事に書きたいと思います！
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2013/0425Thu斉藤優里 たまにさー時刻表の電車が見当たらないと思う時あるよね(´;ω;`)

 
 
 
こんばんたーん( •ㅈ• )
 
 
 
いきなりですが最近の
斉藤の生活は朝から晩まで
『16人のプリンシパル』の
稽古にあけくれてます！
 
 
ここでちょっとだけ
プリンシパルの情報を。うん
 
 
 
前回よりも覚えることが
多すぎて頭をフル回転に
フル回転にフル回転してて
爆発しそうです(´;ω;`)わー
 
だけど去年よりもあたしは
物凄く自分でも分かるくらい
楽しんでいて毎日が充実してます♡
 
 
だから絶対今年のがメンバーの
生き生きしてる姿を見れるん
じゃないかなっと( •ㅈ• )
思ってるの。
 
 
 
だからね、今このぶろぐを
読んでるふぁみたんで
『行こうか迷い中...』とか
思ってる方はぜひ見にきて
もらいたいな♡♡
 
 
 
なんだか最近プリンシパルの稽古を
毎日しているせいか声が大きく
なった気がする( •ㅈ• )うん
 
 
あたしの声おっきいのに
更におっきくなるって...
 
 
 
 
握手会会場に既に響き
渡ってるっていうのに...
 
 
今度は握手会会場から
もれて外にまで聞こえて
しまうかもね♡
 
 
 
 
お団子とてかてかでワイルドに
この衣装も結構お気に入りでした♡
 
 
では、台本覚えてきます！
しゅわっちととと。
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

褒めてくれる方が多くて
とても嬉しかったです。

私服はこちらになります。

『Noela』というお店です

言われたけど2部制だと何か
物足りないな(´;ω;`)んーー

あ、でも逆にふぁみたんが
疲れてしまうかな？(´;ω;`)んー

ずっと楽屋で暇だったから
せいたんとからりんと一緒に
寝てたんだ♡♡
 
それも一つのベットで( •ㅈ• )
 
 
 
あたし達どれだけ仲良すぎなの♡
とか話しながら4部まで楽しんで

初めての収録

思います( •ㅈ• )

3列目の1番はじの上手です♡

どういうことか分かるかな？
 
 
ちゃんとあたしだけ
見ててね♡♡
ってことだよう( •ㅈ• )
 
 
 
新幹線来てるのに八つ橋を
ダッシュで買いに行ったのは
なーちゃんとあたし♡♡
 
 
あーー間に合って良かったwえ
