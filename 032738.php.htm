2016/0612Sun斉藤優里お疲れ様でした

 
 
 
 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
46時間TVお疲れ様でした
MC組、スタッフさん、メンバー、
遊びに来て下さった芸能人の方
そしてちょっとだけでも
配信を見て下さった皆さん
本当にありがとうございました♡♡♡
 
 
 
 
 
 
 
今回も沢山の企画に出させて頂き、
特に自分の冠番組！乃木坂電視台では 
「ノーリアクション女王！」
とうたっておきながら、
電気系と足つぼには弱いということを
お見せしてしまいました( ´;o;` )
 
 
 
 
食べ物系は全て本当に余裕だったのですが...
センブリ茶・デスソース
↑これは何でだろう？
食べ物の味はしっかり分かるのに
全部美味しかったんです( ˙˘˙ )




ただ、足つぼは先生がちょっと
意地悪さんだったのでそれに
耐えられませんでした（笑）
でも先生のマッサージ
痛気持ち良かったです！
マッサージは痛気持ちいに限ります！
最初の感じだったら気持ちいいな〜
って思ったのにな◎




特に言いたかったのが
電気系は本当嫌いです！！！！！
おに魂でもビリビリマシーンが
本当にだめだったり...
今日のスタジオトークで
ビリビリマシーン使ってるとこを
見て恐ろしかったよ〜泣き
 
 
皆、よくあんなに勇敢に
ビリビリマシーンと向き合っていた
乃木坂は逞しい（笑）拍手
 
 
 
 
 
だけどね、今回企画をするにあたって
打ち合わせをしてたんだけど、
ビリビリマシーンの名前を出したんだけど
こういうのってすぐ用意出来ちゃう
ものなんですか( ´›ω‹｀)
 
なんてこったー( ´›ω‹｀)
 
スタッフさん流石に準備早すぎるよ（笑）
 
 
 
 
 
 
 
協力してくれた生駒と若月
本当にありがとう
特に、生駒には罰ゲームを
受けてもらって(´·_·｀)申し訳ない
今回はこの2人が居ないと
成り立たないようなものだったで
本当に有難かったです！！！！
 
 
 
 
 
 
そしてずっと楽しみにしていた
人狼ゲーム\( ¨̮ )/♡
 
 
 
 
1回戦目のかずみんは本当にうまかった
ゲームマスターのように今考えてみると
言葉巧みにまわしてたな〜って◎
 
経験の差が出ましたね( ´;o;` )
 
でも、やっぱりかずみんも
ちーちゃんには嘘をつけなくて
人間性が出て可愛かったな♡
 
 
 
 
人狼やってて思ったのが
市民だったのに追放されることが
何よりも悲しい（笑）
 
 
人狼フェイス脱却したい...
ちーちゃんと一緒に脱却せねば！！！
 
 
 
2回戦目はサユリストの見事大活躍☆
占い師めためた助かったし
あんなに楽しそうにゲーム
していて可愛かったよ( ´･ᴗ･` )
 
こっちまで一緒にるんるんしてしまう（笑）
 
 
 
 
 
 
人狼はやっぱり楽しいから終わった後も
すぐ開催しようねって約束したんだ
楽屋とかでするのが1番楽しいの
初めてのメンバーもルール
覚えられたと思うから
今度はもっと多人数で沢山の役職ありで
やりたいな\( ˙-˙ )/\( ˙-˙ )/
 
私はまた騎士やりたい♡
↑だって1番かっこいいんだもん



 
 
 
 
 
 
 
ひなちまの電視台にはゲスト出演
させてもらって、「ひなちま親衛隊」
として声を出させて頂きました♡
 
 
ちまのコールが無事決まって
本当に良かったです〜
 
皆さん！！！！夏のコールは
「ちまちまひなちまfufu！」
ってぜひ叫んであげてください！
 
 
 
 
私の可愛いちまが絶対
喜んでくれると思うの！
やっぱり喜んでる顔が1番可愛いから♡
お願いします♡
 
 
↑私はひなちまをずっと推していたので
今回はひなちまのファンとして
出演してました（笑）
楽しかった〜♡♡♡
 
 
 
 
 
 
 
 
最後に完成した黒板アートは
皆で頑張ってきた証
更に、きっかけの曲がのっかると
胸があったかくなりました
思わず画面をくいいるように、
見てしまいました◎
 
 
 
 
 
 
 
 
皆の手で完成した作品
 
 
こんなに大きな作品も1人じゃなくて
皆で協力すればあっという間なんですね
沢山の時間を使って作ったものなのに
最後に名前が消えてしまったのは儚いけど、
私達と似ているような気がしました
 
 
 
前に立ってキラキラ輝いている
時間もあるけど、それ以上に裏では
切磋琢磨して沢山の時間を使って
準備しています。
 
 
 
 
そんな似ているようなとこが
自然と見入っちゃたのもあるかな？
また、46時間TV出来るような
素敵なグループで居続けたいな
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
