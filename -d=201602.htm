2016/0226Fri斉藤優里にゃんちゃんなの

 
 
 
 
 
 
 
 
こんにちたーん( •ㅈ• )
収録がはじまるギリギリまで
いつも前髪を固定している
たんたんだよ♡♡♡
 
 
 
待っている間に前髪の分け目が
崩れたりするのが嫌なの...
 
 
 
 
 
乃木坂の4ちゃいのお誕生日と
おに魂の日が被っていたのですが
なんとサプライズで鬼にゃんから
口ファンファーレ？でおめでとう？の歌を
歌ってもらいました( ´･ᴗ･` )
 
 
その時の画像がこちら
 
 
↑うわっ！鬼にゃんめちゃめちゃ
全体的に伸びてるし私なんとなくだけど
目つぶっているように見えるし
なんなんだ！何なんだこの画像！






っていう感じで楽しくやってる
おに魂なんですよそれにその日に
なったんもお祝いしてくれました
うわい、めっちゃ嬉しい♡♡♡
 
なったんもありがとう(ˊᵕˋ)
 
 
↑ちゃっかり 「46」ポーズ
なったんといつもすれ違いになっちゃうから
もっと仲良くなりたいな♡♡♡





ゲストはEDGE of LIFEの
お2人でした( ´›ω‹｀)

 
 
モノマネ大会ありがとうございました
萩尾君にいきなりモノマネを
ふられた時はめちゃめちゃ
びっくりしました( ´;o;` )
 
すみません。メンバーモノマネしか
出来なくて.....泣
 
今村君も、なかなかモノマネが
似てました！
...モノマネの話ばっかり。笑い
 
 
 
 
 
 
またぜひ、楽しいおに魂に
遊びに来てください\( ¨̮ )/♡
 
 
 
 
 
 
 
エンディングにて「乃木坂1の親近感」
を目指しているというお話をしましたが、
口にすれば実現するということを
信じていこうと思います♡♡♡
 
目指せ！「乃木坂1の親近感」
 
 
 
最近モバメでめちゃめちゃ連呼してる
にゃんちゃんはなんとなーく
可愛いかなって思って連呼してます
だからね、あんまり意味はないの。
 
 
えへ☆ にゃんちゃんなの。
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2016/0223Tue斉藤優里東北地方行ったら東 京寒くない

 
 
 
 
 
 
乃木坂4ちゃいの
お誕生日おめでとう\( ¨̮ )/♡
ぱんぱかぱーん
 
 
 
 
 
 
 
乃木坂46時間TVはどれくらい
見れましたか？
全部見ました！って方も
居そうですよね、強者さん♡
 
 
 
 
生放送で46時間は前代未聞
なんじゃないかな？
とてつもなく凄いことを
成し遂げちゃいましたね!!
特にね、この機会でMC組の
真夏 かずみん 生ちゃん まっちゅん 若月
の5人が本当に偉大すぎて...
こんな１番大変な立ち位置なのに
やり切ってくれて本当に感謝しています。
 
 
 
 
 
今回この企画が成立したのも
メンバー、スタッフさん
視聴者の皆さんそしてMC組が
居たからこそ出来たものだと思います。
 
本当にありがとう( ´;o;` )
お疲れ様、沢山ゆっくり休んで欲しい...
 
 
 
 
 
さて、皆が頑張ってくれている間
私はのぎ声リレー担当で
東北地方の宮城・山形・福島の3県に
お邪魔させて頂きました。
 
中継では食べてばっかりだったので
皆に怒られちゃいましたが
ちゃんと声も集めていたんですよ
 
 
 
 
でもね、びっくりしたのが
地方に行くことはその日の朝に
発表したのにも関わらず、その近辺に
お住まいの方は勿論他県から沢山の
皆さんが集まってくれました( ´;o;` )
ありがたいな、しかもね推しタオルとか
サイリウムまで持ってきてくれてるの
もうさ！どんだけ準備いいのよ♡♡♡
 
 
それにお店の方も凄く優しくて
沢山ご飯を食べさせてくれたり
感謝、感謝ののぎ声リレーでした
 
 
 
宮城・山形・福島の皆さん
歓迎してくれてありがとうございました！
 
 
↑とても福島県に来た感じがしたので
急いで写真撮ったけどフォーメーション
綺麗すぎじゃない？( ´･ᴗ･` )笑い
 
ちなみに、宮城・山形でも撮りたかったけど
時間がなかったのでごめんちゃい。
また行った時にぜひ！
 
 
 
↑1軒めの餃子照井さん
インパクト◎にんにく◎
 
 
 
 
↑米沢牛の案山子さん
本当は味わって食べたったかったです...
もっとゆっくりとね...桃のジュースも
最高に甘くて大好きになりました
 
 
↑利久さん
青唐辛子や、とろろも美味しかった
牛タンらぶたん♡♡♡にゃんちゃん
 
 
 
 
 
 
それでは、今度はアンダーライブで
ぜひ！東北の皆さんに会いたいです
日程とかは詳しくホームページに
載っているので良かったら
そちらの方チェック下さいね？
 
 
 
それではこれからも乃木坂ちゃんのことを
よろしくお願いします！
それではお仕事行ってきます♡♡♡
 
 
 
 
今日も良い日になりそうだ
ふぁみたんにも良い日に
なりますように(ˊᵕˋ)
にゃんちゃんなの。

こんにちたーん( •ㅈ• )

選抜発表がありました。

大きく変化があるのかと
ドキドキ、しちゃいました。

今回もアンダーメンバーです。
やっぱり世間からの見え方として
選抜にいる方が良いと考える方が
多いと思います。
私もそう、思う時期もありました。
 
 
 
 
 
ですが、選抜にいてもアンダーにいても
変わらず私のことを応援してくれる
ふぁみたんがいるから皆さんの為に
ステージに立てます。

考えてアドバイスくれたり
ありがとうございます。

ここぞ！とばかりに
自分の持ち味を出して
選抜にくいこめるように
自分を磨いておきたいです。
 
悔しくないと言ったら嘘になるけど
私はもっと、やれることがあるはずだし
頑張れるはずです。
 
 
 
 
 
これからもこんな私ですが
よろしくお願いします。
 
 
 
 
 
 
 
新しい自分に。

センター本当におめでとう

しれないけど、シングルの曲が
凄く楽しみです♡♡♡
一緒に歌えないのはとっても残念
だけど、沢山その分きいたり
応援したいと思います
 
まいまいふぁいと♡♡♡
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
