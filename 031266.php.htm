2016/0327Sun斉藤優里今日は桜の日らしい です

 
 
 
 
 
 
てぃーちゃん遅くなっちゃたけど
お誕生日おめでとう\( ¨̮ )/♡
 
 
 
 
てぃーちゃんといつから
仲良くなったのかな？
って考えてみたけど
全然分からなかったよ...笑い
 
 
 
でも、いつのまにか話が合うな〜とか
テンション感が似てるな〜とかで
こんなに仲良くなったのかなって思う
 
 
 
 
18歳ながらも考えはかなり
大人だし、尊敬してます
でもたまに頼りにされることが
あったりすると嬉しくなるし
まだまだやっぱり18歳なんだなって
思うと凄く可愛いくなっちゃう( ´･ᴗ･` )
 
 
 
 
私の携帯がグアムで壊れた時も
1時間ぐらい私とスタッフさんで
格闘して諦めかけた時に
てぃーちゃんがいじってくれたら
すぐ直った時は本当に凄い子だと
おもった!!!!電子機器に強い子!!!!笑い
 
 
 
 
今年も旅行行ったり
色んなカフェ巡ったりしたいな♡
てぃーちゃんといるとボケとツッコミが
沢山あるから本当に楽しいよ
これからもずっとずっとよろしくね？ 
 
 
 
まだまだ色んなことに挑戦して欲しいし
私はそんなてぃーちゃんを
1番近くで応援したいと思うので
良いとこを伸ばしていってね♡♡♡♡
 
 
 
 
↑大好きloveたんまる◎
 
 



告知やで(♡˙ᵕ˙♡)告知告知〜

・3/30 月刊エンタメ NOGIMANIA
 
大好きなものについて語るコーナーですが
今回「モンハン」についてお喋り
ただ、ライターの方がモンハンを
やられてなかったということで
沢山魅力を伝えたつもりですが
どうだったかな？( ´;o;` )
 
 
沢山伝わってるといいんだけどな〜
あんな楽しいゲームないもん...♡♡♡
 
 
 
あ！でもゲームといえば乃木恋が
配信されてますね( ´›ω‹｀)
私のレアカードはどんな写真なのか
まだ分からない...気になるどの写真なの？
ゲット出来た方はいますか？
 
 
 
 
・3/31 BUBKA
 
こちらは、ちまと2人ではじめての
取材でした( ´･ᴗ･` )
嬉しい企画でした、ただ文章にするのは
ちょっと難しいから...ても、とりあえず！
大好きなちまについて沢山語れたので
中身が楽しみです♡♡♡♡♡♡♡♡♡
 
 
写真も撮って頂いのでぜひ
チェックお願いしますね？
