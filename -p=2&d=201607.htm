2016/0711Mon斉藤優里辛いチキン食べるよ うひゃあ

 
 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
わーーーーお家着いて昨日も
半身浴して(バスソルト混入済)
ジャスミン茶置きながら
ブログ書いてるよ◎
 
 
↑やっぱりお風呂が１番集中して書ける（笑）
 
 
 
 
 
 
 
 
昨日の握手会はいつもとは違くて
レーンの装飾が派手だったり...
綺麗なお花が飾ってあったり...
沢山の方からのお祝いメッセージを
頂いた1日になりました！！！
 
 
 
 
生誕委員の皆さん、そして
斉藤優里の生誕祭に
少しでも携わって下さった皆さん
忙しい中、大切な時間を費やして
くれてありがとうございました
 
 
 
 
 
 
 
一昨年よりも去年、
去年よりも今年！
大勢の方にそして大きな規模で
お誕生日お祝いして頂いています
 
 
 
 
 
 
乃木坂自体が大きくなっていっているので
有り難いことに握手会会場に沢山の方が
来てくれていることが多くなってきているけど、
貢献出来ているかな？って考えた時、
微々たる力しかないのも実感しています
 
 
私に出来ることってなんだろう？
私にしか出来ないことってなんだろう？
他の誰でもなくて、
斉藤優里にしか出来ないこと...
 
 
 
 
 
 
もっと必要とされるように
思ってるだけでは伝わらないので
自分から発信していきたいな
 
 
 
そんな風に新たな決意を感じれた
生誕祭でもありました！！！！！
 
 
 
 
 
 
 
 
生誕記念で作って頂いたプレゼントの
写真集...私の夢の中の１つでもあり、
こんな形で叶えて頂いて本当に
嬉しかったです( ˙˘˙ )
 
 
本当に本当に嬉しくて
お母さんに自慢しちゃいました（笑）
 
 
 
 
 
 
でも、ちゃんと値段を付けて
「乃木坂46の斉藤優里」
としていつか出版してもらえるまで
諦めません！！！
 
だからこそ、この1冊目が
貴重だし宝物になりました
ありがとう◎
 
 
 
 
 
↑ライブもだけどお花いつもありがとう
大好きなピンクばかりで幸せです
 
 
 
 
 
 
 
 
 
 
↑お手紙を読んでくれたのは
れかたん☆全く予想していなかったから
本当に嬉しかったよ( ´･ᴗ･` )
もらったお手紙大切にするね〜
 
 
それと犬会幹事になったので沢山招集かけるからね
 
 
あのね！みなさんの前で
「明後日に集まりたいと思います！」
って言ったと思うんだけど
本当にね皆に連絡したんだ（笑）
 
来れる方だけで開催出来るといいな♡
思い立ったら即行動のO型です（笑）
 
 
 
 
 
 
 
 
 
素敵な1日にしてくれて
ありがとうございました( ˊᵕˋ )
 
 
 
 
いつも個握で予告無く
イベントもののコスプレを
しちゃって見れなかったよ(´._.｀)
って事が今まで沢山あったので
今度からブログ、モバメで告知
していきたいなと思いました！
 
 
 
こっちのが「あー悔しい！」とか
なんて事も少なくなりますよね？
それではなんとなくだけど
一応イベントものの衣装予定を
買いちゃいますね♡♡♡♡♡♡
 
 
 
 
 
 
9/25パシフィコ横浜...浴衣
10/2京都パルスプラザ...パジャマ
10/29幕張メッセ...ハロウィンコスプレ
を予定しています( ˙˘˙ )
 
 
 
 
 
これからも色んなことを
書いてくブログにするから
チェックしてね？♡♡♡♡
 
 
 
 
 
それでは今日のおに魂頑張ってくるね！
聴いてほしいなゲストは「しずる」さん
夜ご飯はチキン食べるよチキン♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

今回のアニメも、もう何本か
第1話ははじまってたりするよね
この前の握手会で「今期何のアニメ見る？」
って聞かれてたんだけどその時は
チェック出来てなくてやっと、
最近チェック出来ました◎
 
 
 
 
 
毎週録画したのはこの
ライナップ\( ¨̮ )/
 
 
・ももくり
・Rewrite
・初恋モンスター
・orange
・ソードアート・オンライン
・Fate/kaleid liner 
プリズマ☆イリヤ ドライ！
・ダンガンロンパ 未来編 絶望編
・クオリディア・コード
 
 
 
 
 
 
ソードアート・オンラインは
やっぱり外せなくて毎回
再放送してる時は見ちゃうの
 
 
それと、ダンガンロンパは外せない！
未来編 絶望編両方とも放送なんて
こんなに嬉しいものもないよ(´·_·｀)♡
 
 
 
 
 
 
 
 
まだ見れてないものもあるんだけど
「ももくり」はとってもきゅんきゅん
で癒された〜\( ¨̮ )/♡わわわわわわ//
こんな笑みをこぼしたながら見たのは
久しぶりだよ...（笑）
私も雪ちゃんと一緒でももくんが
可愛すぎてにやけちゃう♡♡♡♡
 
アニメを見てから興味が出てきたので
漫画も読んでみようかなって
思ってます、ふぁみたんの中で
読んだことある方いますか？
 
 
 
 
 
ネタバレしない程度で良いから
ぜひおすすめして欲しい（笑）
↑ネタバレはだめだよ、本当に（笑）
買うか迷ってるんだから٩( ö̆ ) و

「ももくり」私の中で大ヒットおおおお♡♡♡








あ！ちなみにドラマは
・好きな人がいること
・せいせいするほど、愛してる
・そして誰もいなくなった
・闇金ウシジマくん
 
 
 
を毎週録画にしました！！！！！
私のデッキはそろそろ
容量オーバーしそうで怖いの(´._.｀)
どのぐらいまでいけるんだ？
どのくらいまでためられるんだ？
 
 
 
前それで壊れちゃって本体交換しちゃって
録画してたのが全部なくなっちゃたこと
あるから恐ろしいよ〜それだけは嫌だよ〜
避けたいよ〜ううう(´._.｀)
 
 







ふーーー沢山ドラマ、アニメについて書いちゃったよ◎
ちょっとそろそろ半身浴明日の為にしてたんだけど
限界が来たのでそろそろ出ようかな
ブログ書くのにお風呂って本当最適なんだ！！！！


飲み物も持ってて良い汗かきながら
考え事しながら書き込める
ブログを書く時１番多いのはお風呂かもしれない（笑）




半身浴のお供は入浴剤の「塩」を
毎回いれてます！！！！
発汗作用！！！






とか、書いている内に
本格的にリミットが来たので
そろそろバイバイしたいと思います♡♡♡
 
 

 
 
 
明日の握手会では
1部終了後 真洋
3部終了後 北野
 
そして2部終了後には私の
生誕祭を開いてくれるみたいです！！！
 
 
お時間ある方はぜひ見に来てね？♡
 
 
 
 
 
 
 
あ！これからパックしよう☆☆☆ 
もちもちにするのもちもちもちもちもちもち
もちもちもちもちもちもちも...ち







最後になっちゃたけど
真木子 SKE48 20th single 「金の愛、銀の愛」
の選抜入り本当におめでとう！！！！
 
沢山、発売されたら曲聴くね◎
これからも良い刺激をもらえたり、
与えたり出来る関係でいたいな\( ¨̮ )/♡
 
自分のことのように嬉しいです！

伸びたんだね♡♡♡←違う。

凄く難しかった(´._.｀)

もっと長い時間色んなことかたかったな

またお会い出来るといいですね
勿論、とっても顔馴染みの方も
いらっしゃてラッキーなテーブルでした♡
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
