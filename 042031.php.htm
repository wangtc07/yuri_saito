2017/1205Tue斉藤優里飴ちゃん

 
 
 
 
 
 
昨日は、有線大賞に出演させて頂きました☆
はじめて「インフルエンサー」で
ミリオンを達成して、有線音楽優秀賞
という素敵な賞も頂きました( ˊᵕˋ )
 
 
 
 
まだまだ未熟者の私達にとっては
大きすぎる賞ですがこの頂いた賞に
恥じぬようこれからも素敵なグループに
していきたいです！！！
又、私個人としては2017年のシングル
1発目であり選抜復帰曲でもある
インフルエンサーがこんな沢山の方に
愛して頂けることが本当に嬉しいのと
選抜メンバーとして沢山披露出来る
機会があるのがとても嬉しいです！
 
 
 
 
他にも年末特番が沢山あるので
どの曲を披露するかは分からないですが
活躍出来る機会が沢山あるといいな〜
 
 
 
 
それとね、皆がいる機会が本当に長くて
めっちゃメンバー好き♡
ますます好きなの♡どうしよう〜
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
そして昨日のTHE 魂
有線大賞の為お休みさせて頂きました
かりんちゃんとちーちゃん本当にありがとう(TT)
楽しんでもっとTHE魂のこと
好きになってくれたら嬉しいな♡♡♡
 
 
 
 
11/20のゲストさんは「もう中学生」さん
 
 
 
墓場女子高生以来にお会いできました！
お菓子をプレゼントしたお話を
覚えて下さっていて本当に嬉しかったです
 
もう中さんのまっまり雰囲気本当に
大好きですこれからもよろしくお願いします☆
 
 
 
 
 
そして、11/27のゲストさんは
「水瀬いのり」さん
 
 
めちゃくちゃラブリーさん♡
放送中全く言えなかったのですが
徒然チルドレンのアニメを見ていたので
少し興奮していました（笑）
 
高野さん♡♡♡
 
 
またぜひ遊びに来てほしいです( ¨̮ )
 
 
 
 
 
 
♡告知♡


・12/6 FNS歌謡祭
・12/8 乃木のの 西日本放送
・12/9 BOMB TGC広島イベント
・12/10 シブヤノオト
 
 
 
何か何か、盛り沢山の内容に
なってしまいました（笑）
近いものから書きました（笑）
他にはMステ、スーパーライブとかも
発表になってるけど少し先なので...（笑）
これからもっと増えていくと思うので
皆さんも大変だと思うけどぜひ
見て欲しいな( ˙O˙ )
 
 
ちなみにBOMBはみなみとちまと3人で
グラビアをやらせて頂きました♡
大人になったりヒラヒラだったり...
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
