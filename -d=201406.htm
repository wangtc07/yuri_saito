2014/0630Mon斉藤優里圧縮袋でしゅっ。

 
 
こんばんたーん( •ㅈ• )
たった今おに魂が終わって
帰ってる途中ですっ♪
 
いやー今日は朝から生写真撮影
してお疲れ様でしたな。
 
そして今日はmailで言ってたとおり
大原櫻子さんがゲストさんに来て
くださいましたっପ(੭•͈ω•͈)੭*
 
以前イベントでお会い
したことがあったので2度め
ましてでとても話しやすかったな✩
 
話していて、とても向上心のある方
なんだと知り更に素敵に見えました！
「頑張ったっていいんじゃない」
本当に素敵な曲で本当に
聴き入ってるんですよね♡♡♡
 
 
またどこかでお会いしたいな〜
 
 
 
 
 
...ってぶろぐ書きつつも実は
焦ってます。笑い
 
なぜなら、明日からのフランス
準備が...ドタバタ状態なんです。笑い
 
 
うわおうわお(･･;)
 
 
 
 
 
でね、ふぁみたんに残念な
お知らせなの。泣
 
フランスに行ったら時差が凄くて
ぶろぐがあげられないとのこと
なのでフランス行ってる間は
メンバー統一でぶろぐを更新
出来なくなりました(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
実はモバメも同様なので
ごめんなさい。泣
 
 
本当は早くふぁみたんに
どこどこ行ったよ！とか
同じ景色見て感動を共有
したりしたかったんだけどな☁︎
悲しいーーーーーーーーっ。
 
 
 
だから、ブログもモバメも
更新されなくてもびっくり
したりしないでね？
 
帰ってきたらたくさん載せたり
ご報告するからね♡♡♡♡♡
よく言えばそれまでの
おあずけだな♡♡笑い
 
 
 
それじゃあ、日本よさらば
メルシーーーーーーーー♡♡笑い
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2014/0618Wed斉藤優里何だか後半、眼鏡に ついて熱い

 
 
こんばんたーん( •ㅈ• )
 
ホームページとかね見てたら
個人PVの情報がのってる〜！
と驚きっ！
 
早速あたしもURLからとんで
自分の個人PVを鑑賞しました。笑い
普段自分が出てるのはなんだか
恥ずかしくて見れないのですが
今回の監督さんが「山岸聖太」さん！
前回のせいたんのを撮って
くださった方です。
 
 
せいたんのを見ていてシュールな
感じがあたしのツボにはまり
とっても素敵な作品だな〜☆
と思っていたら今回あたしの
個人PVをとってくださり、
 
撮影中もどんな風に出来上がるのか
物凄くわくわくしていた...
 
だからね、早く見たくて
仕方がなかったの〜♥︎♥︎
 
 
 
予告編だけでも楽しかった〜
ちなみにナレーション全部
早口でずっと録音しぱってで
かんだら「はい、やり直し〜！」
と山岸さんのsっぷりに
うろたえながらも...頑張りました。笑い
 
 
あたし、だって滑舌悪いしよくかむし。ん？ラジオのレギュラーやってるのに。
笑い
 
 
 
というような感じで撮影をめちゃめちゃ
楽しみながらでした(๑¯◡¯๑) にんまり
 
代々木公園でお布団ひいて寝たり
 
バッて勢いよく起きてみたり
 
バナナを持って「お兄ちゃん」と
カメラに向かって言ってみたり
 
 
色んな方に見てもらえる作品だと
思うのであたしの個人PVたくさん
ふぁみたんみーてーねー٩꒰｡•◡•｡꒱۶
 
6/24 月刊ザテレビジョン
6/24 TVfan にどっちも
りな さゆにゃん 真夏と出演
するので「夏のFree&Easy」について
語ったりしているので良かったら
チェックしてみてください꒰˶ु❛ั‧̫❛ั˵ू⑅꒱
 
 
撮影雰囲気をこんな感じに
わちゃわちゃといつも通り
あたしはちょっかい出したり...笑い
 
あれれ？なんだかこの写真
めちゃめちゃ真面目に撮ってる。笑い
 
 
というか！今ぶろぐ書いていて
思ったけど「夏のFree&Easy」って
FとE大文字なのね\(°Д° )/
危ない危ない。笑い
 
 
 
それでは月曜日の話になってしまいますがゲストさんで来てくださった
高橋直純さんとバカボンさんと
写真を撮って頂きました！
 
高橋さんはきっくんさんと
ハネーズというグループを
組むほど仲良しなので
おに魂ファミリーの感じが
すごく濃くて濃くて終始
笑いっぱなしでした〜( •ㅈ• )
 
ハネーズでの新曲楽しみにしてます♡笑い
 
 
 
 
 
 
 
あ...これそうそう！
気づいた？ねぇねぇ気づいた？
まだまだ目が治らないので
MJ終わりにすぐ眼鏡に変えました！
 
本当はコンタクト...
ドクターストップがかかっていて
だめなのですが(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
ごめんよ、あたしの目玉よ。
 
MJは眼鏡で出る訳にはいかない！
なぜならあたしの眼鏡はとんだ
そこいらのお洒落眼鏡でなくて
朝起きた瞬間からこいつがいないと
全く生活が出来ないあたしの相方
なのだからあああああああああ！
 
(※斉藤優里 右目の視力
0.01以下である)
 
だからこの眼鏡がないと生活が
出来ない程の視力を補うもの
なのだから相当の度数なのである。
 
 
(※一般的に眼鏡は視力が悪ければ悪い程目が小さく見えてしまうものである)
 
なのであたしは...あたしは...
眼鏡収録なんてしてしまっ日には
変顔いじょうの顔を晒してしまうので
写真など眼鏡をかけているものは
全て目をつむっているのである。
 
 
 
申し訳ない...(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
全ての眼鏡たんたんを見たがっていた
ふぁみたんへ。目をつむっているので
我慢してはくれないか。
 
 
 
 
 
 
許せ皆の者。

2014/0616Mon斉藤優里豆乳ブーム再来っ！

 
 
こんにちたーん( •ㅈ• )
 
昨日は「16人のプリンシパル」
の最終公演〜っ！つまり22公演め！
 
いや〜さすがに22公演は
長かったですね〜
あっ！でもふぁみたんとかは
どう感じるんだろう？
短いっ！とか感じたりするのかな？
 
 
でも、感じ方なんて人それぞれ
ですもんね(´･ω･`)
 
あたしの「16人のプリンシパル」
のイメージって今までは演技
出来たりする人が選ばれる。
って思うことが多かったのです。
苦手意識が強くて...
ですが今回のテーマがコメディエンヌ
ときいてからは聞いてからは
「あ、今回なら...」と自分の中で
何か変わった気がしました。
 
 
 
そして監督が福田雄一さん
だったので稽古している時も
凄く楽しくて本当に今回の
プリンシパルをしたおかげで
苦手意識が少しずつなくなって
いきましたっ！
 
あたしにとってこれは本当に
大きいことで(´;ω;`)
福田さん達のチームと公演を
することが出来て本当に良かったです。
 
 
そして今回あたしがした役は
・マキア 
・パム
・キャサリン 
・エルザ
・ロザリオ
・クリスティーヌ
 
〜〜アンサンブル〜〜
・メグ
・侍女1
・侍女3×2
 
 
見て頂いたら分かるのですが
去年はあんまり色んな役に
挑戦したりしてませんでした。
 
 
今回はどの役もそれぞれ
面白みがあったので本当に
色んな役に立候補しました。
 
あと、ポリン姫 ベル ルイーダ エステル
で10役制覇だったのに(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
悔しいーーーーーーっ！
 
特に悪役を1回も出来なかった
ことに後悔です。笑い
 
 
 
 
 
でもでもふぁみたんの投票あって
どの役も入れたのですが本当に
感謝しています！
ありがとうございました♡
 
 
 
 
 
そして写真はないのですが
おに魂チームがかけつけて
くれた日もありました！
本当にたくさんの方の力が
あって成功出来たプリンシパルです！
 
 
また是非とも一緒に舞台
させて頂きたいです♡♡♡♡♡♡
 
 
 
 
 
あ\(°Д° )/ちなみにリクエストの
多かった眼鏡の写メはメールで
送っちゃた〜♡♡♡♡♡♡♡♡笑い
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
