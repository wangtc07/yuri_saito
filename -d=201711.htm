2017/1117Fri斉藤優里甘酒

 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
今年も乃木坂46の紅白出場が決まりました
紅白の公式サイトの出演者一覧のとこの
横に書いてある(3)という数字がとっても
嬉しく思います
 
 
 
 
 
これからもこの数字がどんどん
伸び続けていきますように☆
とか、まだ今年の紅白終わってすらいないのに
こういう風に思ってしまいました
 
 
 
またあの年末のドタバタとした
いそがしさを味わえると思うと
嬉しいです( ˙˘˙ )
お仕事あるって有難いことですね〜
 
 
 
 
そして、もう一つ嬉しかったのが
TWICEさんも紅白出場！！！！
本当におめでとうございます♡
 
私にとっては×2倍の嬉しさでした
これからもずっと応援しています( ¨̮ )
何の曲歌うのかな？♡
 
 
 
 
 
 
 
 
この前久しぶりにスイカで
パーティーをしてきました☆
純奈は稽古があったので残念ながら不参加
 
 
その日の献立〜
・はらこ飯
・磯辺揚げ
・トマトとチーズのカプレーゼ
・ゴマ団子
 
 
 
 
 
 
.....と、メニューはごちゃごちゃですが
どれもこれも皆の食べたいものを
合わせたらこうなりました（笑）
 
 
ゴマ団子、沢山作ろうと思ってあんこ
買ったら思ったより使わないのね（笑）
白玉粉は使い切ったのに
あんこだけ残っちゃた（笑）
新しい発見でした( ˙O˙ )
 
 
 
 
 
 
ななん家でスプラトゥーン2やらせて
もらったんだけどあれめっちゃ難しいのね
YouTuberさんの実況とか毎日のように
見てるから出来るつもりでいたら
全然出来ないのね（笑）
 
(※Switch持ってないけど毎日スプラトゥーン2の実況見てる
そろそろ買いなよとか言われるけど今やると
強い人沢山いて、負けちゃうからちょっと(TT))





マリオオデッセイして大興奮したり
個人的にはバイクの乗りこなしが
楽しかったことや、プレイしていない
皆までうずうずしてたり
お腹痛くなるほど笑いました（笑）
 
 
そんな毎日が楽しいスイカです（笑）

 
 
 
 
 
それでは、また更新する時まで〜
 
 
甘酒大好き♡いつもは鍋で煮て
作ってたんだけどミルク割り
っていうのをはじめてのんだの♡
物凄く好みでした甘酒ミルク♡♡♡
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2017/1114Tue斉藤優里水色

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )

限定100名の方との楽しい時間
しっかりと企画案も出せて頂きました！！！
 
 
 
 
「ソロライブ」をさせて頂いたのですが
いかがだったでしょうか？
やっぱり全体ライブやアンダーライブでも
なかなかソロで歌うこともないので（笑）

お歌大好きとしてはライブを
やってみたかったのでこの機会を
使わせて頂きました♡♡♡

普段、乃木坂のファンの皆さんだったら
きっとお耳苦しいかと思いますが
私のファンの方なら許してくれるかなとも
思い甘えちゃいました( ˙˘˙ )
 
 
 
 
とっても楽しんで盛り上がれた
そんな1日に出来ていたら
大成功かなと思います！！！
 
 
 
 
 
 
 
また、こういうソロライブが
出来る機会があったら開催
したいなと思います（笑）
 
 
 
その時もこれに懲りずにぜひ遊びに来てね（笑）
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
昨日のゲストはTHE 魂に
なってからははじめて！！！
ですがおに魂含めると2度目ましての
「椎名慶治」さん( ¨̮ )
 
 
 
 
 
 
こらでもかーーっ！ってくらい
沢山お喋りさせて頂いたのですが
帰り際に「あーー全然喋れなかった！」
って仰ていて椎名さんが恐ろしかったです（笑）
 
 
また、ぜひ遊びに来てください！
その時もTHE 魂で
喋り倒していって欲しいです（笑）
 
あ〜凄い楽しかったな♡♡♡♡♡

2017/1112Sun斉藤優里ドキドキ

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
先日の7.8日におこなわれた
東京ドーム公演来て下さった皆さん
ありがとうございました☆
 
 
 
それに有難いことに見切れ席や
バックステージ席など全て完売という状況...
 
 
 
まだまだ、握手会などで「行きたかったな〜」
なんて声を頂いたりしました
皆が来たいって言ってくれるのが本当に
嬉しくてそんな気持ちを全部受け取って
もっともと色んな所でライブ出来たらな
とか東京ドームでもうちょい出来たらな
とか思いました(TT)
 
 
 
 
これからも乃木坂はきっと
ライブをやると思うので
悲しい気持ちでいないで下さい
また、きっと会える日が絶対来るから
その時を楽しみに待っていてほしいな
 
 
 
 
個人的にはアンダーブロックが
今まで1番曲数が多かったことが
なんだか胸にくるものがありました
卒業発表をしためーちゃん 万理華が
より輝いて見えましたね
 
 
 
 
 
何よりも今回の真夏の全国ツアーFINALは
選抜メンバーとしても活躍出来たので
出演パートがとっても多くて
誰よりもドームのステージを沢山
踏んでましたきっと！！！！！！
 
 
 
このステージに立てたこと
一つの目標でもあったけど
まだまだこのグループなら
色んな所に行けると私は信じています！
 
 
でも、ファンの人がいなかったら
ドームというステージに立つことは
出来なかったと思います
いつもそばにいてくれてありがとう
 
 
 
 
 
来れなかった方の分まで
もっと大きくそして素敵な場所へ皆さんを
お連れすることを約束します♡♡♡♡♡♡
 
 
 
 
 
 
 
 
 
それでは今日の乃木恋イベント
来れる方は一緒に楽しみましょうね〜
サイリウムとタオル忘れずにですよ（笑）
 
持ってなくても声を大きく出して
頂けたら（笑）
 
いや、とりあえず楽しんでくれたら
私はそれでいいや♡♡♡♡♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
