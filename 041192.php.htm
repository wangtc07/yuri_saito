2017/0928Thu斉藤優里Cookie☆☆☆

 
 
 
 
 
 
MVがもう公開されていますが
皆さんチェック出来ていますか？
 
 
 
 
 
大迫力の薙刀をしている中
女子高生の観客役として参加しています（笑）
映画でも女子高生！！！
MVでも女子高生！！！
なんとジャケ写も女子高生！！！
 
 
 
 
 
...とその近辺のお仕事はずっと
制服を着ていました☆（笑）
自分ではそろそろあれかなと
思ってましたがふぁみたんからは
まだいけるよ！との声がめっちゃ
多くてびっくりしました（笑）
 
皆の、優しさなのかもしれないけど（笑）
 
 
 
 
うん！これは優しさだねきっと（笑）
 
 
 
 
 
 
ポイントは久しぶりのリップシンクが
あってとっても嬉しかったですし
まいちゅんとぎゅっ♡してるとこ( ˙˘˙ )
目を凝らして瞬きしないで見てもらえたら♡
 
 
 
 
 
それとね、9/23発売のB.L.T.は
チェックしてありますか？
欅ちゃんが表紙の号に真夏 生駒ちゃん
みなみ ちま 小百合 万理華 まいちゅん
かな 北野で載っているのでチェック
お願いします( ˙˘˙ )
 
 
 
 
 
 
(※くまったんにはじめて会えたのが
とっても嬉しかった( 　˙³˙　 )
やはり可愛かった愛らしい)
 
 
 
 
 
(※みなみが撮ってくれたんだけど
くまったんと遊んでたらでーーん！
って倒れてきて潰されちゃった(TT)
なんなんだ、この子（笑）)




テーブルの上に置いてある料理は
作ってもらいとってもお洒落な味でした
終わってからもそれをお昼ご飯変わりに
するくらい黙々と食べました（笑）
 
 
そんなことも思いながら
ぜひ見てもらえたらな♡

 
 
 
 
 
 
握手会では毎回、お花の凄い量
ありがとうございます！！！
いつもレーンがピンクやラブリーなお花で
賑わっていて本当に嬉しいです♡♡♡♡♡
 
 
 
 
 
 
握手会もねいつも私のこと
楽しませてくれてありがとう！
沢山お話をする為にネタを
考えてきてくれているんだなって
思うと本当にぐっとくるよ♡♡♡
 
 
 
 
これからもずっと皆と仲良しがいいな(ˊᵕˋ)
 
 
 
 
 
 
 
 
告知♡

・10/8 23:00〜 ガクたび！
香川県の琴平高校に行ってきました☆
生徒さんわいわいいぇいいぇいとっても
可愛い子ちゃん( ˊᵕˋ )
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
