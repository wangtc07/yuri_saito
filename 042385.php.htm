2017/1223Sat斉藤優里好きです、もう本当に好きです

 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
昨日のMステは皆さん、チェック出来ましたか？
19:00〜23:10までという長丁場の中、
タモリさんや弘中さんは勿論
アーティストの方々お疲れ様でした☆
 
 
 
 
 
個人的には「インフルエンサー」を
披露する機会がとても最近の歌番組では
多く嬉しいです、披露する前に振り固めをして
挑めたので今までで１番と言っていいほど
振りが揃っていたのかなって本番の動画を
見ながら喋っていました( ¨̮ )
 
 
 
 
 
 
あとね、万理華と久しぶりに踊る
インフルは楽しかったな〜
特別ポジションが近いとかではないけど
やっぱり万理華だと安心する！！！！！
ファンの人もきっとびっくりだったよね？
 
 
 
 
 
動画を見ている時にここのポジション
私が居なくなったら優里やりなよ
って凄い軽いノリで万理華に言われた時は
めちゃくちゃびびりました（笑）
そして万理華じゃないとダメだよっと言って
丁重にお断りさせて頂きました（笑）
 
 
 
そんなことも言い合えるようになったのも
犬めんと一緒に過ごす時間があったからかな
と思うとあの時間には沢山の意味が
あったと感じ全てが愛おしくなります
 
 
万理華、本当にお疲れ様！！！
楽しそうに踊るカッコイイ姿は忘れないよ
それとすぐ犬会しようね（笑）
 
 
 
 
 
 
 
 
前回のブログにも書いていた
嬉しいことなのですが、TWICEさんに
ご挨拶させて頂きました♡♡♡♡♡♡
 
 
 
 
 
 
 
 
この写真撮る時に乃木坂のメンバーも
せっかくだからって前に行きなって
言ってくれたり、
ナヨンさんとチェヨンさんが
ここに来な？と間に座らせてくれたり
何もかもが嬉しいことばかりで(´･ ･`)♡
 
 
 
乃木坂のCDをお渡しさせて頂いたので
聴いてくれてるといいな(´･ ･`)
 
 
 
これからもずっと応援させて頂きたいです
ますます大好きになりました♡♡♡♡♡
可愛くて可愛くてたまらない(´･ ･`)
 
 
日本を沢山、堪能してくれたら嬉しいな〜
 
 
 
 
 
胸がいっぱいの1日でした
 
 
 
JELLY JELLY聞きながら帰ります
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
