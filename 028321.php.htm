2015/1215Tue斉藤優里新商品って手に取り たくなるよね

 
 
 
さて！ここで問題です。ででん。
これは何の雑誌のオフショットでしょうか？
難しいかな？♡♡♡
 
 
今流行りのおフェロ♡♡♡
にしてもらいました(ˊ•̤ω•̤ˋ)るん
 
 
 
 
 
 
 
 
 
 
 
答えはもう少し画面をスクロール↓↓
それでは皆さんさんが考えている間に
最近のお話を....笑い
 
 
 
 
 
 
 
 
冬になるとなんだか
赤が着たくなる(ˊ•̤ω•̤ˋ)
夏にそんなこと全然思わないのにな〜
なんでだろう？
クリスマスマジックなのかな？
 
 
 
 
 
 
 
 
 
最近、赤のバック買ったり⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
        赤のパンプス買ったり⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾

赤があんまり好きじゃなかったんだけど
買う物ほとんど赤なの◎
 
 
 
 
 
 
 
 
 
何か変わってきたのかな？
でも、相変わらずピンクも大好きなんだ♡
 
 
 
 
 
 
 
 
 
答えはもう分かったかな？
正解発表の時間だよ、
12/15 PF(ポーカーフェイス)の発売日(ˊ•̤ω•̤ˋ)
凄い可愛くて雰囲気が大好きな感じの
撮影だったのでとっても楽しかったです！
一番上のおフェロの写真もこの時の
オフショットだったんです！
 
 
メイクも甘い感じなので本当に
私のしたいようなお仕事そのもの
なので本当に嬉しかったです♡♡♡
 
 
 
 
 
 
絶対見て欲しいな〜特にふぁみたんには！
皆可愛いお洋服きてて本当に
可愛かったな〜♡♡♡
 
 
 
 
 
 
 
 
 
ねーねー絶対見てね？絶対見てね？
よろしくお願いします♡♡♡♡
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
そして昨日はおに魂感謝祭
ルーリーさんが来てくれたのは本当に
嬉しかった〜(ˊ•̤ω•̤ˋ)
 
 
滑舌が良くなる方法を教えてもらったり
勉強になることが多かったです！
また絶対遊びに来てくだい
 
 
 
 
 
ただただ( ･᷄ὢ･᷅ )
わたしは不服だ( ･᷄ὢ･᷅ )
今回の罰ゲームがビリビリだったこと( ･᷄ὢ･᷅ )
 
 
 
 
 
 
あれ、本当に見た目以上に痛いんです....
ただ罰ゲームから逃げることはやっぱり
自分でも駄目だなって思いました。
でも痛いんです( ･᷄ὢ･᷅ )本当に。
 
 
 
 
 
もうお願いだから小野ちゃん特製
センブリ茶なら何杯でも飲むから
それに変えて欲しい。泣
お願いします。 神様 仏様 おに魂様。
 
 
 
 
 
 
センブリ茶時代のおに魂に戻って〜
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
このあと、アンダラのことに触れるか
迷ったけどまだふぁみたんのこと
焦らしておきたいからあえて
何にも書かないもん♡♡♡
 
 
来てからのお楽しみなんだもん♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
