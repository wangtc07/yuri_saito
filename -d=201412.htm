2014/1228Sun斉藤優里めーたむ八つ橋くれ たロミロミ〜

 
 
こんばんたーん( •ㅈ• )
 
握手会で何人の方とお約束した通り
ブログ更新しましたよ〜♡
 
約束した方は見てくれてるかな？
 
 
2014年締めの握手会なので
たくさんの方が来てくださったな
中でも「やっぱり締めの握手は
ゆったんで(ˊ•̤ω•̤ˋ)」って言って
くれるのが本当に嬉しいんだよね〜
 
 
元気をたくさん与えられてるのかな？
って実感したりするからさ。
 
 
2014年もふぁみたんにはたくさん
お世話になりました〜
あたしのお話をたくさん
聞いてくださってありがとう♡
 
こんなにも思いをたくさんの方に
聞いてもらえて幸せいっぱいです！
 
 
それでは1部のコーデ\( ˆωˆ )/
 
大好きなsnidel！でも1部の時に
ハプニングがあったのーーーー。泣
握手会激しくしすぎて靴下についてる
飾りのポンポンが壊れたの。
 
右足だけボロッてとれた(´°ω°`)
うおおおおおおおおおお。
 
 
 
2部3部コーデ\( ˆωˆ )/
 
コスプレしてるつもりないのに
「何のコスプレ？」と聞かれることが
めっちゃ多かった...笑い
 
スタッフさんにはアナのコスプレ？
と聞かれ あるメンバーには
芸人の髭男爵のひぐち君さんなの？
と聞かれ...私服なのに〜( ･᷄ὢ･᷅ )む
 
その話をあみが聞いていて
ファンの方に言われたの
投票とっといてよ！と言われ
3部に心の中で投票が入ってくるの
待ってみた所、結果は...
 
 
ひぐち君 5票
アナ      1票
 
 
でした( ･᷄ὢ･᷅ )
ひぐち君さんが多い( ･᷄ὢ･᷅ )
 
 
でも、全然ふぁみたん優しいから
ちゃんと私服って理解してくれた方が
多くてあまりツッコまれずに済んだよ。
 
 
 
最近私の中で芸人の
バンビーノさんと8.6秒バズーカーさんがとっても熱いーーーーーーーー♡♡
 
 
めっちゃ好き〜
 
 
好きすぎてななとのホテルでも
ずっとネタ見てて帰りの新幹線でも
見てて楽屋でもななとずっとしてる！
 
 
両方のネタを完コピしたいの❥
今の夢はこれ！
 
本当に好きすぎてコスプレ
したいぐらいなの♡♡♡♡
 
お会いしたいな〜
生でネタ見してもらいたいな〜
 
 
帰りながらまたネタずっと
リピートしちゃおう♡♡♡♡♡
 
 
なな大好き〜( ･᷄ὢ･᷅ )
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2014/1227Sat斉藤優里北野にマスクしゃき ーん

 
こんばんたーーん( •ㅈ• )
 
朝から撮影して、収録に行って
明日はなんと京都☆
 
 
2014年最後の握手会は京都♡♡
 
来れる方は色んなお話しましょう？
例えば・2014年の感想とか
全然2014年締めに関係ないお話とか
本当に何でもOKなんだからね〜
 
 
ただ、きおつけてほしいのは
あたしいつもテンション高いので
心構えしとくとおすすめです。笑い
 
...なんなんだ。このおすすめの仕方。
 
 
声が大きいから圧にやられ
ちゃったりする方も多いから(´°ω°`)
楽しい握手会にしたいあまり
なんですよ....泣
 
ごめんね〜

なんて方まさかここにはいないよね？
え？
 
 
ちょっと...見なくていいかな？
なんて思ってたら斉藤怒ります。笑い
 
 
少しでもそんなふぁみたんに見て
もらいたいからOFF SHOT
載せちゃいます♡♡♡♡♡♡
 
実はこれ可愛いラプンツェルを
イメージして撮った写真なの
 
ユウリンツェルは紫のワンピース
じゃなくて白のワンピース。笑い

街におりてきて子供達が遊んで

イメージしてメイクさんが
作ってくれたの⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
もう、大満足♡
 
 
だからたくさんの、ふぁみたんに
見てもらいたいんだ〜
ということでぜひ！見たよ！って方は
明日の握手会とかで感想
きかせてください(ˊ•̤ω•̤ˋ)
 
 
 
それでは、そろそろビタミン入りの
ジュースのんでゆっくりしようかな〜
 
 
ちょこん。

2014/1225Thu斉藤優里最初に口にしたのは あんみつ

 
 
めりーくりすます(ˊ•̤ω•̤ˋ)
 
ふぁみたんはどんなクリスマス
過ごしてるのかな？
 
お肉食べた？ケーキ食べた？
 
 
クリスマス楽しいよね〜
 
イベントごとやっぱり大好きだな♡♡
 
 
 
 
でも、あたしはハロウィンのが好き♡
 
 
 
 
でもねそんな昨日は凄い幸せな1日に
なったんだ！実は前々からてぃすと
クリスマスデートしようね٩꒰৹৺৹ઌ꒱
って話しをしててデートブランを
前もって作ってみたり....
 
昨日は無事デートブラン大成功！
 
まずは行ってみたかった
ティム・バートン展に行ってきた꒰✩''ω`ૢ✩꒱
 
 
混んでたけど凄いこういう世界観が
大好きだから楽しくて楽しくて
ちょっとだけグロい感じのね。笑い
 
とくに2人でテンションあがったのが
ナイトメアー・ビフォア・クリスマスのコーナー。あのね、あたしずっとゼロっていうわんちゃんが好きなんだけど展示されていて物凄く可愛いかった〜
 
いやーーーあれは持って帰りたかった。笑い
 
 
お土産にポストカード買えた♢
激混みの中必死に手を伸ばした。笑い
 
写真じゃ見づらいかもなんだけど
あたしの上のポストカードは
サンタさんが間違えて子供に
悪いクマを渡しちゃって
子供が食べられちゃう。っていう
絵が書いてあるの❥❥
 
お気に入りなの〜\( ˆωˆ )/
 
 
その後はお買い物行ったんだけどね
優柔不断過ぎて凄い時間かかるの
でも、自分のことのように考えたりするの大好きだから全然苦じゃない。
 
だからメンバーのお買い物に
振り回されるの好きなんだ！
 
 
それで昨日は色んなイルミ見れたんだ
これでもかっていうぐらい見たな〜
街中に溢れてて終始、てぃすと
撮影会だった\( ˆωˆ )/にやにや
 
 
その後はあたしん家で
クリスマスパーティー
飾り付けてぃすが手伝ってくれたんだ！
 
カーテンに装飾物ぶらさげるのが
かなり苦労したけどね。笑い
背が小さくて届かなかった...
 
 
 
若干パジャマもクリスマス意識。笑い
今度のデートの場所までどこ
行くか決めたりしたんだ〜
てぃすと行きたい場所
たくさんあるーーー⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
 
 
 
それではふぁみたん今日も素敵な
クリスマスをお過ごしください〜\( ˆωˆ )/
