2016/0226Fri斉藤優里にゃんちゃんなの

 
 
 
 
 
 
 
 
こんにちたーん( •ㅈ• )
収録がはじまるギリギリまで
いつも前髪を固定している
たんたんだよ♡♡♡
 
 
 
待っている間に前髪の分け目が
崩れたりするのが嫌なの...
 
 
 
 
 
乃木坂の4ちゃいのお誕生日と
おに魂の日が被っていたのですが
なんとサプライズで鬼にゃんから
口ファンファーレ？でおめでとう？の歌を
歌ってもらいました( ´･ᴗ･` )
 
 
その時の画像がこちら
 
 
↑うわっ！鬼にゃんめちゃめちゃ
全体的に伸びてるし私なんとなくだけど
目つぶっているように見えるし
なんなんだ！何なんだこの画像！






っていう感じで楽しくやってる
おに魂なんですよそれにその日に
なったんもお祝いしてくれました
うわい、めっちゃ嬉しい♡♡♡
 
なったんもありがとう(ˊᵕˋ)
 
 
↑ちゃっかり 「46」ポーズ
なったんといつもすれ違いになっちゃうから
もっと仲良くなりたいな♡♡♡





ゲストはEDGE of LIFEの
お2人でした( ´›ω‹｀)

 
 
モノマネ大会ありがとうございました
萩尾君にいきなりモノマネを
ふられた時はめちゃめちゃ
びっくりしました( ´;o;` )
 
すみません。メンバーモノマネしか
出来なくて.....泣
 
今村君も、なかなかモノマネが
似てました！
...モノマネの話ばっかり。笑い
 
 
 
 
 
 
またぜひ、楽しいおに魂に
遊びに来てください\( ¨̮ )/♡
 
 
 
 
 
 
 
エンディングにて「乃木坂1の親近感」
を目指しているというお話をしましたが、
口にすれば実現するということを
信じていこうと思います♡♡♡
 
目指せ！「乃木坂1の親近感」
 
 
 
最近モバメでめちゃめちゃ連呼してる
にゃんちゃんはなんとなーく
可愛いかなって思って連呼してます
だからね、あんまり意味はないの。
 
 
えへ☆ にゃんちゃんなの。
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
