2016/0427Wed斉藤優里チェック出来ました か？

 
 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
4/24放送のシブヤノオトは
見て頂けましたか？
新番組になってから1発目の
放送によんでもらえました( ˙˘˙ )
なんて光栄なことでしょう...
 
 
 
 
放送日にアンダラをしていたから
ブログとかでふれずっだったのですが
チェック出来ましたか？
 
久しぶりの太陽ノック♡♡♡
 
 
 
 
 
 
 
一年前くらいのシングルになるのに
この曲を選んでもらえて、
びっくりしたのと素直に嬉しかったです
時期は違うし沢山ある曲の中から
太陽ノック、それとオリジナルメンバー
で披露できたのも嬉しかったな( ´･ᴗ･` )
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そして遅れてしまったけど
4/18おに魂感謝祭のゲストさんは
ゆるめるモ！からしふぉんさんとあのさん
が遊びに来てくれました( ´･ᴗ･` )
 
 
真面目担当のしふぉんさんが
凄くしっかりしてて、あのさんが
とっても可愛くてバランスが
とれててとっても素敵でした♡
 
またぜひ遊びに来て欲しいです〜
 
 
 
 
 
 
 
 
4/25のゲストさんは大石昌良さん
 
 
私がお会いするのは2回目ですね
この前約束した通りに
おに魂のジングルを本当に
作って頂けました( ´;o;` )
 
お忙しいのに本当にありがとうございます
 
 
月曜の朝6:00頃に完成したそうで
本当にありがとうございます！
 
 
 
 
 
また生演奏を目の前で見せて頂きましたが
２回目でもやはり手先を見てしまう...
耳でも楽しませてくれるし
目でも楽しませてくれる
なんて素敵なんでしょう♡
 
また、ぜひ遊びに来てください
 
 
 
 
 
 
 
 
 
そして、今日はじめて「乃木恋」の
CMを見ました( ´›ω‹｀)
もう、アプリがダウンロード
出来るみたいですね♡♡♡♡
 
 
しかも、ランキング見てみたら
2位でした皆さんのおかげですね
ありがとうございます
まだダウンロード出来てないよって方は
ぜひチェックしてみてね？♡♡♡♡♡♡
 
 
 
にゃんちゃんとぜひ恋愛はじめませんか？












たんたんたん ゆったんたん( •ㅈ• )
