2014/0831Sun斉藤優里てるてる坊主の効果 だよね

 
 
 
こんにちたーん( •ㅈ• )
 
 
昨日は真夏の全国ツアーラストは
明治神宮野球場での野外LIVE♦︎
本当にたっーーくさんの方に
起こし頂きました！
 
前日や朝方まで雨マークだったのに
本番になったら雨なんて降らず
綺麗な西日のさすなかでのLIVE
 
夏のFree&Easyで登場した時には
思わず、鳥肌がたってしまうほどの
光景でした。たくさんLIVEで
景色を見てきたけど明治神宮野球場
は本当に1番素敵で.....
これからもずっとあの景色
忘れられません(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
大阪公演からはじまってたくさんの
経験をさせて頂きました！
 
まずはトロッコにのってスタンド席の
皆さんにご挨拶に♡♡♡♡
ずっと乗ってみたかったし
やっぱり距離が近いと
少し照れてしまう。
 
が...何かそのドキドキ感が
たまらーーーーーーーん(ˊ•̤ω•̤ˋ)
やっぱりあたしは皆さんの近くで
パフォーマンスしたりしてるほうが
好きだな〜だからね？トロッコ
お気に入りになってしまった！
 
 
もう一つ、ラジオでの真夏の全国ツアーの見所は？と聞かれた時に
「普段のあたしからは想像
出来ないことをします！」と
お答えしたのですが実は
世界で一番孤独なLoverのゾンビver
の時にまりか かな 若月で側転を
してましたっ❣❣❣
 
よく、「ゆったんってスポーツ
出来なさそう✩✩✩」って
言われるのですがいがいと体
動かすのは得意なんですよ？
 
それに動かすの大好きだから
マラソンとか好き。
唐突に好きなものあげたけど。笑い
 
 
 
 
そんな楽しい真夏の全国ツアーの
お話で締めたかったのですが
ふぁみたんにお伝えしなければ
ならないことがあります☹
 
実は昨日の明治神宮野球場でのLIVE
ゲームのサインボール1000という
企画の時にあたしの不注意で
階段から転落してしまい
そこで右足を痛めてしまいました。
 
 
まだまだ色んな曲数に出演
する予定だったのですが、
無口なライオンと
僕が行かなきゃ誰が行くんだ？
 
13日の金曜日だけは
どうしても出たくて色んな方にお願いして。
この曲だけは皆が優里の曲だからと
言ってくれるのが凄い嬉しくて。
 
出させて頂けることになりました！
気持ちで届けたいと
思って精一杯の自分をぶつけました！
 
 
13日の金曜日おわりにハケて
いるときに右足を庇って
階段をおりていたら、
踏み外してしまい左足も
痛めてしまいました。
自分の鈍臭ささに腹がたち、階段下で
悔して涙がこぼれてしまいました。
 
 
 
そこから最後の挨拶だけ出る形になりました。
メンバーに申し訳ない気持ちや
スタッフさんにたくさん助けて
もらったり、あたしを見にきて頂いてる
皆さんや、全国ツアーをこんな
形で終わらせてしまいました。
 
せっかくメンバーでいいものを
作り上げようと意識も変わって
LIVEに取り組む姿勢も改まって
だったので本当に悔しいです。
 
 
 
LIVE終了後に病院に連れてって
もらい診断してもらったところ
左足が捻挫と右足が1週間みてみて
痛みがひかないようなら靭帯の方に
異常があるみたいなので
MRIで検査になるらしいです。
 
 
 
これからお仕事にも支障を
きたしてしまうし、
皆さんにも心配かけてしまうし
本当にごめんなさい(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
ですが、明日のラジオも出演するので
そこでまた自分の口からちゃんと
元気なことをお伝え出来たらなと
思ってます！
 
心はいつも通りのうるさいくらいの
お喋りなあたしですよーー？♡♡♡
 
 
 
 
 
 
 
 
それでは、またラジオでお会い
しましょうーー？\( ˆωˆ )/
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

こんばんたーん( •ㅈ• )
 
今日は真夏の全国ツアー2014
地方LIVEが最後ということで
名古屋に集まって下さった
皆さん本当にありがとうございます！
 
 
なながLIVE中にも言ってたのですが
きっと神宮ではセットリストが
変わってしまうから、今日でこの
セットリストは終わってしまう。から
一つ一つの曲を大事にと、メンバーで
話しながらパフォーマンスさせて
頂きました٩(๑òωó๑)۶
 
その感じが伝わってると
いいのですが...❥❥❥
 
 
 
 
 
でも、皆さんの持っている内輪に
書いてあるメッセージにはたくさん
レスを送ったつもりです⋆｡˚✩  ༘*ೄ˚
送りきれてない方いたら
ごめんなさい〜泣。
 
あたしレスおくるの大好き〜
だから絶対ねあたしに向けて
書いてあるのじゃないのにも
勝手におくっちゃう。笑い
 
例えば「さゆりんごパンチして！」
とかね...「ズッキュンして！」とか
これとか今日見つけたから
レスおくりまくりましたよ。笑い
 
 
アンダーライブを経験してたくさん
学んだと思っていたけどやはり
まだまだ学ぶことは多かったです！
今回のライブでたくさん
気づけたからそれが1番
大きいかなって思います◡̈♥︎
 
もっと、ライブ中あたしのこと
たくさんの方に見てもらいたいの
自分の推し以外も見る！
ていうのはやっぱりライブ
だけだと思うからもっと色んなことに
挑戦したいって思わせて
くれるライブでした！
 
 
そして今日のお昼公演では
あたしの友達のSKE48さんの真木子
がSKE48さんの方達と見に来て
くださいました❣❣❣
 
ライブ中に内輪持ってる
真木子発見したんだ！
 
え？誰の内輪持ってたの？って？
 
それは...恥ずかしいから嫌だ！
って言ったのにあたしの
持ってたんだよ(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
全くもう...でも久しぶりに真木子に
会えて楽しかった〜♡♡♡♡
 
そしでも見に来て下さった
皆さんとの集合写真も！
 
今度はあたしもSKE48さんの
ライブ見に行かせて頂きたいな♡
今度、真木子に頼んで見よう〜
 
 
ライブ前の楽屋は真剣なお話
以外はいつも通りわちゃわちゃ☆
自撮りでぶりっ子してみたり
 
からの皆でガウガウしてみたり
 
 
最近はね、何かしらの公演のたびに
まいやんちゃんかななと髪型
おそろにしてることが多いんだ！
 
今日の夜公演ではまいやんちゃんと
ツインテールおそろでしてたり♥︎
LIVEの時は乃木坂ツインテール率
あがるのよね。笑い
 
若い子気分ね。笑い
 
 
 
それでは今日の楽しい気分を
そのまま心に残して少し
ゆっくりしたいと思います♪
 
 
 
今日をありがとうございました！
