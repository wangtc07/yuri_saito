2013/0731Wed斉藤優里骨だと思ってた部分 は実はコリ。

 
 
 
 
 
\(°Д° )/はっ。
 
※これより第一回斉藤優里の
ガールズトークをはじめるので
男性諸君のふぁみたんは今すぐ
ブラウザを閉じることを
オススメします。
 
というか強制終了だ‼
 
 
 
 
 
 
 
 
......。おい、そこのあなた
男の子だけど見ちゃお♪なんて
甘い考えしてちゃいけないよ‼
 
 
 
 
 
ガールズトーク開始( • ̀ω•́  )
 
 
 
 
 
 
嬉しいことがありました♡えへ
20歳になったということで
まいやんちゃんが誕生日プレゼント
をくれました(❁´ω`❁)*✲ﾟ*
 
 
....にやにや。
 
 
 
 
...に...やにや....。
 
 
 
 
その中身がなんと‼
 
 
 
 
まいやんちゃんととおそろの
下着をもらいました♡♡
きゃあきゃあ♡♡照れ
 
 
写メをあげたいぐらい
可愛いのですが。笑い
 
やはり写メはNGという
スタッフさん判断を頂きました。笑い
 
 
 
 
ダメ元で軽ーく聞いてみたんだけど
撃沈しました(´;ω;`)くっそー
軽い感じで聞けばいけると
思ったんだけどなーくっそー♡笑い
 
 
いやーでも本当にまいやんちゃん
ありがとう♡大切に着用する♡
今度同じ日に着用しようね♡♡笑い
 
あ、まいやんちゃんの誕生日も
楽しみにしててねちゃんと
お返しは下着で返すよ♡♡笑い
 
 
あーまいやんちゃん大好きだー(´;ω;`)♡
 
 
 
 
※以上の文章はガールズトークで
お送りしました。
 
 
ふぅ。楽しい楽しい
ガールズトークでしたね♡
 
 
 
む( • ̀ω•́  )✧男性諸君はこっから
たんたんブログを読むべし‼
 
 
けしてスクロールして上の
ページに戻ってはいけないぞ‼
 
 
今日はFNSうたの夏まつりに
出ますぜーー☆ちなみに
出演時間は19:20以降から☆
 
色んなアーティストさんと
コラボもさせて頂くので
たくさん乃木坂を見ることが
出来ると思いますよ( • ̀ω•́  )
 
 
 
20歳の色気を画面越しで
飛ばすし会場にいる方には直球
投げますので受け取ってね♡♡
 
びゅーーーーーんとね。
 
 
 
はい、びゅーーーん♡
 
 
 
それではまた画面越しでね( •ㅈ• )
チャンネルはそのままで。
 
 
 
......言ってみたかったんだよ。
 
抱きついちゃったー♡
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2013/0728Sun斉藤優里20歳の色気はたっぷり

 
こんばんたーん( •ㅈ• )
 
昨日ブログ書こうと思ったら
途中でいつの間にか
寝てしまいましたー。あー
 
ごめんなさい(´;ω;`)
 
 
そしてそして昨日は宣言していた
通り浴衣での握手会♡♡
 
 
大人な女性になりたくて
自分で着付けてみました♡
 
いつも握手会ではツッコミを
いれられることが多いのですが
浴衣効果でたくさん褒められて
いつもとは違った握手会でした(o´ω`o)
 
 
つまり照れたんだ！！
 
 
昨日もたくさんの方から
おめでとうコメントなど
頂いてまだまだ誕生日気分に
浸れて幸せ♡♡
 
そしてなんと握手会休憩の合間に
サプライズも....(o´ω`o)えへ
 
ふぁみたんにお誕生日の歌を
歌ってもらい絶対泣かないぞ！
って決めてたのにまた泣いて
しまいました。うー
 
20歳になって涙腺が弱く
なった気がします...笑い
ふぁみたんのせいだからな！
 
 
 
 
 
 
本当にありがとうございます( •ㅈ• )
この写真ずっとずっと大切に
するからね？ふぁみたんも
待ち受けにしなよーー♡♡
 
 
生誕委員の皆さんもお疲れ様でした！
アルバムやTシャツ本当にどれも
可愛く出来ていてあたしのツボを
的確についております。笑い
 
こんな素敵な誕生日は初めてです
何回も言っているようだけど
ふぁみたん大好きだ♡♡
 
 
たまにはデレたんだからね(´･ω･`)
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
