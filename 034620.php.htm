2016/0927Tue斉藤優里goodenough♡

 
 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
日曜日の横浜握手会来て下さった
皆さんありがとうございました\ ♪♪ /
しかも、嬉しいのが4部全て完売！！！
という中での握手会！！！
 
 
 
 
 
 
レーンに沢山のふぁみたんが
待っていてくれる嬉しさと言ったら...
語彙力がないから上手く
伝えられてるか分からないけど.....
全身でその会話に答えてあげたく
なるくらい嬉しいんです(´·_·｀)♡
 
 
 
 
 
 
改めてなんだけどお喋りする楽しさとか
握手が1番実感するし、直接ふぁみたんと
お話する機会なんてこういうタイミング
じゃないとないから貴重だよね
いつもありがとう！
 
 
 
 
 
 
今までよりもっと、1人1人の方と
親身になって深い所まで
お話していきたいな...
 
 
 
 
 
 
 
 
 
 
そして次の16thの握手会も
日程が発表されたみたいで
9/29 15:00〜から受付開始
されるみたいです、
今回もフル完売目指して
頑張っていくので
よろしくお願いします◎
 
 
 
 
 
 
 
取り急ぎ、個握の衣装のお話だけど
12/18...ポートメッセはクリスマス
コスプレをするつもりでいます(´･ ･`)
 
 
 
 
 
 
 
この前はブログで書いてた通り
全部数、浴衣を着てました◎
 
 
 
 
 
 
 
 
ちょっと夏の時期は過ぎて
しまっていたけど大好評だったので
着れて良かったなと思いました( ¨̮ )
 
 
 
 
また機会があれば着たいな〜
 
 
 
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
 
 
そして昨日のおに魂
『鬼にゃん』との最後に
なってしまいました(´._.｀)
 
 
 
 
 
まだまだ、足りない所は沢山ありますが
お喋り好きと気づかせてくれた鬼にゃん
今まで過ごしてきた時間は人生の中で
短いかもしれませんが、
私は一生忘れることのない時間になりました
 
 
 
 
 
これからも鬼にゃんとの思い出を胸に
月曜日のおに魂をHappy だんばらさんと
共に頑張っていきたいです！！！！！
 
 
 
 
とっても暖かくどんな時でも
ニコニコの太陽みたいな
鬼にゃんがとっても大好きです◎
本当に本当にありがとうございました
 
 
 
 
 
 
 
 
 
そしてHappy だんばらさん！
これから沢山お世話になります
よろしくお願いします( ˙˘˙ )
 
 
 
早く仲良くなりたいです〜
くしゃくしゃ笑顔がとっても可愛い♡
 
 
 
 
 
 
そして、リスナーさんこれからも
おに魂月曜日を一緒に、
作っていって欲しいです
 
リスナーさんあってのラジオだと
私は思うのでこれからも
傍に寄り添い合って素敵な番組を
作っていきましょうね？
 
 
 
 
 
 
 
 
 
 
 
それでは、舞台の稽古に
いってきますにゃにゃにゃ〜
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
