2017/0727Thu斉藤優里スプラッター

 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
この前のアルバム個握は
全部数完売していました
皆さんアルバムなのに本当に有難うね
ちゃんと楽しめたかな？( ˙˘˙ )
 
 
握手は楽しむことが第1だと思うし
お喋りするのって何より面白いよね〜
 
 
 
 
 
 
その日は生誕祭が全部数の間にあり
沢山の方がお祝いされていました
私もその中の1人でお祝いして頂きました☆
 
 
 
今回は大好きな海外ドラマの中から
ウォーキングデッドをテーマに
レーンを装飾されていたり
Tシャツなどもデザインされてました
 
 
 
 
 
 
レーンを通るスタッフさん達からは
「何で優里だけスプラッターなの？」
って沢山聞かれちゃいました（笑）
 
 
 
あんまり生誕祭でこんな激しく
血痕とかもないですもんね（笑）
でもね、皆の手作り感とか
とっても嬉しくて生誕祭終了後
からまた雰囲気を変えて装飾し
直してくれたりとっても愛が
伝わってきました♡♡♡♡♡♡
 
 
 
 
 
それとねびっくりしたのがいつも以上に
お花が沢山あったの♡♡♡♡♡♡♡♡
ありすぎてさ、本当にお花畑状態（笑）
なんならお花屋さん出来たよあの日（笑）
 
 
 
 
せっかく送ってくれたのに全然
写ってないよって方いたらごめんなさい(TT)
でもねでもね、私のフォルダの中には
しっかり入ってます！
 
 
 
 
 
又、今年も私のセカンドフォトアルバムを
作って下さって今回のタイトルが「彼女感」
には本気で笑いました（笑）
 
 
 
こんな私で大丈夫かな？
タイトルに負けないようにもっと
女の子磨かなきゃね♡♡♡♡♡♡
私にとっては写真集は一つの夢でもあるから
こういう風に少しでもって
形にして頂けること本当に嬉しいです
宝物の一つになりました( ˊᵕˋ )
 
 
 
 
24歳お祝いして頂いて本当に
ありがとうございました♡♡♡
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そしてこの前のTHE 魂には
チョコレートプラネットさんが
遊びに来て下さいました！！！
 
 
 
松尾さんのIKKOさんのモノマネを
生で見られたことが凄い嬉しかったですし
何故か毎度毎度、笑ってしまいました（笑）
 
 
最後の写真も私のポーズをずっと
真似してきてくれました（笑）
おそろい♡♡♡♡♡♡♡♡♡
 
 
 
またぜひ遊びに来て下さい( ˙˘˙ )
 
 
 
 
 
 
 
 
 
 
それでは、また更新する時まで
ばいばい☆☆☆
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
