2016/0721Thu斉藤優里ないしょの落書き

 
 
↑キラキラ加工というものを覚えたので
早速使って見たけどどうかな？
お洒落で可愛いから沢山活用しちゃいそう♡
キラキラ大好き♡♡♡
 
 
 
 
 
 
嬉しいこと続きなんです！！！
 
 
前回のブログのコメント本当に
沢山の方がコメントして下さって
まだまだ読め切れていないのですが
長文で書いて下さるかたや
短めにズバッとお祝いして下さる方
形は違えど、どれも私にとっては
全部全部嬉しい言葉達\( ¨̮ )/♡
 
 
 
 
 
 
いつも私の事を思って色んなことを
教えてくれたり、喝をいれてくれたり
本当にありがとうございます！！！
 
 
コメント本当に大切に大切に
読ませて頂きますね
 
 
 
 
 
 
 
それとね、今凄い嬉しくてね
沢山のメンバーがブログや直接
お祝いの言葉をくれたんだよ( ˊᵕˋ )
 
 
こんなお祝いしてくれるなんて
有難いな〜にゃんちゃんなの
 
 
 
 
 
暴露、されてしまったけどちまが
送ってくれたお祝いのお返事に
 
 
｢これからも推しはずっと「たま」だよ｣
 
 
と打ち間違えをしちゃいました〜
うわああああ、大好きで大好きで
仕方ない「ちま」の名前を
打ち間違えるなんて(´·_·｀)
 
 
 
 
 
言われるまで、全然気づかなかったんだ
ちまごめんなちゃい(´·_·｀)
 
 
 
 
 
とっても優しいから笑って許してくれました
そんな、ちまの優しさにいつも
助けられていますありがとう♡♡♡♡
 
 
 
 
 
 
 
 
そして、ふぁみたんはお気づきですか？
この写真といえば...そう！！！！！
 
15thシングル裸足でSummerの
カップリング曲
「シークレットグラフィティー」
のMVの衣装なんです\( ¨̮ )/♡
 
 
 
 
 
 
 
この撮影は本当に楽しくて愉快なMV、
皆の仲良しな感じがダイレクトに
伝わってくると思います( ´･ᴗ･` )
 
 
 
 
共演させて頂いたのがセンター樋口に
かけて髭男爵の「ヒグチ」さん （笑）
 
 
 
しかも、ご挨拶をしてから1発目が
あの1番最後のシーン（笑）
本当にごめんなさいという気持ちを
心の中にありながら撮影していました（笑）
もおおおお、本当に申し訳ない（笑）
 
 
そのお陰あってか？！
とっても良い作品が出来たと
思うので本当に良かったです♡♡♡
 
 
 
皆が我を忘れて、言葉を叫ぶシーン
面白かったな〜（笑）
誰が何を言ってるか聴き取れたかな？
私のは...内緒にしておこっと（笑）
 
 
 
聴きとってもらいたいもん( ˊᵕˋ )
 
 
 
 
 
 
 
最後のダンスシーンの
休憩中の合間にて（笑）
 
 
 
 
本当にこの写真ナニシテルの...
多分撮ってくれたのは
スタッフさんなんだけど...（笑）
 
1人でポーズ決めてナニシテルの...
 
 
何て不思議？自分でもよく分からない
写真があったり...
 
 
 
 
 
 
 
習字のシーン終了後、
お遊び習字について
思い思いのことを書いた結果（笑）
 
 
 
 
やっぱりちーちゃんは流石！！！
めちゃめちゃ字が綺麗だ
女性としてそういうとこ凄く
羨ましく思います！！！！！！！
 
 
 
てぃーちゃんのは...
好きな食べ物を書いたんだね♡うん （笑）
 
 
 
 
 
 
 
 
 
それではこのブログを見てちょっとでも
「シークレットグラフィティー」
気になってくれたら嬉しいな〜
 
 
 
 
 
 
もう、既にYouTubeの乃木坂46
公式のとこにフルで上がっているので
ぜひチェックしてね♡♡♡♡♡♡♡♡
 
 
 
 
 
 
※あ！ちなみにお家で見ることをおすすめしますす（笑）
ニヤニヤしてしまうから...
 
 
私、僕、俺！ ニヤニヤ気にしないぜ！！！
なんて方はぜひ、公共の場で見てね（笑）
 
 
 
 

 
そして個握の受付がはじまりました
今日の15:00から明日の14:00まで
受付しているので良かったら
ちょっとでも気になる方が居たら
ぜひ遊びに来て欲しいです！！！
 
 
絶対後悔はさせないよ♡♡♡♡♡
 
 
 
ちなみに衣装の予定↓↓
 
 
9/25パシフィコ横浜...浴衣
10/2京都パルスプラザ...パジャマ
10/29幕張メッセ...ハロウィンコスプレ
を予定しています( ˙˘˙ )
 
 
 
 
あんまり完売状況とか
よく分からないけど
もし気になった衣装のとこが
あればぜひ♡♡♡♡♡♡♡♡
 
ライブの話でもたわいもない話でも
ネタでもなんでも大歓迎！！！！！
 
 
それでは明日からの全国ツアー
初日は大阪公演とっても楽しく
私らしく歌って踊ってとばしていくぜ！！！
 
 
 
ぶんぶーん！！！
 
 
 
サイリウムはピンク×白なので
沢山振って下さると探しやすいので
ぜひよろしくお願いします( 　˙³˙　 )
 
会場でサイリウムカラー見つけると
とってもテンションあがります
沢山いるメンバーの中から
このカラーを選択してくれていることや
広い会場のなかから、見つけ出せたこと♡
 
 
 
 
 
 
 
それではまた更新するね♡
にゃんちゃんなの
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
