2017/1017Tue斉藤優里影ナレ昨日でした

 
 
 
 
 
 
 
こんにちたーん( •ㅈ• )
 
 
 
 
 
 
私達は最近アンダラ三昧の日々です♡
ふぁみたんはどうお過ごしですか？
まさか私達と同じスケジュールなんて方も
いたりするかもしれませんよね？（笑）
 
 
 
 
大分公演が終わり福岡公演も2日目に
差し掛かろうとしていますが、
空き時間が出来た時に
純奈 伊織 てぃーちゃんと
お部屋に集まって遊びました( ¨̮ )
 
 
お部屋で遊ぶって言っても
かなりすることは限られているので
YouTube見たり、ボードゲームしたり
てぃーちゃんが持ってきてくれた
ひらがなポーカーがスーパーおもろい♡
 
 
 
 
 
ルールとか凄い伝えたかったんだけど
ちょっと書くの大変そうなので
気になった方はぜひググって見て下さい（笑）
 
そんなことをしながらホテルを堪能してます
 
 
 
 
少し後悔してるのはお家から沢山
入浴剤持ってきたら良かったなって...
2.3個は持ってきたんだけど
あきらかに足りなくて...
あの体癒す効果のが欲しい...
普段選ぶ時はピンクとか可愛いの♡
とかなんだけど今はとにかく
体を癒す効果のものを欲してる（笑）
 
 
 
でも、毎日楽しいです( ¨̮ )
色んなことがある毎日
さて明日や今日はどんなことが
起きるのかな？毎日わくわくです
 
 
 
 
 
福岡公演の初日にてMCの時に
少し大事なお話をするお時間を
頂けたのですが、練習してたけど
途中で上手く言えなくなってしまい
大号泣してしまいました
 
いつもヘラヘラしてるので
こういう時はしっかりと気持ちを
伝えたかったですし、かっこよく
ありたかったのですが
出来ない自分への悔しさや
不甲斐なさからくるものでした
 
 
ですので、ファンの皆さんを
不安にさせてしまったこと
謝らせて下さい、ごめんなさい
 
 
もしまたお話する時があったら
今回みたいなことにならないように
頑張りたいと思います
又、優しく見守って聴いて下さった
ファンの皆さんメンバー本当にありがとう(TT)
 
 
 
 
今日の公演も楽しんできます♡
サイリウム 団扇 タオル毎公演、
とっても嬉しいですありがとう
皆さんの気持ち私に届いてますよ♡
 
 
 
 
 
 
 
※「13金企画」沢山のご参加ありがとうございました
4000件以上のコメントにびっくりしました（笑）
御返事はもう少し待っててくだちゃい








たんたんたん ゆったんたん( •ㅈ• )

さて、ライブですがやはりアンダラはいつもの全体ライブと違ってメッセージ性があると言うか、皆の意気込みもまた全体ライブの時とは一味違うといつも感じさせられます。
それがアンダーの皆がお互いの存在をより特別に思ってるからなのか、それぞれの気持ちから来るものなのかわからないけど毎度僕の気持ちを熱くさせます。

全体の時より人数が少ないこともあって、ゆっちゃんやかりんちゃん含め皆の表情やダンス、表現をしっかり観ることができる、それもアンダラのいいとこだと思います。そして、その度に皆の新たな魅力に気付かされます。

そして、ますますゆっちゃんのことが好きになったよ。これからも一緒に頑張ろうね！

ずっと応援しています。
