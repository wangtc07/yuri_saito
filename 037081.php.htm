2017/0226Sun斉藤優里Thank you!

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
ななみん卒業ライブが終わり
今年のbirthdayライブも無事終了し
何だか寂しくなってしまいました
 
だってすぐ日程の決まっている
イベントとか目に見えて
分からないんだもん〜
きっと皆もそうだよね(TT)
 
 
何か何か物足りないじゃないけど
寂しかったりするよね(TT)
 
 
 
 
 
 
 
 
卒業ライブはセットリストを
決めてくれたりまりかにサプライズ
してみたりとっても手作りな感じや
あんまりしんみりさせたくないという
ななみんの気持ちが全面に出ていた、
とっても素敵なライブになりました◎
 
 
 
サヨナラの意味で1人でムービングステージ
で歌う姿はとってもかっこよくて
いつもより背中が大きく感じました
それと同時に遠くに行ってしまうという
実感がとても強くなって胸にくるものが
ありました、それはきっと私やメンバーだけ
でなくスタッフさんやファンの皆さん
全員が感じて居たんじゃないかな
 
 
 
涙を流さずに送り出したかったのに.....
私達が泣いていたらきっと
卒業していくななみんが
辛いじゃないですか.....
だから笑って送り出したかったんだけどな
無理でした（笑）
 
 
 
 
ななみんにとっての乃木坂での生活
楽しい思い出の中に私も一緒に居るといいな
ご飯食べに行ったりしたことが
本当に楽しかったしななみんとご飯
行く時は99.9%焼き鳥屋さんだったの
覚えてるよ( ˙˘˙ )
いつか、また会えるときが来るといいな
 
ずっとずっと応援しています！
 
 
 
 
 
 
 
そして終演後に面白い写真を
沢山撮ったので載せちゃいますね☆☆☆
 
 
 
 
 
 
ライブ後のメンバーは本当にテンション高くて
大好きなんだ〜皆の周りに
星がいっぱい飛んでる感じ！！！
って言ったら分かりやすいかな？（笑）
 
 
 
 
 
 
 
♡告知♡

2/28 BUBKA「私たちが雑誌を作るなら」
という取材をして頂きました
なかなか面白い発想などあったので
ぜひチェックしてみてね？
 
明日20:00〜おに魂
いつも通りの楽しい放送をお届け！
 
 
 
 
 
それではまた更新しますね（笑）
明日のおに魂まだまだ親知らず
腫れてるかもですが
よろしくお願いしますね♡（笑）
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
