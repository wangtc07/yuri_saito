2017/1029Sun斉藤優里ぎゅっとねぎゅっー

 
 
 
 
最近ハロウィンが近いからなのか
分かりませんが色んなコスプレ？仮装？
衣装？を着せてもらう機会がとっても
多いです☆☆☆☆☆
 
 
 
私的にはとっても楽しくて楽しくて
色んなものを着せてもらえるのも又、
アイドルならではなのかなとも
思ったりしています（笑）
 
 
 
 
そんな色んなお洋服を着た時のお仕事報告
していきたいなと思います（笑）
 
まずはセブンイレブンの店員さん！！！
 
 
 
この日は大阪で１日お店の店員さんを
してきました\ ♪♪ /
通い慣れたセブンイレブンでも
店員さんになるとまた気分が違ったり（笑）
 
 
昔のバイトしてた時代を思い出しました（笑）
バイト大好きだったな〜特に声出して
ドライブスルー担当するのが好きでした♡
 
それとレジ打ちも大好きでした♡
 
 
 
 
今度は仙台でもあるということで
楽しみにしています！！！！！
雨の中だったけどご来店して下さった
皆さんありがとうございました！！！
 
 
 
 
続いては、昨日の握手会でも「見たよ！」
って方が物凄く多かった「CDTV」
 
 
 
醜い私のユニットチームで撮ってみました
一応この中にも設定があって
私は毎日バスケに取り組む生徒
みさはそんな私の先生そして、
まいちゅんは私が突き指をした時に
見てくれる優しいナースさん♡
(※バスケ部だから突き指の怪我が多いかなって...)




そしてこの衣装でのガルルは
赤いのがとっても目立っていましたね
何よりもユニフォームっていうのが
元々スポーツするように作られているから
誰よりも1番動きやすかったと思うの（笑）
 
 
 
 
そして昨日の握手会はこんな感じ♡♡♡
 
1.2部はうさぎちゃん
 
 
打ち合わせなしで純奈と奇跡のシクロ率
にびっくりしちゃった〜
だってうさ耳ちゃん同じお店の
カチューシャだったの凄すぎよね♡♡♡
 
 
 
3.4部はダルメシアン
これね凄いファンの人に
「かりんちゃんが作ったやつ？」
って聞かれたんだけど3期生が
イベントで着てた衣装だったんだ〜
 
桃子のを借りました（笑）
ちなみに桃子には内緒で（笑）
 
 
犬めんなだけに犬 わん！！！！
ちなみに若月も同じタイミングで
着てたんだけど写真撮ろうとしてた
時にはもう既に着替え終えてたの(TT)残念
 
 
 
 
 
 
やっぱりイベントごと大好きだな〜
何か気分がウキウキするのよ
特にハロウィンね！！！！！
 
本当はパーティーとかしたいんだど
少しいそがしくてまだ出来そうにないな(TT)
色んなことが落ち着いたらパーティー解禁だね☆
 
 
 
 
 
 
 
 
それではまたブログ更新する時まで
まったね〜今日はゆっくりお湯に浸かるぞ！
 
あ、台風がきてるから外出中の方は
気をつけて下さいね？♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2017/1021Sat斉藤優里宮崎マンゴー食べれなかった

 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
アンダラ千秋楽無事、18人で完走
することが出来ました！！！！
自分と沢山戦ってくれた
めーちゃんや北野お疲れ様！！！
そして私の周りにいる大好きなメンバーも
よく頑張りました！！！
 
 
 
 
めーちゃんや北野のMCを聞くまで
どういう気持ちなんだろうとか？
今何を考えているのかな？とか
私も思っていました
でも、MCで反感をかうかもしれないのにも
関わらず勇気を出して事実を喋ってくれたこと
本当にかっこよかったですし、
私も2人の気持ちを彼女達の口から
きけたのが何よりも嬉しかったです
 
 
 
 
 
 
アンダラって全体のライブとは違って
距離が近いのも勿論だけど
アンダラに来てくれるだけあって
やっぱりアンダーへの愛が強い方が
とっても多くて全体のライブではなかなか
自分の気持ちを喋る機会をもらえるわけ
ではないからその分、喋りたいこととか
そこに全部ぶつけています
 
 
でも、そういう気持ちをファンの方は
正面からあたたかく受け止めてくれる
何かこういうのっていいなって思います
 
 
 
 
 
 
 
後、セットリストの中に「僕だけの光」
が入っていたのですが個人的に
アンダラで歌うこの曲はひと味違いました
歌を前より自信を持って歌えるように
なれたのもまた良い一歩というか
この機会のおかげでもあります
 
 
 
 
 
 
アンダラ近畿・四国ツアーの開催が
決定しましたね、きっとまた
メンバーが素晴らしいものを
みせてくれるに違いないと思います！！！
 
 
 
 
 
 
 
 
 
 
最後に宮崎でのハイライト（笑）
 
 
 
 
 
 
 
♡告知♡

・10/24 発売のB.L.T.にてかなとのペア
グラビアをやらせて頂きました
とっても楽しい一日でした
内容は濃くは言えないのですが、
コテージで遊んだりお花畑に
行ったりしてきたよ( ˙˘˙ )
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2017/1017Tue斉藤優里影ナレ昨日でした

 
 
 
 
 
 
 
こんにちたーん( •ㅈ• )
 
 
 
 
 
 
私達は最近アンダラ三昧の日々です♡
ふぁみたんはどうお過ごしですか？
まさか私達と同じスケジュールなんて方も
いたりするかもしれませんよね？（笑）
 
 
 
 
大分公演が終わり福岡公演も2日目に
差し掛かろうとしていますが、
空き時間が出来た時に
純奈 伊織 てぃーちゃんと
お部屋に集まって遊びました( ¨̮ )
 
 
お部屋で遊ぶって言っても
かなりすることは限られているので
YouTube見たり、ボードゲームしたり
てぃーちゃんが持ってきてくれた
ひらがなポーカーがスーパーおもろい♡
 
 
 
 
 
ルールとか凄い伝えたかったんだけど
ちょっと書くの大変そうなので
気になった方はぜひググって見て下さい（笑）
 
そんなことをしながらホテルを堪能してます
 
 
 
 
少し後悔してるのはお家から沢山
入浴剤持ってきたら良かったなって...
2.3個は持ってきたんだけど
あきらかに足りなくて...
あの体癒す効果のが欲しい...
普段選ぶ時はピンクとか可愛いの♡
とかなんだけど今はとにかく
体を癒す効果のものを欲してる（笑）
 
 
 
でも、毎日楽しいです( ¨̮ )
色んなことがある毎日
さて明日や今日はどんなことが
起きるのかな？毎日わくわくです
 
 
 
 
 
福岡公演の初日にてMCの時に
少し大事なお話をするお時間を
頂けたのですが、練習してたけど
途中で上手く言えなくなってしまい
大号泣してしまいました
 
いつもヘラヘラしてるので
こういう時はしっかりと気持ちを
伝えたかったですし、かっこよく
ありたかったのですが
出来ない自分への悔しさや
不甲斐なさからくるものでした
 
 
ですので、ファンの皆さんを
不安にさせてしまったこと
謝らせて下さい、ごめんなさい
 
 
もしまたお話する時があったら
今回みたいなことにならないように
頑張りたいと思います
又、優しく見守って聴いて下さった
ファンの皆さんメンバー本当にありがとう(TT)
 
 
 
 
今日の公演も楽しんできます♡
サイリウム 団扇 タオル毎公演、
とっても嬉しいですありがとう
皆さんの気持ち私に届いてますよ♡
 
 
 
 
 
 
 
※「13金企画」沢山のご参加ありがとうございました
4000件以上のコメントにびっくりしました（笑）
御返事はもう少し待っててくだちゃい
