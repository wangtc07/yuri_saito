2018/0110Wed斉藤優里台湾謝謝

 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
1/2からお休みを頂いたので
友達と台湾旅行に行ってきました 
はじめての、台湾♡♡♡
 
 
寄りたい場所を事前に調べてたんだけど
凄く行ってみたい所が多くて大変だった（笑）
夜市に行って小籠包、タピオカ飲んだり...
終電までそこで遊んでたよ（笑）
 
 
 
台北から新幹線？みたいなもので1時間
離れた台中にも移動してカフェ「I'm talato」
黄おじいちゃんが家屋にペイントした「彩虹眷村｣
にも遊びに行ってきたの( ¨̮ )
 
 
 
 
 
びっくりしたのが地面にもペイントされてること！
こういう場所であまりここまでペイント
されているのを見たことがなかったから感動☆
何より、色使いがなかなか日本では
見ることが出来なくて素晴らしい☆
 
 
 
 
ここでは、友達と交代に写真を撮っているのを
みかねた「現地のおじいちゃん」が私達の写真を
めちゃくちゃ撮ってくれたんです(ˊᵕˋ)
しかもね、ポーズとかも教えてくれて
本当に良い思い出の写真が出来ました（笑）
おじいちゃんはきっと私のことなんて
1人の観光客としか思わないかも
しれないけど私達にとっては本当に
有難い存在です♡♡♡♡♡♡♡♡♡
 
 
おじいちゃんぶろぐ見てくれたらいいな（笑）
 
 
 
 
そして、台湾を調べていた時に
めちゃくちゃ行きたいーって
ここは外せないー！って
心のInstagramが騒いだスポットが
こちらのカフェ「I'm talato」♡♡♡♡
 
 
 
 
 
これさ、え？どういう状況なの？
ってなるでしょなるでしょう？
カフェに隣接された撮影スポットなんです！！！
テーマはプールがイメージ？なのかな？
プールサイドのようなタイル貼りの地面
だったりがより一層可愛いアイスを
ひきたててる♡♡♡♡♡♡♡♡♡
 
 
 
 
 
アイスに溺れてきた♡♡♡
しかもね可愛い可愛いピンクとホワイトの配色♡♡♡
 
 
 
あーもう大好き〜幸せ♡♡♡




といった行きたかった場所に
行けた大満足の1日ですが...
台湾といえばここは行くよね〜みたいな
マストの場所は次のブログに書しますね！
 
 
 
 
本当は1つのぶろぐでも良いかな
って思ったんだけど写真の都合上で...（笑）
やっぱりさ？ほら？沢山載せたいじゃん？（笑）
 
 
友達がねしっかりしたカメラを
持ってて撮ってくれてたんだけど
大体1日で240枚くらい撮影してたの（笑）
やばいよね？送ってもらった時保存するのも
大変だった（笑）
でも、それほど残したいものが
沢山ある素敵な台湾！！！！！






それじゃあ、また次のブログ楽しみにしててね？
 
 
 
 
 
 
 
こいつ、愛くるしい♡♡♡











たんたんたん ゆったんたん( •ㅈ• )
