« 実はまだ行けてません｜斉藤優里TOP｜あ、免許更新にだよ？ »

2016/0628Tue斉藤優里やっと行けました( ˊᵕˋ )

 
 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
おに魂ではねいつも「乃木坂マスター」
ってコーナーがあるんだけどその後の
曲は必ず乃木坂なんだ！！！！！
 
 
 
 
そこでいつも毎回どの曲にしようかな〜
とか今回このメンバーがおに魂では
ピックアップされてるから
このメンバーが歌唱してる楽曲にしようかな
って考えたりしてるんだけどね、
昨日は急に「きっかけ」が聴きたくなって
ラジオ中だからかな？
自分で1人で聴くのとはまた違って
大体、音楽聴く時って移動中だったり
するんだけどねラジオ中だから
しっかり聴き入っちゃた( ´;o;` )
 
 
 
 
 
 
 
本当に良い曲で雨降ってる時に
聴くとより一層ジーンって
来るものがありますよね
 
 
 
 
 
今日も沢山聴こう〜っと\( ¨̮ )/
 
 
 
 
 
 
 
 
そして土曜日の「それぞれの椅子」
握手会関東は久しぶりの4部制
だったのではりきってました（笑）
 
今日も喋り倒してやろう♡♡♡と（笑）
 
 
 
 
 
 
この前のぶろぐでアンケートの結果発表にて
「太陽ノックライブver.」を1部は着たよ
ただね、この前のと全く一緒だと
前来て下さった皆さんに申し訳ないかなと
なので形違いのてぃーちゃんの形を
着てみました( ´･ᴗ･` )
 
 
 
この形、私的には結構好きなんだけど
やっぱりふぁみたんには
あれ？なんか違う...（笑）
と言われてしまいました（笑）
 
↑やっぱりふぁみたんは透け透けが好きなんだな♡と
だから次の握手会からちゃんと自分の形着るね\( ¨̮ )/♡
 
 
 
 
 
 
2部は「今、話したい誰かがいる」
 
 
この衣装なーんだ？って質問したら
正解率がめちゃめちゃ低かったよ( ´;o;` )
衣装凄い可愛いから見てーーーー♡♡♡♡
 
その時の正解率2割以下だったと思う
真面目に（笑）
 
 
 
 
もー全く正解してよ正解！٩( ö̆ ) وぷん
 
 
 
 
 
 
３部「ハルジオンが咲く頃」
 
 
これは実は中のワンピースが紫とピンクver.
あるんだけど私は勿論ピンクだよ♡♡♡♡♡
大好きなカラーだから(ˊᵕˋ)
 
ベットのどこかしらや
パジャマなど、どこかしらに
必ずピンクは入ってるんだ◎
 
 
 
 
 
 
 
4部は「不等号」
 
 
実はこの衣装も襟元？のレースが
それぞれ違ったりするんだよ( ´･ω･`)
細かいとこまで拘ってる...
気づいた方はいるかな？
 
 
 
 
 
 
 
 
 
 
次の握手会も衣装が着れたり、
だったりすると楽しみが増えるよね
さーて何着ようかな♡♡♡♡♡♡♡
 
 
それとね、最近髪の毛のアレンジが
沢山したくなってきたねといつも
髪の毛をしてくれるメイクさんと
喋っていたのでこれからまた
前くらいの長さに戻そうと思うんだ〜
 
 
 
 
早く髪の毛伸びろ( 　˙³˙　 )
北野に亜鉛を摂取すると
早く伸びると聞いたんだけど
 
手軽な亜鉛のサプリで試してみよう（笑）
 
 
 
 
 
 
他にもこれで髪の毛めっちゃ早く伸びたよ！
なんて方がいたら教えてください...（笑）
 
わかめを食べると伸びるのが早いのは本当？
 
 
わかめお味噌汁にいれて
摂取してるけどどうなのかな？
専門家の方そこんとこぜひ
教えてくだちゃい！！！！（笑）
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
 
冒頭でもおに魂のお話したけど
昨日のゲストは ユナクfrom超新星から
ユナクさんが来てくださいました
 
 
 
 
流暢な日本語でとっても
楽しいラジオをお送り出来ました！
これからのリリースイベント
応援しています\( ¨̮ )/♡
 
 
帰り際にCDを頂いたので
お家で沢山聴きたいと思います◎
 
 
 
 
 
 
それではまた更新したいと思います〜
雨降ってて肌寒いからあんまり
薄着で出掛けないように
気をつけてくだちゃい！！！
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
