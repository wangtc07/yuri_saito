2015/0630Tue斉藤優里花火みたいな

 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
じょしらくの千秋楽が終わってから
少し日があいてしまいましたが
じょしらくぶろぐお待たせしました♡
 
 
 
 
 
 
今まで舞台経験がなく、それに役では
とっても台詞量の多いキャラで
不安しかない状態での稽古でした
 
 
 
 
ちーむが発表されてたから
そのメンバー全員でなかなか合わせたり
する時間がなかったりと....
稽古に出れなかったりすると
知らないことが沢山増えていたり....
 
 
 
 
苦手意識が強すぎてあんまり
その気になれなかった時期も
正直ありました。
 
 
 
 
 
でも、自分でも本当に不思議に
思ってるくらいで稽古していて
川尻さんが色んな演出をつけて
下さっているうちに凄くワクワク
してきたんです。
 
本当に今まで決して好きとは言えない
舞台を楽しんでいました。
 
 
 
 
 
じょしらくは私に舞台の面白さを
教えてくれました。
本当にまりぃちゃんを演じれて
よかったです。
他の舞台も沢山見て色んなこと
勉強したいし体感したいです。
まだまだだけど、舞台もいつか
斉藤優里個人としても出演してみたいです。
 
 
 
 
 
本当に温かい拍手を下さったふぁみたん
ありがとうございました
ちーむら のマイペースでまったりしていて
それぞれ自分を持っている皆が大好きです。
 
 
 
 
 
そして千秋楽にはお母さん達も仲良く
なっていて5人のお母さんでご飯に
行ってたりと....私達の知らないとこで。笑い
 
 
 
 
それにそれに、お花まで
頂いちゃって嬉しかったな〜◎
 
 
 
 
私達もお仕事落ち着いたら打ち上げ
しよーね？(ˊ•̤ω•̤ˋ)
 
 
 
 
 
 
 
告知です⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
 
 
 
着信御礼！ ケータイ大喜利 
7/18 23:30〜に出演します♡♡♡
メインでお喋りさせて頂きました
 
 
 
お腹抱えるほど笑っちゃた〜
せひチェックしてみてね？
カットされてなければ....笑い
 
 
 
 
 
 
それでは夜まで空き時間が出来たので
フラフラしてきたいとおもいます♡♡♡
 
 
 
みょーーーん♡♡♡
 
 
 
 
 
 
 
まいまいで遊ぼう♡♡♡
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2015/0618Thu斉藤優里牛乳飲んだらいいの かな

 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
ととととととととととととと
とりあえず、無事 チームら
初日公演終えましたああああ。
 
 
 
 
 
緊張で本番中、喉乾きすぎて
ペットボトル飲みほした\( ˆωˆ )/いぇい
 
 
 
 
 
 
 
1人一役、任せられること自体
はじめての舞台。
あんな大きな舞台に5人で挑むこと。
怖いという気持ちや、
楽しみたいという気持ち。
 
 
色々な感情の中でそれぞれ思うことも
あるかもしれませんが、私はまず
5人揃って舞台に立てたことが
嬉しかったです。
 
 
 
それに一緒に稽古してきたメンバーから
応援の連絡がきていて....
なんだろう、舞台に立っているのは
5人なんだけど気持ちは皆と一緒に
いるようなそんな気持ちでした。
 
 
 
 
 
 
 
良いスタートダッシュ！迎えられたと
思います、でも、もっともっと楽しんで
皆さんの笑顔が見れる舞台が
作れる思います。
 
 
 
 
残りの4公演、悔しい気持ちで絶対
終わりたくない！
私、頑張るの。
 
 
 
 
 
 
それでは初日は２度とないので
今日来てくださった皆さんと
同じ時間を過ごせて
本当に良かったです。
とってもとっても暖かい愛のある
拍手をありがとうございました。
 
 
 
 
 
 
チームら チームく チームご
どのグループも素敵な舞台が
千秋楽まで続きますように....
 
 
 
 
 
 
それでは最後になってしまいましたが
みおな♡緊張を見せないしっかりとした
落語披露の姿、舞台袖から見ててとっても
かっこよかったよ！
 
 
 
 
 
 
明日は落語のお稽古
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
