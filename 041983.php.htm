2017/1130Thu斉藤優里シルバー気になる

 
 
 
 
 
 
こんにちたーん( •ㅈ• )
 
 
 
 
 
この前はベストアーティスト2017に
出演してきました、755にも皆との
動画載せたから見といてね♡♡♡♡
 
 
 
 
 
今回は自分達の持ち曲
「いつかできるから今日できる」
の他に嵐さんとのコラボ曲も
披露させて頂きました
 
 
 
 
大スターの方と同じステージに
立たせてもら︎えて更に一緒に歌わせて
頂くなんてとても畏れ多かったですが
精一杯歌わせて踊らせて頂きました
 
またとない機会を頂いたこと
本当に感謝しています、とっても楽しかったです
嵐さん、ありがとうございました(ˊᵕˋ)
 
 
 
 
♡告知♡

・12/3 乃木のの 文化放送
・12/8 週刊プレイボーイ増刊号

 
 
 
ぜひチェックお願いします☆
乃木ののは小百合と久保ちゃんと
めっちゃ楽しめたよ( ˙˘˙ )
かなり、わいわいしてた気がする（笑）
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
