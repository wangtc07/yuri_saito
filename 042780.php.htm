2018/0109Tue斉藤優里現在の充電 61%

 
 
 
 
2018年あけましておめでとうございます☆
 
 
 
 
2017年は私にとっては物凄く良い
スタートからはじまり、
悔しい思いも正直ありました
そしてグループとしてもとても大きく
成長出来た1年でした
決して、嬉しい思いや楽しい思いだけでは
ここまでよかったと思える1年には
ならなかったと思います
悔しいことや辛いこと、
そういう感情もあるからこそ
一層感情の起伏があって嬉しいことは
更に嬉しく輝いて感じることが出来るのです
だから、悔しい感情も嫌いじゃない...
むしろありがとうなんて思えたり...
 
 
 
 
レコード大賞を頂けたり
東京ドームで公演が出来たり
自分達だけではなし遂げられなかった
ことも沢山ありました
スタッフさん ファンの皆さん
本当に去年は素敵な年にして
頂いてありがとうございました
 
 
 
 
 
 
まだまだ手にしてないもの
見たことのない世界があります 
今年はそれを1つでも集めたり、
そんな場所に足を踏み入れてみたり
色んな冒険が出来たらいいな
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
そして何と嬉しいことでしょう
2018年1発目のゲストは
「乃木坂46」でした（笑）
 
 
 
 
 
蘭世 絢音ちゃん 遊びに来てくれてありがとう
そして1/10発売の
「僕だけの君〜under super best」を
よろしくお願いします( ´･ᴗ･` )
 
 
 
個人的には新しい曲が4曲も
入ってるのが最高だし何より
それぞれその曲のセンターが
違うのが尚更良い！！！！！
 
 
 
そして、今まで衣装についてピックアップ
されたことがなかったけど今回のブックレット
にはそれぞれの楽曲衣装が掲載されているので
この時はこんなの着てたんだな〜なんて
思って懐かしんだり.............
乃木坂の衣装って本当に凄く可愛く
作って頂いているので細かな所まで
写真で見れるのって本当に良いよね！！！
それだけでも、ずっと眺めてられるし☆
 
 
 
 
 
 
それじゃあ、また更新する時まで( ˙˘˙ )
 
 
※ちょっとだけ時期ずれちゃった？（笑）












たんたんたん ゆったんたん( •ㅈ• )
