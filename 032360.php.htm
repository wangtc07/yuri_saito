2016/0522Sun斉藤優里美味しいものがいっぱい

 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
早速ですが！！！前回のぶろぐに書いてた
Lancomeのジューシーシェイカーを
使ってみたよ♡♡♡♡♡♡♡♡♡♡
 
 
 
 
 
 
私が買ったのは352の
「ワンダーメロン」
他にもこれを含めて
今発売されてるのは14色みたい(ˊᵕˋ)
 
 
 
凄くいい匂いがするんだけど
調べてみたらウォーターメロンの
香りなんだって、あんまりグロス
とかでそういう香りがないから新鮮！！！
 
 
 
 
最近のお気に入りの
青みピンクだから
はじめての方にも挑戦しやすくて
ポンポン付けるとほんのりついて
しっかり着けた時はグロスのように
つけたり出来るからおおすすめです♡
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そして今日はめーちゃん まあやと
バズ☆ライブドリームにトークメンバー
として出演してきました( ˙˘˙ )
 
 
 
普段自分達でライブをしたり
する側だったりするので
こうやって沢山のアイドルさんと
共演が出来て本当に良かったですし
凄く楽しかったです♡♡♡♡♡♡♡
 
 
 
又、千原せいじさんも凄く優しくて
いつもより緊張しないで
喋れたから良かったな〜
 
 
放送はまだちょっと先なので
内容は言えないですが
楽しみにしててね？
 
 
 
 
 
 
そして一緒に写真を撮ってもらった
方をご紹介＼＼\\ ٩( ˊᵕˋ )و //／／
 
 
 
右上 愛乙女★DOLLの太田里織菜さん
左下 愛乙女★DOLLの愛迫みゆさん
右下 愛乙女★DOLLの吉橋亜理紗さん
 
 
 
 
 
自己紹介の時にライブを熱く！
しているというのを聞いて
私もライブがめためた好きだから
凄く見に行きたいです\( ¨̮ )/♡
 
時間が少なかったからあれだけど
もっと色んなお話してみたかったな〜
 
 
 
 
 
 
そして左上のアフィリア・サーガの
ユカフィン・ドールさん！！！！！
 
おに魂にゲストで来て下さっていたので
見かけた瞬間に話しかけてしまいました
わ〜だってすごい嬉しいんだもん( ´›ω‹｀)
いつ見ても可愛いの♡♡♡♡♡♡♡♡
 
 
 
また、どこかの現場で絶対
お会いしたいと思える素敵な
アイドルさんです！！！！
 
 
 
 
 
 
 
 
 
それでは放送日が決まったらまた
お知らせしますね◎
今日のブログは化粧のことについて
書いて見たのでもし良ければ
女の子は参考にしてみてください〜
 
 
また、こんな美容のことについて
ブログ書けたらいいな◎
 
 
 
ほな＼＼\\ ٩( ˊᵕˋ )و //／／
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
