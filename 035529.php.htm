2016/1120Sun斉藤優里はじめての味

 
 
 
 
↑大好きなTWICEのTTマーク(TT)
 
 
 
 
 
 
 
 
昨日はお仕事が早く終わって
お買い物行くか悩んでる所に
かりんちゃんにね
「今日はパパとママと鍋するんだ〜来る？」
とお誘いをしてもらいました( ˊᵕˋ )
 
 
 
 
 
ということで、急遽ではありますが
かりんちゃん家にてぃーちゃんと
お邪魔してえび味噌鍋を
頂いちゃいました♡♡
 
 
 
 
とっても美味しくてそれに
かりんちゃんのパパもママ
優しくて私が来ることを
知ったパパは更に買い出しに
行ってくれたみたいです♡♡♡
 
ポテトもチキンも全部全部
美味しかったな〜
 
 
 
 
 
家族の団欒お邪魔するのは
あれかなって思ったけど
とっても気さくでまた、
会いに行きたくなっちゃう（笑）
むしろもう会いたいくらい（笑）
 
 
 
 
 
 
 
 
とっても仲良くなれたんだ\ ♪♪ /
そんな楽しい１日でした
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そして先日のおに魂ゲストは
「アルカラ」です！！！
 
 
 
 
 
 
まさかの真夏の全国ツアーにも
いらしてたなんてそして
ペンライトまで嬉しい限りです(TT)
 
 
 
 
お話の内容も自分達のことよりも
乃木坂のことを沢山喋って頂いて
何だかとっても不思議感じでした（笑）
 
またすぐ遊びに来てくださいね？
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
