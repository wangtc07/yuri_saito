2015/0729Wed斉藤優里はにゃん

 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
一昨日、昨日の握手会来てくださった
皆さんありがとうございました♡♡♡
そしてなんといってもセブンイレブンライブ！
 
 
 
 
 
沢山楽しいことがつまってた
2日間でした。
 
 
 
 
 
セブンイレブンライブの方のMCで
ななが言ってたのですが、関西の地域では
セブンイレブンのことを「セブイレ」って
訳すらしくて....驚きました(ˊ•̤ω•̤ˋ)
前々から話してる時にちょいちょい
セブイレってワードは出てくるなって
思ってたけどそうやって呼ぶのですね
東京の方では「セブン」って呼ぶことが
私の周りでは多かったので
なんだか面白かったな〜
 
 
 
 
おいシャンのいつもの掛け声は
関西の地域の方に合わせて
生駒「コンビニといえば〜？」
乃木坂「セブイレーーーー！」
と急遽変更したんだ⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
 
裏で皆で伝言ゲームみたいにして
回したんだ、楽しかったな〜
 
 
 
 
 
まだまだセブイレライブは残っているので
楽しみにしていてくださいね？
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そして個握の2部終わりに私の生誕祭が
行われました⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
おに魂をそのまま忠実に再現....
永島がいつもの鬼にゃんの立ち位置を
してくれて私がゲストさんという形でした。
 
 
 
 
とっても新鮮な生誕祭りをありがとう
ございます♡♡♡
この生誕はラジオパーソナリティー
してる人にしか味わえないものですもんね
とっても特別だな( •ㅈ• )
 
 
 
 
サプライズで鬼にゃんのコメントが
流れたり....本当に嬉しかったな
しっかりと番宣がはいってたの
そこが皆さん１番笑ってた気がする( ･᷄ὢ･᷅ )む
 
ゆっちゃんのとこで沢山
盛り上がってよー( ･᷄ὢ･᷅ )む
 
 
 
 
でねでね生誕祭の企画で
斉藤優里のココが好き！という
企画をやらせてもらったんだけど
ふぁみたん( ･᷄ὢ･᷅ )変なとこばっか
好きなのねーーーーーーーーーー。
まったくもう♡♡♡
 
 
 
 
居なかった方のために特に私が
気になったものを....
 
 
 
 
斉藤優里のココが好き！
「存在すべて!!」
 
？えーと....ありがとうございます。
私生きてるだけで好んでくれたのですね。笑い
 
 
 
斉藤優里のココが好き！
「ワキ!!」
 
....え？ まってまって、次はかなり
ピンポイントすぎるやろ(ˊ•̤ω•̤ˋ)
しかも、個人的なフェチ入ってきてる
じゃないかーーーーーーーーーー！笑い
 
脇....これからも可愛いく素敵な
体でいようと決意しました。笑い
 
 
 
 
 
 
 
 
いやーそれにしても皆さん私のどこを
好んで推してくれているかとか
本当に様々でとっても面白いものを
見してもらいました♡♡
 
 
 
生誕委員の皆さん本当に
ありがとうございました。
これからもずっと一緒に
いれたらいいな(ˊ•̤ω•̤ˋ)
 
 
 
 
 
そして最後になりましたが
永島さん、全握のペアそして
私の生誕祭のMCありがとうございました！
永島さんにしてもらえて本当に
良かったです♡♡♡
 
 
 
 
 
あ、そういえばお手紙で
「一緒にご飯食べに行こう！」
って約束は昨日もう叶ちゃっいました。笑い
 
 
でも、また行こうね♡♡♡笑い
 
 
 
 
 
 
 
ふぁみたん大好きだよ？
 
 
 
 
今日はFNSうたの夏まつりです\( ˆωˆ )/
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2015/0721Tue斉藤優里はっぴー

 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
ぶろぐ何を書いていいのか
分からない程沢山書きたい
ことがあるんです♡♡♡
 
 
 
まずは、ずっとずっと御報告
したかったお仕事について書こうかな〜
 
この度、北野と飛鳥の3人で
ANNA SUIの秋冬モデルを
担当させて頂くことになりました
 
 
 
この話を頂いた時に素直に
なぜ、この３人なのか疑問が
浮かんできました
 
 
 
お話を聞いているうちに
雑誌でも大きく掲載されたり
街にもこのポスターが貼られたり
するということでますます
？ばかり浮かんできて....
 
 
 
 
暫く信じられませんでした(´°ω°`)
 
 
 
 
私の夢の一つであるモデルという
お仕事、それも日本だけじゃなく
世界的の有名なANNA SUI
 
本当に本当に嬉しくて
すぐ家族に連絡したのを覚えてます
 
 
 
直接担当の方とお話した時に、
この話を頂いた時が
アンダーメンバーだったのですが
 
「今アンダーメンバーが勢いがあって
頑張っているから」というお言葉を
頂きました。本当に嬉しくて嬉しくて。
アンダーメンバーにもこんな大きなお仕事を
任してもらえるほど成長したんだなって
思えたら胸がいっぱいです。
 
 
 
今までアンダーと選抜を行ききしている
私ですが、どこにいてもちゃんと
見てくれてる方はいるんだなって
本当に思えました。
勿論、ふぁみたんにも選抜にいてほしい！
って思ってくださる方もいらっしゃると
思うけど、どこにいても私を見つけて
くださるふぁみたんがあっての私です。
 
これからもずっとずっと
応援よろしくお願いします。
 
 
 
 
本当にこんなお仕事自分が
出来ると思っていなかったので
本当に嬉しかったです
ANNA SUIのファンの皆さんに
認めてもらえるような大きい人に
なりたいです。
 
 
 
 
最後にわたし達を選んで良かったと
言ってもらえるようになりたいです
 
 
本当にこのお仕事が出来て幸せです
ありがとうございます
そしてよろしくお願いします
 
 
 
 
 
 
撮影現場がNYだったので
沢山オフショット載せたいと思います♡♡♡
 
 
セルカ棒の使い方がずっと分からない....
 
 
 
 
飛鳥と北野と行けて本当に良かったな〜
今度は皆で行きたいな。
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
昨日の7/20に22歳になりました
乃木坂に入ったのが19の時だったので 
そう考えると物凄く早く感じます
 
 
 
 
21歳は私にとって、とっても大きな
お仕事をやらせてもらえて本当に
いい一年になったなって思います。
 
 
 
 
これからも人と人との縁を大切にして
22歳はもっと大きいことを
してみたいなって思います。
歳を重ねる度に人として成長出来て
去年の自分を超えたい。
 
 
 
 
これからもよろしくお願いします♡♡♡
ふぁみたんと出会えて色んな季節を
一緒に過ごせて、色んな壁を乗り越えて....
沢山の力をいつもありがとうございます。
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
