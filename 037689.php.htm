2017/0331Fri斉藤優里お花だらけ


 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 

 
TWICEのshowcaseが発表されてる
ううう(TT)日本で6/28にアルバム発売
されてから7/2にshowcaseなんて
めちゃめちゃ最高じゃないですか！！！
 
 
 
 
私もお仕事がなければ絶対行きます♡
 
 
 
 
knockknock振りコピ出来ました（笑）
でも別に誰に見せるわけでもなく
披露するわけでもなく親友達と一緒に
覚えてみました( ˊᵕˋ )
でもダンス苦手だから出来てるかは...
完璧に自己満足の世界（笑）
 
 
 
それにこの前は日本に来てくれてたみたいで
何のお仕事なのか分からないけど(TT)
嬉しいな♡これからもずっと応援し続けます！
 
 
 
 
 
 
 
 
そして最近はお仕事がゆったりモード
だったので友達とカフェに行ったりしてました☆
まったり出来て嬉しかったな〜
 
 
 
↑ニコライ・バーグマンのカフェ
店内がお花だらけでとっても素敵
テーブルもお花だらけで
本当に可愛かったな♡
 
この日はスムージーも飲みました( ˊᵕˋ )
 
 
 
 
 
↑キューポットカフェ
期間限定のさくらあじ
凄い可愛くて沢山写真
撮っちゃたんだ〜







 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
 
 
そして昨日の木曜日でおに魂が終了
してしまいました(TT)
最後のエンディングコーナーは
各曜日が全員集まって盛大なものに！！！

 
※アナは遊びに来てくれました☆（笑）






しんみり終わるのはおに魂らしくない
ということで本当に最後の最後まで
わちゃわちゃしてるそんなパーソナリティー
の皆さんが大好きです♡♡♡♡
 
 
すぐにかけつけてくれるリスナーさん
本当に素晴らしい番組の一員になれたこと
本当に誇りに思えます\( ¨̮ )/
 
 
 
 
 
 
最後の最後に鬼にゃんとお喋り出来て良かったな
話した内容は内緒なんだけどね( •ㅈ• )
沢山嬉しい言葉やこれからに向けての
優しい言葉を頂きました
鬼にゃんがいなかったら私ここまで
出来てなかったんじゃないかな？？
 
 
卒業してもずっと鬼にゃんのことは
忘れません大好きです
 
 
 
 
 
 
 
 
そしてNUTTY RADIO SHOW THE 魂
1回目は月曜からはじまります
新しい時代の開幕ですね
ここからまた新しい時をリスナーさんと
築けたらと思っています
まだまだな私なりに精一杯
だんちゃんと頑張って行きたいと
思いますのでよろしくお願いします◎
 
 
 
 
 
 
 
  
 
 
それでは明日の全握で披露する
インフルエンサーも楽しみです
どんな反応してくれるんだろう（笑）
握手は小百合とのペアです( ˊᵕˋ )
 
ぜひ遊びに来てくださいね？
待ってますよ？
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
