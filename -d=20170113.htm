2017/0113Fri斉藤優里何が起こりそう〜

 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
13日の金曜日皆さんはいかがお過ごしですか？
私はね朝、スクランブルエッグ作って
ココア飲みながら録画してた
クレイジージャーニーを
見て過ごしてました( ˙˘˙ )
 
 
 
 
その後に13金企画で沢山の方から頂いた
コメントを見ながらお返事書いたよ！！
何人の方が参加してくれたのって
レベルで沢山の方からのコメント
とってもびっくりしちゃいましたが
凄く嬉しかったです◎
 
 
皆さん参加してくれて本当にありがとう♡
 
 
 
 
 
 
それではお手紙を書いたので
早速載せたいと思います\ ♪♪ /
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
  
 
 
 
 
 
 
 
 
こんなに一気にお手紙書いたのは
はじめてでした( ´･ᴗ･` )
でもとってもやりがいのある企画ですし
ふぁみたんが喜んで頂けるのが1番
なのでこれからも続けて行きたいなと
思います♡♡♡♡♡♡♡♡♡♡♡♡
 
 
 
 
その時を楽しみに今回選ばれなかった方は
また再チャレンジしてみてね？
 
 
 
 
 
 
 
 
次回は10/13でした（笑）
それまで長すぎる（笑）
 
 
世間的には不吉なイメージの日だけど
こういうことや13日の金曜日という
歌をもっと知ってもらってまた
今までとは違った13日を感じて
もらえたらいいな〜
 
 
 
 
 
 
 
それでは明日の握手会
来れる方は色んなお話しようね？
来れない方はんーーーーーーー
心で念じといて（笑）












たんたんたん ゆったんたん( •ㅈ• )
