2014/0430Wed斉藤優里ベンジャミンって前 あったような

 
 
こんにちたーん( •ㅈ• )
 
 
昨日はユアエルム presents with you
SPRING FESTIVALに出演してきました〜♪
 
パフォーマンスありのトーク
ありだったのではぴはぴ
 
 
途中のトークコーナーでは
愛の告白を16人1人ずつやらせて
もらえたのですが....
いや⁈むしろ16人全員も
やらせてもらえること滅多に
ないから物凄くありがたかったです！
 
 
 
...でもさでもさ愛の告白って
ほらー告白して可愛い♡♡♡♡
ってなるメンバーと笑いが起きる
メンバーの2人になるじゃない？笑い
 
 
いや、ほら何て言うか....
たんたんってどちらかと言えば
純粋に可愛い♡♡♡とか言われたいのに
いつも笑いが起きるでしょう？
 
その他のメンバーあげちゃうとさ
(※その子推しの方には申し訳ない
のですが꒰⁎×﹏×⁎꒱՞༘✡泣)
永島さんとかあみも笑いになっちゃう系メンバーじゃない？
 
 
 
だからね、物凄く楽屋で緊張
してたのよ！あたし達っ！
 
 
 
 
滑り芸になりたくないっ！
とか言ってね？\(°Д° )/
これはフォローし合わなきゃと。
 
 
 
結果そこまで笑いが起きる
わけでもなく滑り芸？に
なったのかな？笑い
 
 
 
 
トークは楽しいから喋るのは
好きなんですよ꒰ू๑͒•౪•๑͒꒱
ただただ純粋に可愛い♡♡♡のを
狙って行くのは...ね...
 
ともあれ！パフォーマンスも
出来たので良かったかな〜
 
気づいたら片想いでは若月ポジ！
1番の♪絶対恋なんか〜っていう
歌詞のとこで永島さんと歌えて
嬉しかったな〜♡♡♡♡♡♡♡
 
 
制服のマネキンは以前もやったことの
あるまいやんちゃんポジっ！
 
 
 
またこういうイベントに出演したいです！どこのポジションでもいいので！
個人的に女の子達に
きゃーきゃー言われたのが嬉しい♡
 
 
同性にきゃーきゃーなんて言われる
ことないもんっ٩꒰*´◒`*꒱۶ෆ͙⃛新鮮
手たくさん振ってくれたふぁみたんも
ありがとうございましたっ！
 
 
 
帰りのバスではななとモンハン♢
家帰ってからもななとメイクさん
4人でモンハン♢
家にいる時はモンハン♢
 
 
...もう、それしかやってないよーー笑い
 
 
 
 
 
 
 
 
そして！嬉しくて嬉しくてモバメ
で送ろうかと思ったのですがやはり
たくさんのふぁみたんに言いたくて..
 
 
な、なんとーーーーーーーー！
SEKAI NO OWARIさんのDVDを
頂きました☆☆☆うきゃあーー
 
 
貰った瞬間ジャンプしながら
喜びました！こんな幸せが
あっていいものなのか！
 
まだ時間がなくて見れていないのですが
何回も何十回もお家で見たいと思います！
 
 
うおおおおおおお☆☆
本当にありがとうございます！
 
 
 
そして30日のさいたまスーパーアリーナで開催する
『炎と森のカーニバル -スターランド編-』に行く予定だったのですが
お仕事の都合上行けなくなってしまいました(´;ω;`)うおおお
 
泣ける...。
 
あぁ、グッズだって買いたかったよ。泣
 
大きな声だして応援したかったよ。泣
 
 
 
 
 
 
 
 
 
はぁ...お仕事頑張ります(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
 
ちなみにこれ16人のプリンシパルパンフの
衣装なんだけどきづいた？笑い
 
皆にそれぞれあって全員違うの
着させて頂いてるだよっ！笑い
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2014/0427Sun斉藤優里お風呂に水2リットル

 
 
 
こんばんたーーん( •ㅈ• )
 
 
最近やたらと乃木坂のホームページを
開く回数が増えましたあたし！笑い
 
んー特になぜとか理由はないのですが。
 
 
したらねしたらねっ！
『乃木ここ⁈』がUPされてる
じゃないですか ͛♡꒰ू･･ﾐ ͛꒱ ͟͟͞
早速拝見しましたが.....あたし
たくさん泣いている映像ばかり。笑い
 
そうなんですよね...MCがね
とてもとても悔しかったですし。
他のメンバーもそれぞれたくさん
悔しい部分があったからたくさん
泣いてたんですよね。
 
 
 
でも涙を流した分だけ強くなる！
 
 
 
だから5/3のアンダーライブは頑張りたいです☆
楽しかったって言ってもらえるのは
勿論のことちゃんとあたしを
魅せれるように✩⃛꒰⁎⁍̴◡⁍̴⁎ ॢ꒱うん
 
 
 
 
それと遅くなってしまったのですがメンバーで昨日まあやの誕生日をお祝い♡♡
 
 
まあやねお祝いされるの全然予想
してなかったからすっぴんなんだよ！
 
だから目ぎゅってつぶってるの♡
 
...予想なんて出来ないよね、でも。笑い
あたしはこの日は化粧してたから
ちゃんとぱちくりしてますよっ
 
 
 
みなみの腕を指差しながら
にこにこしております✩⃛꒰⁎⁍̴◡⁍̴⁎ ॢ꒱
 
ちなみにみなみの腕には生ちゃんが
書いたスフィンくんがいるんです。笑い
 
...あれ？これ？何か
隠れミッキーみたいに探すのが
楽しいーみたいになっちゃたぞ
 
 
 
 
 
 
おお...今このブログ見てるあなた！
読み終わったら隠れスフィンくん
探そうとしてるだろう〜
 
バレバレなんだぞっ！
 
 
 
いやそれか！もう読み終える前に
写真をUPにして探し終えたかの2択
しかはいだろう。はっはっはー
 
 
ニット帽似合う♡ってスタイリストさん
に言われてテンションあがってるたんたんおりますよ。笑い
 
 
凄い嬉しくなっちゃて撮影
終わってもしばらくつけてたよ♢迷惑
 
 
 
 
 
 
 
 
 
....実は生ちゃんの腕にはみなみの
ラクガキもあったりなんて。笑い
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
