2015/1223Wed斉藤優里仲良し

 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
ぶろぐをね、書こうと思った時に
書くことが多すぎて何から
書いていいか分からなくて
焦っています。泣
 
 
 
 
ひとまず、アンダラのことを
書こうと思ったけど少し時間が
かかりそうだったので
Merry X'mas Show2015について
書いちゃいますね⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
 
 
 
 
 
 
 
 
ライブは物凄い好きで、
やることはしっかりして
心から楽しむことが出来たもん勝ち
だと思っています♡♡♡
 
 
 
 
煽りの時にファンの皆さんと
わちゃわちゃしたり、
出来るだけ目に付いた団扇や
タオルにはレスを送ったり、
ただ全員にはやっぱり返しきれないので
そこはごめんなさい(´;ω;`)
 
 
 
 
毎回、手作りで作ってくれた団扇を
見るとこの日のために貴重な時間を
さいて作ってくれたんだなって
思うと胸がいっぱいです。
本当にありがとう。
 
 
 
 
 
 
ただ今回のセットリスト4曲目からの
出演だったのでなんだか物足りなさを
感じてしまいました。
18 19日にアンダラをしているから
どうしてもそうなってしまったり
全体のライブだから仕方ないけど
もう少し早い曲から
皆さんの前に立ちたかった、
皆さんに会いたかったです。
 
 
どうしても曲数が少なくなってしまう
アンダーメンバーはその限られた曲の中で
どれだけ自分をアピール出来たかな？
どれだけ応援してくれている方に
笑顔を届けられたかな？っていうことを
ライブ終わりに私は考えたりします。
 
 
 
 
 
 
 
 
どれだけ伝わったのか私には分からないし
その方の感じ方にもよるけど
これからも、私は皆さんに感謝の気持ちと
少しでも応援して下さる
皆さんのためにステージに
立ち続けたいと思います。
 
 
 
 
サンクエトワールの影に隠れてしまって
いましたが実は「隙間」も
初披露だったんです( •ㅈ• )
 
 
 
他の星からユニットメンバーは
ありがたいことに3曲もユニット曲を
頂いているのでこれからも歌わせて
もらえたら嬉しいなって思います。
 
よくばりしちゃうと、他の星からしか
MVがないから次、もしもらえるなら
MVも欲しいな♡なんて....
 
 
 
 
 
 
 
 
12/26に全握でも「隙間」
披露するのでもし良かったら
遊びにきてください♡♡♡
 
 
 
 
 
告知
・12/18 乃木坂物語
・12/23 ALL MV COLLECTION
             〜あの時の彼女たち〜
・12/24 B.L.T
 
 
 
 
 
 
発売中の物もあるしこれからのも
ありますがぜひよろしくお願いします
B.L.Tは乃木坂全員出演ですよ(ˊ•̤ω•̤ˋ)
 
うふふな写真も撮らせて頂きました♡
 
 
 
 
 
 
 
 
 
 
そしてなんと12/25のMステスーパーライブ
歌う楽曲が「太陽ノック」なので
今の選抜＋私とまいちゅんも
参加することになりました⁽⁽ଘ( ˊᵕˋ )ଓ⁾⁾
 
やった、やった嬉しいよ♡♡♡
テレビに出演できますっ！
 
 
 
ぜひ、ふぁみたんには見逃さずに
見て欲しいです♡
見れないって方は今すぐ！
録画の予約を....笑い
 
 
 
 
 
12/25クリスマスの夜に
乃木坂ちゃんをよろしくお願いします♡
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
