2017/0524Wed斉藤優里今夜も一緒に？

 
 
皆さんお待たせしました！
THE 魂(ソウル)シールが
やっと完成しました♡♡♡
 
 
 
 
シール何枚もドーン！って
スタンプ押してるのになかなか
届かなかったよね(TT)
ごめんね、でもねいつもね
シール作るの遅いんだ（笑）
リスナーさんに喜んでもらおうと
考えているとどうも沢山色んな案が
出てきしまいなかなか作るのが
遅くなってしまうのかも？
 
 
 
 
 
これからもどんどん皆さんからの
お便りお待ちしておりますね？( 　˙³˙　 )
そして乃木坂ちゃんから
みなみとまあやが遊びに来てくれました♡
 
 
 
 
今日発売の「生まれてから初めて見た夢」を
4人で持ってぱしゃりと♡♡♡♡♡♡♡♡♡
 
 
 
 
のほほんとした２人らしいトークが
沢山出来て良かったです
そしてTHE 魂ではその日は
乃木坂の曲が3曲もかかけてもらいました！感謝！
醜い私・満月が消えた・君が扇いでくれた
どの曲も素晴らしくて思わず
聴き入ってしまいます( ¨̮ )
 
 
 
 
全タイプそれぞれジャケットが
違うのは勿論、フォトアルバムの
個人写真も全て違いました♡♡♡
是非ともチェックしてね？
衣装のスカート長いメンバーが
8割がたスカートをたくし
上げてました♡♡♡（笑）
 
 
 
 
 
 
そしてそんな今日は
24:00〜  西野七瀬のオールナイトニッポン
超直前スペシャル！をshowroom配信
しちゃいます( ˙˘˙ )
 
25:00〜　西野七瀬のオールナイトニッポン
にも出演しちゃいます♡♡♡♡♡♡♡♡♡
 
2人でのラジオ〜めちゃめちゃ楽しみだ
しかも夜だよ夜（笑）うきうき
ななの楽しい放送を沢山してきたいと
思いますのでよろしくお願いします
 
 
 
 
そして27:00〜はまいちゅんの
オールナイトニッポン0 もあり
あみ ちーちゃん まあやが
出演するのでそちらも
続けてチェックしてね！！！！
 
 
 
寝れない日になってしまうね（笑）
でも頑張ってね（笑）
 
 
 
 
 
それではまた夜お会いしましょうね
あ、まって告知させて（笑）
 
 
 
 
 
♡告知♡



本日発売の「週刊ザテレビジョン」
表紙とグラビアを真夏 なな まっちゃん
蘭世とやらせて頂きました
なんと関東版と関西版で表紙が
違ったりするので見てみてね？
 
 
 
 
チームレモンとしての活躍の場が
今後あるといいな...？？？
凄く楽しい現場でご飯食べながら
取材して頂いたり温かな雰囲気でした！
勝手にお硬いイメージを持っていたので
緊張はじめの方はしていたけど
カメラマンさんも編集者の方も凄く
ニコニコで癒されました〜♡
 
 
是非よろしくお願いします◎
 
 
 
 
 
 
 
あ、これでちゃんとばいばいだ（笑）
それではまた夜にお会いしましょう〜
 


 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
