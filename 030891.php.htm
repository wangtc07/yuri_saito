2016/0312Sat斉藤優里福島県お邪魔しました

 
 
 
 
今日は「震災から５年"明日へ"コンサート」
に出演してきました。
そして、歌わせてもらった楽曲は
「悲しみの忘れ方」
 
 
 
 
 
福島県になかなかお邪魔する
タイミングがなかったので
今日足を運べて、福島県民の方や
東北地方の方にお会い出来て
良かったです\( ¨̮ )/♡
 
 
 
 
帰りに「来てくれてありがとう！」
というボードを持って下さった方が
いたり、あたたかい方達が沢山
いらっしゃいました。
 
こちらこそ、寒い中待っててくれたり
ありがとうございました。
風邪ひかないでくださいね？
 
 
 
 
 
 
「まつり」 「世界に一つだけの花」で
降ってきた花びら達が頭の上に
沢山積もったので記念にぱしゃり♡
 
 
 
 
 
 
私と誰の後ろ姿か分かるかな？(ˊᵕˋ)
 
 
 
 
 
 
 
 
 
 
 
 
 
正解は、ななでした♡♡♡
このぶろぐを書いてる途中に
まだ髪から花びら達が
おちてきました( ´;o;` )
 
 
 
 
 
まさか、まだ残ってるかも...
 
 
 
 
 
 
 
久しぶりに選抜メンバーと一緒に
お仕事出来て楽しかったのは勿論
歌番組にも出演出来たのがなお、
嬉しさ倍増でした( ´･ᴗ･` )
 
 
 
こうやって、沢山いるアイドルや
アーティストの中から選んでもらえて
当たり前のように出演していますが、
本当にこれって凄いことだし
恵まれていると思います。
もっと日々感謝しなきゃいけないですね
 
 
 
 
 
 
 
そして、このコンサートを通じて
改めて昨日という日がとても
大切な日であって忘れてはいけない日
ということを実感しました。
 
 
 
これからも乃木坂46として
1人の人間として、微力ながらですが
私に出来ることをしていきたいと思います。
 
最後になりましたが、
震災により亡くなられた方々に
御冥福をお祈り申し上げます。
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
