2016/0324Thu斉藤優里ラジオのこと盛り沢山

 
 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
最近おに魂のゲストさんが
全然紹介できてなかった
うわ〜ごめんなさい(´;ω;`)
 
 
 
↓どんどん載せてっちゃいたいと思います



Ｘ21の吉本美憂さん
とっても綺麗な方で話し方も
上品で見習いたいとこばかりでした
 
今度はほかのメンバーと
ぜひ来て欲しいなって思います♡♡♡
 
 
 
 
 
 
 
 
ベイビーギャングのお2人は
とても楽しいトークを毎回
ありがとうございます♡
 
そして、りんたろーさんは
彼女さんとは大丈夫だっかな？
これからもずっと彼女さんと
お幸せに( ´･ᴗ･` )
そして北見さんの本は沢山
の方に見てもらえるといいな
私も全部読んで面白かったから
 
 
 
 
 
 
 
 
 
最後は3/23発売のハルジオンが咲く頃
リリースの乃木坂ちゃん♡♡♡♡♡♡♡
初登場のちーちゃん&おに魂準レギュラーの
まいまいが来てくれました( ´›ω‹｀)
 
 
 
 
 
まいまいが凄い残念なんだけど
これで最後かもしれないって時に
別のお仕事の都合で
エンディングからの登場に
なってしまいました...凄い残念です
 
 
でも、これだけじゃ物足りないから
やっぱりまた来て欲しいなって
思っちゃうのは私だけかな？
 
 
↑きっとリスナーさんもそう思って
くれてますよね？\( ¨̮ )/♡
 
 
 
ちーちゃんは見事に楽屋での
私達のじゃれ合いを暴露
してくれましたね...笑い
くっそーーーーーーー٩( ö̆ ) و
にゃんちゃんなの。はモバメとかで
しか出してなかったのに。
くっそーーーーーーー٩( ö̆ ) و
生放送恐るべしっ！
 
 
 
 
 
その日嬉しかったのが手書きでの
メッセージが届いていて
11才の子からでした
 
 
 
 
似顔絵も書いていてくれ
本当にうれしいです
皆でめっちゃ鬼にゃん
似てるねってお話してたの(ˊᵕˋ)
私も凄いラブリーに書いてくれてて
本当に嬉しいです大切にするね？
 
 
 
 
 
こうやってずっーとラジオとかだと
特になんだけどリスナーさんとの絆？
じゃないけどそういうのが
うまれてくるから素敵ですよね( ´･ᴗ･` )
 
おに魂が出来て良かったな〜
 
 
 
 
 
今のお仕事だけに満足している
わけではないけどお喋りが
沢山出来て私は嬉しいです
もっと色んなお仕事も
してみたい気持ちもあるから
これからも、チャレンジ精神！
大切にですね
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
