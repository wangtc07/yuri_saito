2013/0228Thu斉藤優里タン塩3人前で。

 
 
こんにちたーん( •ㅈ• )
 
今日はとあるお仕事の
打ち合わせに行ってきます♡
まだ情報解禁してないので
詳しく言えないのですが
とっても楽しみです。
 
 
3/2もベイFM、3/3もi-BANに
出演します！
嬉しいかぎり( •ㅈ• )
 
 
結果を残せるようにしてこなきゃ。
それと聞いてる人に面白いって
思ってもらえるようにだな。うん
 
 
腹筋がね、ないの。
 
 
...あたし。
 
 
それでね、せいたんが
毎日腹筋してるらしくて
この前ペア組んでストレッチ
してたらめっちゃ出来てて
びっくりした( •ㅈ• )わお
 
 
あたしは出来なくて
7回ぐらいでぐったりしてら
せいたんにしごかれました(´;ω;`)泣
 
その次の日のあたしは
普段使わない筋肉使ったから
痛くて痛くて(´;ω;`)
寝るのにも一苦労。
 
 
 
それを昨日パッと寝る前
思い出したから昨日から
腹筋毎日20回することに
決めました♡
 
自分に甘いかな？..20回w
 
 
 
 
きっといつかあたしの腹筋は
板チョコのようになってるで
あろう( •ㅈ• )おほほ
 
 
 
 
うわーーーなんか想像したら
気持ち悪いw
 
 
せめてウエハースのように...。
 
あれウエハースって板チョコより
編み目細かいな..ありゃま
どうしましょう。
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2013/0223Sat斉藤優里 乃木坂46一歳のお誕生日おめでとう！ メンバー『あ...バブーバブー。』

 
 
こんにちたーん( •ㅈ• )
 
 
うーーー2/22昨日は
乃木坂Birthday Live♡
 
 
幕張メッセに足を運んで
くださった方ありがとう
ございました。
 
今回行けなかった(´;ω;`)
なんて方落ち込まないでね
今度きっと会えますから♡
 
今回のLiveでは普段やらない
ようなことばかりしたので
凄く充実しました。
 
特に『ハウス！』では
客席のほうまで走って
いけて皆さんとたくさん
ハイタッチ出来て嬉しかったです♡
 
 
あたしがお立ち台たったとき
皆さんがたんたんが来てくれて
凄く盛り上がれた！なんて
思ってもらえてたらいいな( •ㅈ• )
 
 
そしてそしてなんと言っても
アンダー楽曲の13日の金曜日♡
本当はふぁみたんに早く
伝えたくてうずうず...
 
 
この度斉藤優里はアンダー楽曲で
センターを務めさせて頂きます♡
 
 
アンダー楽曲でセンターを
目標としてきてあたしは
やっとふぁみたんとの約束を
はたせることができました( •ㅈ• )
 
でも、今はセンターだけれども
次の楽曲では違うかも
しれません。
しっかりと気を抜かないで
いたいと思います。
 
 
だけど、思う存分センターを
楽しみたいと思います♡
みさみさにPVを撮り終わったあとに
『たんたんがセンターだったから
楽しいPVが出来たんだよ。』
と言われて本当に嬉しかった。
 
センターになったからには
アンダーメンバーの顔となるので
責任を持ってもっとアンダーメンバーが1人1人が今よりもっと
輝いていけるような環境に
していきたいと思いました。
 
 
 
 
 
 
えへ( •ㅈ• )ちょっと真面目に
語ってしまった..恥ずかしい。
 
 
 
アンコールでカラーボール
投げてキャッチしてくれた方
ちゃんと飾っといてね♡
 
たんたんボール。
 
ふぁみたんの方がボールキャッチ
してたら凄い運命だな♡
 
 
あ、でもふぁみたんの方で
キャッチ出来なかった方は
今度郵送しとくね( •ㅈ• )
 
 
 
なんてな♡おん
 
 
 
最近行けてなかったディズニーに
今度いってきます♡
つまり、デートってやつだ！
 
 
 
 
なかなかラブラブなんだからな♡
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
