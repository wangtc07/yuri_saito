2017/0325Sat斉藤優里オフショット☆

 
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
昨日はかりんちゃんとディズニーランド
に行ってきました♡♡♡
 
 
 
新エリアの「キャンプウッドチャック」
誘っておいてなかなかタイミングが
あわなくてやっとでした（笑）
その間、彼女はディズニーに何回も
訪れていたんだけど私との約束が
あったので行かずに待ってて
くれたみたいです(TT)優しい
 
 
 
 
 
 
フラッグやリスのマーク？など
各所各所可愛くて特に、
手作り感溢れるフラッグは
最高に可愛くて2ショットを
撮ってしまう程でした〜
 
 
後ね、ドナデジのグリーティングにも
並んだんだけどその間に隠れミッキーとか
いないか探してみたんだけど見つけられなくて...
変わりにリスマーク？を沢山見つけてました
隠れミッキーならぬリスマーク可愛かったな
 
 
 
 
 
ご飯屋さんの内装も凄く凝っていて
とっても素敵でした（笑）
お目当てのワッフルサンドも
メープルソースをかけて食べてみたけど
最高に美味しかったな〜
あれはまた食べたくなってしまう味
 
チキンもさくさく♡♡♡
 
 
 
 
 
本当に堪能出来た1日でした
ちなみにその日はかりんちゃんと
おそろっちの帽子を被ってたの
 
 
 
海外で売ってるやつだよ♡
かりんちゃんはミッキー
私はミニーちゃん
 
 
今度はイヤハも被って行きたいな
次はイースターだな、うさちゃんの耳つけるの
 
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
  
 
 
 
3/23はアップトゥボーイさんの
発売日でした( ¨̮ )
 
 
久しぶりのグラビアでとっても
気合が入って前日もパックして
美容室行って〜るんるん！！！
 
 
 
 
 
仕上がりは皆が思ってくれた
感想をダイレクトに聞きたいな〜
ちょっと怖いけどね（笑）
 
 
 
今回のテーマも結構何か...激しい（笑）
「セクシーな猫になってしまう女教師」
という風にかかれているんですけど
撮影中もそんな感じで楽しみながら
お仕事してました( ˙˘˙ )
 
 
 
 
 
 
 
ちなみに撮影中仮の生徒の名前を
「まもる」と名付けて遊んでました（笑）
 
 
全国のまもる君！！！
沢山名前呼んじゃってごめんね（笑）
 
 
 
 
 
 
 
とっても面白くてグラビアの中でも
かなり楽しめる作品になっているので
ぜひ沢山の方に見てもらいたいです♡
 
 
 
よろしくお願いします☆
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
_
