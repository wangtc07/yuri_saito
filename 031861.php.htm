2016/0423Sat斉藤優里せんべい汁

 
 
 
 
 
 
こんばんたーーーん( •ㅈ• )
 
 
 
 
 
 
 
 
今日ね物販で販売されてる
コーチジャケットをメンバーも
頂いちゃいました♡♡♡♡♡♡
 
 
 
大きさがほどよく女の子が
Sサイズを着るとちょっと大きめに
なるのでシルエットが可愛いです\( ¨̮ )/♡
お気に入りです◎
明日もTシャツの上から羽織るんだ
 
 
 
 
 
 
 
 
 
 
 
 
今日の青森公演でアンダラ
東北ツアーが折り返し地点に
ついに、なってしまいました
 
 
 
 
 
ライブに来て下さる方はその土地に
住んでる方が多いのか
初めましてのお顔の方が物凄く多いです
アンダラは一人一人の顔が、
見えるのがとっても良いところです
だから今まで東京ばかりでしてたのが
勿体無いくらい東北ツアーで沢山の方に
乃木坂を知ってもらえて嬉しいです
 
 
 
 
 
 
 
乃木坂ってどんなライブするのかな？
って思ってる方にはびっくりして
もらいたいから、
アンダラって色々試行錯誤して
スタッフさんやメンバーの意見を
とりいれた熱いライブなんです◎
だからぜひ色んな方に見て頂きたい
メンバーの思いがダイレクトに
受け取れると思います
 
 
 
 
 
 
 
 
 
 
そして毎日撮っていた会場限定シールが
今日はもらえてなくて
ごめんなさい( ´;o;` )
なのでまた後日もらえたら
またちゃんと写真をあげたいと思います
 
 
 
 
 
 
 
 
だから今日はただの
ノーマルな私達...( ˙˘˙ )
↑ぎゅうぎゅう詰め
 
 
 
 
 
しかもねこのソファ2人掛けくらいなのに
皆が座るから椅子の影さえも、
見えない状況に...笑い
2人掛け用に6人で座ればそうなるわな
おしくらまんじゅう状態か！
 
 
 
 
 
 
 
 
青森ではせんべい汁も飲めて
大満足した1日でした
せんべい汁はじめてだったけど
お野菜たっぷりでとっても
美味しかったです♡♡♡♡
 
 
 
 
 
 
 
 
 
 
それでは明日は絢音ちゃんと
生駒の出身の秋田県でライブです
楽しんできたいと思います♡
 
 
 
告知です( ˙˘˙ )
 
「シブヤノオト」4/24 17:05〜
に出演します、ぜひチェックしてみてね
何の曲かはお楽しみに にゃんちゃんなの
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
