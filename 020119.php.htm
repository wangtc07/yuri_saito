2014/0831Sun斉藤優里てるてる坊主の効果 だよね

 
 
 
こんにちたーん( •ㅈ• )
 
 
昨日は真夏の全国ツアーラストは
明治神宮野球場での野外LIVE♦︎
本当にたっーーくさんの方に
起こし頂きました！
 
前日や朝方まで雨マークだったのに
本番になったら雨なんて降らず
綺麗な西日のさすなかでのLIVE
 
夏のFree&Easyで登場した時には
思わず、鳥肌がたってしまうほどの
光景でした。たくさんLIVEで
景色を見てきたけど明治神宮野球場
は本当に1番素敵で.....
これからもずっとあの景色
忘れられません(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
大阪公演からはじまってたくさんの
経験をさせて頂きました！
 
まずはトロッコにのってスタンド席の
皆さんにご挨拶に♡♡♡♡
ずっと乗ってみたかったし
やっぱり距離が近いと
少し照れてしまう。
 
が...何かそのドキドキ感が
たまらーーーーーーーん(ˊ•̤ω•̤ˋ)
やっぱりあたしは皆さんの近くで
パフォーマンスしたりしてるほうが
好きだな〜だからね？トロッコ
お気に入りになってしまった！
 
 
もう一つ、ラジオでの真夏の全国ツアーの見所は？と聞かれた時に
「普段のあたしからは想像
出来ないことをします！」と
お答えしたのですが実は
世界で一番孤独なLoverのゾンビver
の時にまりか かな 若月で側転を
してましたっ❣❣❣
 
よく、「ゆったんってスポーツ
出来なさそう✩✩✩」って
言われるのですがいがいと体
動かすのは得意なんですよ？
 
それに動かすの大好きだから
マラソンとか好き。
唐突に好きなものあげたけど。笑い
 
 
 
 
そんな楽しい真夏の全国ツアーの
お話で締めたかったのですが
ふぁみたんにお伝えしなければ
ならないことがあります☹
 
実は昨日の明治神宮野球場でのLIVE
ゲームのサインボール1000という
企画の時にあたしの不注意で
階段から転落してしまい
そこで右足を痛めてしまいました。
 
 
まだまだ色んな曲数に出演
する予定だったのですが、
無口なライオンと
僕が行かなきゃ誰が行くんだ？
 
13日の金曜日だけは
どうしても出たくて色んな方にお願いして。
この曲だけは皆が優里の曲だからと
言ってくれるのが凄い嬉しくて。
 
出させて頂けることになりました！
気持ちで届けたいと
思って精一杯の自分をぶつけました！
 
 
13日の金曜日おわりにハケて
いるときに右足を庇って
階段をおりていたら、
踏み外してしまい左足も
痛めてしまいました。
自分の鈍臭ささに腹がたち、階段下で
悔して涙がこぼれてしまいました。
 
 
 
そこから最後の挨拶だけ出る形になりました。
メンバーに申し訳ない気持ちや
スタッフさんにたくさん助けて
もらったり、あたしを見にきて頂いてる
皆さんや、全国ツアーをこんな
形で終わらせてしまいました。
 
せっかくメンバーでいいものを
作り上げようと意識も変わって
LIVEに取り組む姿勢も改まって
だったので本当に悔しいです。
 
 
 
LIVE終了後に病院に連れてって
もらい診断してもらったところ
左足が捻挫と右足が1週間みてみて
痛みがひかないようなら靭帯の方に
異常があるみたいなので
MRIで検査になるらしいです。
 
 
 
これからお仕事にも支障を
きたしてしまうし、
皆さんにも心配かけてしまうし
本当にごめんなさい(´°̥̥̥̥̥̥̥̥ω°̥̥̥̥̥̥̥̥｀)
 
 
ですが、明日のラジオも出演するので
そこでまた自分の口からちゃんと
元気なことをお伝え出来たらなと
思ってます！
 
心はいつも通りのうるさいくらいの
お喋りなあたしですよーー？♡♡♡
 
 
 
 
 
 
 
 
それでは、またラジオでお会い
しましょうーー？\( ˆωˆ )/
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
