2016/0918Sun斉藤優里よいしょーー！！！

 
 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
昨日のセブンライブ今日の握手会、
両日とも沢山の方が来て下さって
とっても嬉しかったです
遠いとこから、わざわざ
来て下さる方も至りして
本当に感謝感謝の2日間！！！
 
 
 
 
 
 
 
 
 
 
私のペアは当初、万理華との予定でしたが
体調を崩しちゃったみたいで、
ひめたん かりんちゃんのいる13レーンに
お引越ししました(´·_·｀)
 
 
 
 
 
 
 
 
朝早く来て並んで下さって居た方や
わざわざレーンにまた並び直して
下さった方ごめんなさい！！！
 
 
とってもひめたんのファンの皆さんも
かりんちゃんのファンの皆さんも
とても温かく迎えてくれました♡♡♡
それが、何よりも1番嬉しかったです！
2人もありがとう◎
 
 
 
 
 
 
 
 
 
 
万理華や小百合、そして琴子
本当にお大事に☆
ゆっくり休んでね☆
 
 
 
 
 
 
今度は万理華と一緒に握手ペアに
なれたらいいな〜☆☆☆
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
そして昨日の、のぎ天2を握手会でも
何人もの方に見たよ！と声を
かけていただけました( ¨̮ )
 
 
 
生放送だからなかなか時間が合わないと
厳しいかもしれないけど、
そうやって言ってもらえるのはやっぱり
嬉しかったりします（笑）（笑）
 
 
 
 
 
 
 
その、のぎ天の中で『墓場女子高生』の
配役を発表させて頂きました
 
 
 
 
 
日野..........万理華
チョロ......ちま
メンコ......愛未
ナカジ......優里
ジモ..........絢音ちゃん
ビンゼ......まいちゅん
西川..........小百合
武田..........純奈
 
 
 
 
 
詳しいことまではまだ言えませんが
少しでも観に来て下さる方の
楽しみとかドギドキ♪が
増えたらいいな〜なんて（笑）
 
 
 
 
 
 
 
 
 
まだ稽古でも全然ダメダメだけど
早くふぁみたんにいい所、
魅せられるようにいっちょまえに
なりたいな！！！！！！（笑）
 
 
 
ビシバシ、鍛えてくれる演出の方なので
皆で支えながら毎日取り組んでいます♡
きっと千秋楽の頃にはひとまわり
演じることに対して、考えた方や
舞台にたったときのありかたなど
成長出来着たって自信をもって
言えると思うので、
今はそれを全力で受け止めて
いくしかないですよね？\( ¨̮ )/
 
 
 
 
 
 
 
早く、楽しく稽古に取り組めるくらい
に出来るようになりたいな〜
完成系に近づいてきたほうが
より一層面白味っていうのが
溢れてくるんですよね♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
 
そして先日のおに魂のゲストさんは
CHICO with HoneyWorksさんから
CHICOさんが遊びに来てくださいました！
 
 
 
 
 
 
 
顔出しはまだあれみたいなので.........
右下の細い腕がCHICOさん
の腕になります（笑）
 
 
 
 
腕出しはOKらしいですよ（笑）
 
 
 
 
とっても美味しそうなグラタンのお話
だったり虫刺されの話や.........
勿論、最新曲の『カヌレとウルフ』の
お話もさせて頂きましたが、
２曲ともとっても素敵な曲で
私も思わず携帯にいれてしまいました\ ♪♪ /
 
 
 
 
 
 
 
ちなみに昔にもCHICO with HoneyWorks
としてリリースしているシングル
『世界は恋に落ちている』も私の中で
ずっと聴いている曲でもあります\ ♪♪ /
 
 
 
 
また、ぜひ遊びに来てくだいね？
 
 
 
 
 
 
その日に、鬼にゃんの月曜日卒業が
発表されてしまいました(´._.｀)
最初から何にも分からないトークの
下手くそな私の話にとっても
親切にフォローをいれてくれたり
本当に大きな存在でした。
鬼にゃんありがとうございます！
 
 
 
 
 
こんなにラジオを楽しく出来たのも
鬼にゃんだったからだと思います( ´;o;` )
握手会でも、鬼にゃんとゆったんの
月曜日が大好きだよって言って下さる方も
多く本当に沢山の方が愛してくれました
 
 
 
 
寂しいって思っちゃうけどやっぱり
鬼にゃんは火曜日にずっといるから
もう会えなくなっちゃうわけじゃないから
っと自分に言い聞かせて残りのラジオ
めちゃくちゃむちゃくちゃ！に
楽しんでいきたいです◎
 
 
 
 
 
本当に本当に鬼にゃん大好きだよ\ ♪♪ /
まだまだ入れるヨーデルからの
コールとか沢山やろうね（笑）
 
 
 
 
 
 
 
 
 
 
それでは、おやすみなさい☆☆☆
ぐーすかーぴー☆☆
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
