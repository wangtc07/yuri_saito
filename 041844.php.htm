2017/1114Tue斉藤優里水色

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
先日は乃木恋の彼氏イベントがありましたね
限定100名の方との楽しい時間
しっかりと企画案も出せて頂きました！！！
 
 
 
 
「ソロライブ」をさせて頂いたのですが
いかがだったでしょうか？
やっぱり全体ライブやアンダーライブでも
なかなかソロで歌うこともないので（笑）
せっかくなので今回のイベントでは
お歌大好きとしてはライブを
やってみたかったのでこの機会を
使わせて頂きました♡♡♡
 
 
 
 
 
皆さんはいかがだったでしょうか？
普段、乃木坂のファンの皆さんだったら
きっとお耳苦しいかと思いますが
私のファンの方なら許してくれるかなとも
思い甘えちゃいました( ˙˘˙ )
 
 
 
 
とっても楽しんで盛り上がれた
そんな1日に出来ていたら
大成功かなと思います！！！
 
 
 
 
 
 
 
また、こういうソロライブが
出来る機会があったら開催
したいなと思います（笑）
 
 
 
その時もこれに懲りずにぜひ遊びに来てね（笑）
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
昨日のゲストはTHE 魂に
なってからははじめて！！！
ですがおに魂含めると2度目ましての
「椎名慶治」さん( ¨̮ )
 
 
 
 
 
 
こらでもかーーっ！ってくらい
沢山お喋りさせて頂いたのですが
帰り際に「あーー全然喋れなかった！」
って仰ていて椎名さんが恐ろしかったです（笑）
 
 
また、ぜひ遊びに来てください！
その時もTHE 魂で
喋り倒していって欲しいです（笑）
 
あ〜凄い楽しかったな♡♡♡♡♡
 
 
 
 
 
 
 
 
♡告知♡
 
・11/16 22:54〜23:00 my first baito
 
 
 
そして明日のベストヒット歌謡祭に
出演します、お時間ある方はぜひ
見てください〜☆

 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
