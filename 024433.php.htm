2015/0718Sat斉藤優里帝一の國やっとみれ たの

 
 
 
 
こんにちたーん( •ㅈ• )
 
 
 
 
今日はね今日はねずっーーと
じょしらくの稽古から
行くね！って話してた
帝一の國をみにいってきました♡♡♡
 
 
 
 
 
楽しみすぎてあらかじめ
調べておいたんだ。笑い
したら、笑うところが沢山ある！
って書いてあってうきうきしながら
見てました(ˊ•̤ω•̤ˋ)
 
 
 
 
 
情報通りで本当に面白くて
声をあげながら皆で笑ってたの。
サユリスト本当にお疲れ様です
 
 
 
 
 
ネタバレになっちゃいそうだから
深く書けないのだけど舞台中の
ダンス？曲？がずっーーと頭に残ってて
ついつい口ずさんでしまいます(´;ω;`)
 
 
 
 
.............体勝手に.............笑い。
 
 
 
 
今度はちまのver.もみたいな〜
行ける日があったら行ってみよう♡
ちまver.は映像化されないみたいだから
かなりレア....。うん....行きたい....
 
 
 
 
 
舞台って本当に面白いのね
私もますます舞台欲？っていうのかな
出てきちゃいました....
いつか、さゆちゃんや、ちまみたいに
乃木坂の舞台じゃないのに
出てみたいです(ˊ•̤ω•̤ˋ)♡♡♡
 
 
 
 
 
 
やりたいことの1つだな〜
いつか実現させたいな
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
告知です
今日の夜23:30〜ケータイ大喜利！着信御礼
が放送されますよ〜皆チェックしてね？
 
 
 
 
そして初森べマーズが放送されていますが、
当初じょしらくが忙しくて出れないと
思っていたのですが急遽出演することに
なりました。やった。
 
 
 
 
 
何話とは言えないのですがいつか
私も出演するのでよろしく
お願いします♡♡♡
 
 
 
 
 
 
 
 
それでは明日はめーちゃんの
ラジラーに出演するので
よろしくお願いします！
生放送ですよ、生放送♡♡♡
 
 
 
 
 
 
 
 
夜ご飯に大学芋食べれた
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
