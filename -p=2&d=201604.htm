2016/0420Wed斉藤優里優里の牛タン

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
元気っ子同士の北野とだよ
何だかこの子この写真ではちょっと
ムッて顔してる٩( ö̆ ) و
北野なんでムッ٩( ö̆ ) وなの
なんでムッ٩( ö̆ ) وなの
可愛いからいいけど٩( ö̆ ) و
なんでムッなの٩( ö̆ ) و
にゃんちゃんなの◎
 
 
 
 
 
 
 
 
 
今日の仙台のライブのことは
本日中にブログにかけました♡
良かった良かった〜
昨日は日付がまわっちゃてたからね( ´;o;` )
ちょっとだけそういうとこ
気にしてたりします...笑い
 
 
 
 
 
 
 
ライブ前の影ナレをちーちゃんと
伊織でさせて頂きました
いつも影ナレでは皆さんが、
お行儀よく返事を返してくれて
いつも可愛いなって思いながら
聞いていたので今日も楽しみでした( ´･ᴗ･` )
仙台に来てくださった皆さんも
大きな声で返事してくれて嬉しかったです
 
 
 
何だかこの時ばかりは
幼稚園の先生になった気分に
なっちゃうの♡♡♡♡♡♡♡
↑この気持ちはきっと
メンバーとしか分かり合えないかも 笑い








今日は昨日と環境も変わったので
また新しい気分でライブが出来ました
昨日とは違う気持ちの作り方をして
仙台公演のぞみました。
1日1日ふぁんの皆さんでその時の
空気も変わるのでつねに変化あるものが
楽しくて仕方ありません
 






明日はまたどんな雰囲気になるのか
どんな風にトークをしようかなど
考えたりするとわくわくなんですよ( ˙˘˙ )
特に皆でとあるMC部分での
話し合いをしている時が1番楽しいです!!!!!!
まるで動物園にいるかのように
いっせいに喋りはじめるから
収拾がつかなくなるの...
でも、それほど元気ってことだよね!!!!
↑ポジティブに考えたまる◎
でも本当皆さんにきいてもらいたいくらいだよ

 
 
 
 
 
 
 
メンバーがわちゃわちゃしながら
話し合いするほど盛り上がるトーク？
もうライブを見てくださった
皆さんならわかるかも？
 
 
 
 
 
 
 
 
「ふぅ〜♡♡♡」って沢山、
言いたくなっちゃうようなとこですよ？
↑これからのアンダラでもここ期待してて良いですよ
私もメンバーの見てるとめちゃめちゃ面白くて
笑っちゃうもん\( ¨̮ )/♡

 







それとね、毎日その会場のシールを持って
写真を撮ろうねってめーちゃんと話したので
毎日ぶろぐに上げたいと思います♡♡♡
その時その時に撮れるメンバーで
撮影しちゃってるので毎日メンバーが
変わる変わるかもしれないけど
皆に映ってもらえるように
頑張ろーーーーーっと♡♡♡


 
 
 
 
 
 
 
今日も楽しい1日をありがとうございました
ちょっとでも私達のライブに来て
良かったなとか、元気になれた！って
思えてもらえたらなによりです♡♡♡
 
 
 
 
 
私はそのためにステージに立っています。
皆さんと楽しい気持ちを
一緒に味わいたいから、
それで来てくださった皆さんが
笑顔になれるんだったら
体がボロボロになってもいいから
最後まで全力でパフォーマンスします
 
 
 
 
明日も楽しむぞ！おう♡♡♡
 
 
 
 
 
 
 
 
 
 
 
告知です\( ¨̮ )/♡
明日4/21発売のBRODYで
ちょっこだけグラビア？を
させて頂きました
他のメンバーも沢山載っているので
ぜひチェックしてみてください◎
 
とあるもの？とある乗り物？と一緒にね♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

2016/0420Wed斉藤優里コシヒカリ

 
 
 
 
 
 
 
 
 
 
 
いぇーーーい、にゃんちゃんなの( •ㅈ• )
 
 
 
 
 
 
 
 
 
昨日からはじまりました
アンダーライブ東北ツアー
初日は福島に行ってきました〜
 
 
 
開演の直前まで皆でリハしてて
どたばたの準備でどうなることやら
でしたが、なにはともあれ
初日無事終了しました(ˊᵕˋ)
 
 
 
 
 
 
 
 
 
福島の印象トークで
「強面のおじさん」のお話を
させて頂きましたが、
乃木坂46時間TVでまいちゅんと純奈と
とうほう・みんなの文化センター
(福島文化センター)にポスターを
貼らせて頂いたお話もしました♡♡♡♡
 
 
 
 
その時ポスター見てくれたか調査
をしたところ半分くらいの方しか
手が挙がらなくて...\( ˙-˙ )/\( ˙-˙ )/
絶対見て帰って下さいねって
言ったけど見てくれたかな？
どうなんだろう？
 
 
 
 
 
見忘れたーなんて方もきっといるだろうな...
そんな人はにゃんちゃんなんだから。
↑特に意味はないです。










いつもとは少し違うライブだったから
戸惑ってしまう方もいらっしゃた方と
思いますがこれも新しいチャレンジだと
思いますし、変化あるものを沢山
お届けしたいのでどうぞ最後まで
よろしくお願いします( ˙˘˙ )





このツアーが終わるころには、また
アンダラに魅力が増えたのかな
と思うと何だかわくわくします
そして自分でもまた何か、
見つけられたらいいなと思いつつ
一公演一公演新鮮な気持ちで挑みたいです
 
 
 
 
 
 
 
 
福島のライブに来てくださった皆さん
あたたかい声援をありがとうございました
楽屋でのスタッフさんとの会話でも
あたたかいというお話が
沢山出ました♡♡♡♡♡
 
 
 
 
個人的になっちゃうけど嬉しかったのが
沢山コールで名前を
よんでもらえたことです( ´›ω‹｀)
会場との距離が近いからかな？
声がダイレクトに私に届くの
呼んで下さった方、本当にありがとう
 
 
 
 
 
 
 
 
 
また絶対会いに来たいです
幸せな一日をありがとう福島
 
 
 
 
 
 
 
 
 
今日は仙台でのライブ！
昨日の反省点をいかして、
よりよいものを皆で
作り上げていきたいな...
 
 
 
そして全員で東北ツアー完走だ！
ファンの皆さんも一緒に
燃え尽きましょうね〜完全燃焼！
 
 
 
 
 
 
 
 
おう！















たんたんたん ゆったんたん( •ㅈ• )

お見せすることが出来ませんでした

間奏がダンスパートみたいな

なっているんです٩(´0`)۶

いいなって思ってます

パフォーマンス上がったね
って言ってもらえるように
頑張りたいです♡♡♡♡♡

↑嬉しかったし楽しかったよ
まっちゅんとは先日犬会があって
わちゃわちゃしたりしてたから
そんな感じの楽しいうきうきした握手になったよ






皆さんと沢山のお話したけど

個人PVのお話しで片腕が
マシンガンがついてる役だったので
「ばーーーーん！」って握手会に

まっちゅんはガチャ子さんの印象的な
決めポーズ？「ガチャリンコ」をして
擬音だらけの握手会になりました◎
 
 
 
 
 
 
 
 
 
 
そして個別握手会では今回4人の
生誕祭が行われたよ( ´･ᴗ･` )
てぃーちゃん・まあや
めーちゃん・まいまい
 
 
 
 
めーちゃんとまいまいのは
時間が無くて行けなかった残念( ´;o;` )
皆お誕生日おめでとう♡♡♡
 
でも、まいみんが作ってくれた
エプロンと眼鏡はまいまいから頂きました！！
まいみんの皆さん凄いセンスが良くて
大切に使わせて頂きますね♡♡♡♡♡
 
 
 
 
 
 
それでは明日からのアンダラ
今日のおに魂感謝祭
全部全部、全力で頑張るぞ！
うわーーーーーい楽しみだ！
ツイキャスも配信されるので
お時間ある方はぜひチェック
してみてください♡♡♡♡♡
 
 
 
 
 
 
 
 
最後になってしまいましたが
握手会で今回の地震の被災者の方へ
向けての募金箱を設置させて頂きました。
沢山の方にご協力頂きました。
本当にありがとうございました。
 
いっこくも早くいつも通りの暮らしに
なれることを願っています。
それと家族同然のわんちゃんや猫ちゃんなど
早く飼い主の元に戻れますように...
