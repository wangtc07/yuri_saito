2017/0426Wed斉藤優里ピンクのステップ

 
 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
ずっと前にこの写真撮ってたんだけど
貼れずにいたよ〜
服装とかさよく見るとちょっと前の
握手会の時とかだもんな（笑）
 
 
3期生の珠美ちゃんと蓮加ちゃん♡♡♡
写真撮るのはじめてだったしなんなら
3期生と撮るの自体はじめてだ（笑）
 
 
 
 
 
 
まだまだなかなか深くお喋りとかは
出来てないから1人1人と仲良しに
なりたいな〜皆可愛いの( ´･ᴗ･` )
小さい子可愛い好きーー♡♡♡♡
 
 
 
 
 
 
そして先日の個握もありがとうございました
最近ますますはじめまして！の方が
増えてきて本当に嬉しいです( ¨̮ )
 
 
 
もっと色んな方と色んなお喋りしたいの
コミュニケーションとるのめっちゃ好きよ☆☆
知らない人を知るのが好き
この人はこういう考え方なんだとか
何でそこに辿り着いたんだろう？とか
面白いよね人間って（笑）
 
 
 
仲良くなると言葉遣いとか
ちょっと悪くなっちゃうんだけど
そこはご愛嬌で♡（笑）
 
 
 
↑三つ編み..これ実はただの三つ編みじゃないの（笑）
私が持っている方はかずみんが編んでくれました♡

メイクさんに「髪の毛変えてくださーい！」
って頼んでたら丁度セットしてた
かずみんとメイクさんが一緒に返事してくれて...（笑）


きっとノリで返事してくれたんだろうなって
思ってたらまさかのまさか！！！！
私の髪の毛本当にセットしてくれました♡
ありがとう、かずみん♡





そして今回もラブリーちゃんなお花
全部全部大切で可愛いの
 
 
 
韓国のファンの方からもお花を頂いて...
大好きなTWICEの新曲
knockknockという文字まで
入っていたんです嬉しい〜(TT)♡♡♡♡♡
 
 
 
 
 
また握手会で皆さんに会えるの楽しみにしてます
アルバムの握手会もはじまって
きっとこちらは歌衣装だと思うので
また新鮮かななんて思ってます( ¨̮ )
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
そして先日のTHE 魂(ソウル)には
上野優華さんが遊びに来てくださいました！
 
 
 
 
とっても可愛い19歳！！！
自称人見知りと言ってましたが全然
そんなことなくてお喋り大好きなの
かと思わせるくらいのコミュ力でした☆
 
 
また絶対遊びに来てほしいな〜
 
 
 
 
♡告知♡
