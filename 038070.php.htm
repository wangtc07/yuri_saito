2017/0414Fri斉藤優里猫ちゃんパジャマ

 
 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
※1枚目の写真はなかなか最近
やってなかったエア握手だよ〜
来れなかった方はぜひ（笑）
 
 
 
 
 
 
 
 
 
この前の握手会は個握！！！
「インフルエンサー」
はじめての5部制で未開拓地だったので
正直ファンの人には
「どうなるか分からない〜（笑）」
とかうきうきしながらお喋りしてたの♡
 
 
 
 
 
いざはじまってみて、沢山ふぁみたんが
応援に駆けつけてくれたりしたし
体力も残っていて元気でした( ˙˘˙ )
改めて5部性になってみては本当に楽しいし
ふぁみたんとお喋りしてたら私も元気
貰えるからいつも感謝してます！！！！
そして有言実行出来たことそれが何より
嬉しいですありがとう！！！
 
 
 
 
 
それからいつも毎回本当に沢山のお花
ありがとうございます、どれも可愛くて
嬉しいですしテンションもあがります
1枚の画角にに収まり切らないほど♡♡
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そして、THE 魂(ソウル)がはじまって
更に今まで以上にリスナーさんと
関われる機会が増えました！！！
 
 
 
魂Qクイズがふつおたからいきなり
選ばれたり...新しいチャレンジ
ばかりしているとても楽しい番組☆
 
 
 
リニューアルしてからリスナーさんは
戸惑ってしまうかなって心配だったのですが
皆さんのみこみがとっえも早くて
色んなボケ回答をもらえてありがたいです
より一層リスナーさんとの絆が深まりそうな
内容の番組になっていますので良かったら
皆さんぜひ聴いてください！！！！！！
いつでもお待ちしております( ´･ᴗ･` )
 
 
 
 
 
 
※ラジオだから出来ること（笑）
たまに私アカペラで歌ったりしてます（笑）
 
 
 
 
 
そんなTHE 魂(ソウル)1回目のゲストは
だんちゃんと仲良しのRaychellさん！！
 
 
 
 
とっても綺麗だし身長高くて本当に
同じ女性として憧れる所ばかりです
とっても気さくで楽しくお話出来て
とっても嬉しかったです( ˙˘˙ )
 
またぜひ遊びに来てくださいね？
 
 
 
 
 
 
 
そして2回目のゲストさんは
Nakamura Emiさん！！！
 
 
 
とってもパワフルな元気溢れる方でした
笑顔ステキ( ˙˘˙ )( ˙˘˙ )
手書きのメッセージ本当に嬉しかったです
大切にさせて頂きますね？
 
 
またぜひ、遊びに来てくださいね？
 
 
 
 
 
私は明日から「インフルエンサー」の
キャンペーンの為広島に行ってきます
広島の皆さん仲良ししてください♡
 
 
 
 
 
 
♡告知♡
 
 
 
 
4/14 BLTグラフ

放送予定日時4月20日（木）20：10〜
