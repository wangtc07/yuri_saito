2016/0719Tue斉藤優里苦手な食べ物克服中

 
↑バナナミルクシェイク♡♡♡♡♡♡
 
 
 
 
 
 
 
 
こんにちたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
「バナナ」が食べれるけど
あんまり自分からは
チョイスしなかったんだけど、
「バナナミルクシェイク」を
飲んでみたらあらま〜びっくり！！！
 
めちゃめちゃ美味しかったんです♡♡♡
 
 
 
 
 
ふぁみたんにもきっとあるよね？
そういう食べ物だったり
飲み物だったり.......
 
 
結構食べられるけど別に自分で
選んだり買ったりしないな〜
なんてもの、好き嫌いはあんまり
ないけどそういうのは多かったりするな
 
 
他だと酸味が強い系はあんまり
ご飯でも選んだりしないかな〜
 
 
 
しゃぶしゃぶはポン酢より胡麻ダレ派！！！
 
 
 
サラダもイタリアンドレッシング
より胡麻ダレ派！！！！
 
 
 
 
 
 
胡麻ダレ最強！！！！
 
 
 
 
 
 
 
 
そういうものがあったら
ぜひ教えて欲しいな( 　˙³˙　 )
握手会とかのお喋り内容に
出来たりするしね！！！！
よろしくお願いします〜
 
 
 
 
 
 
 
 
 
そして昨日のおに魂では
ちょっと早めにお誕生日を
お祝いしてもらいました( ˙˘˙ )
 
 
 
 
まずは前回のマル決司令にして
サリオちゃんが出した
あいうえお作文を見事に正解したら
ペイジさんがガラスの外コンコン！
しに行くという約束を本当に守って
昨日、コンコンしに来てくれました♡
 
 
 
 
 
ペイジさん赤ちゃんが産まれたというのに
面会時間15分で切り上げて、おに魂に
来てれました(´._.｀)申し訳ない 
でも、めちゃめちゃ嬉しかったです
本当にありがとうございます！！！（笑）
 
 
 
そしてSHINOBUさんご出産
おめでとうございます！！！
お身体ゆっくり休めて欲しいです(´._.｀)
 
 
 
ペイジさんの赤ちゃんにいつか
会わせて欲しいです♡♡♡♡♡♡♡♡♡
 
 
 
 
 
 
 
なったんからもわざわざ、歌と
プレゼントなんとお手紙まで
頂いちゃいました\( ¨̮ )/♡
 
 
なったんと同じ匂いのボディコロン
大切に使わせて頂きますね
 
 
 
お手紙もとっても心がこもっていて
ほっこりしたの！！！！
やっぱりお手紙って嬉しいね
 
 
 
 
 
 
そして昨日のゲストは「SKY-HI」さん
 
 
 
 
 
私と鬼にゃんとリスナーさんが
リクエストした単語で即興ラップを
作って頂きました( ˙˘˙ )凄い！！！
 
 
私は「誕生日」というワードを
いれてもらいましたが、
ラップのプレゼントをしてもらったのは
SKY-HIさんがはじめてでした♡♡♡♡
 
 
ラップをしている姿本当に素敵で
かっこよかっです！！！！
沢山、ナナイロホリデー
聴かせて頂きますね\( ¨̮ )/♡
 
 
 
 
 
 
 
それでは、おに魂で頂いたケーキと
ばいばいだよ〜これ本当に幸せ味
おいちかったまる◎
 
 
 
 
 
最後になってしまいましたが、
イジリーさんご結婚
おめでとうございます！！！
奥さんとずっとずっと仲良しで
いて欲しいです( ´･ᴗ･` )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
