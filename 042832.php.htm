2018/0112Fri斉藤優里阿妹茶樓

 
 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
台湾旅行第2弾のブログになります（笑）
その次の日の日は「九份」に行ってきました
千と千尋の神隠しの舞台となった場所！！！
 
 
あのね、台湾来る前に千と千尋の神隠し
復習しとけば良かったなって少しだけ
後悔しちゃった（笑）
より一層、そっちのが世界観とか
楽しめたのかなって(TT)
 
 
 
 
 
 
 
 
 
 
でも、日本にはない建物の形とかが
沢山見れてとても新鮮でした
湯婆婆の屋敷の「阿妹茶樓」でも
お茶もしてきました☆
 
 
 
作法を日本語で教えてくれたの！
分かりやすかった！それに
香りを嗅ぐためのコップがあるなんて
知らなかったの色々、学べたよ！
 
 
 
 
 
そして何よりお茶が美味しくて美味しくて
仕方ないのよ、お茶菓子も最高♡♡♡♡
甘い梅干しが初体験だったから
少しびびってたんだけど
全然、美味しかったな( ˙˘˙ )
 
お茶の作法を教えてくれるお兄さんが
日本語凄く上手で、丁寧に教えてくれたんだ！
感謝☆☆☆
 
 
 
 
 
↑階段の真ん中に座ってたわんちゃん♡
全然人が周りを歩いてても気にせずでした（笑）


 
 
 
 
とっても楽しくて休みを満喫出来ました
また遊びに行ってみたいな
かき氷食べ忘れちゃったから
今度行く時は忘れずにだな〜
 
 
 
 
 
そして本日のMステに出演します
「いつかできるから今日できる」
ぜひチェックしてみて下さい！！！
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
