2017/1221Thu斉藤優里手を繋いでね

 
 
 
 
 
 
こんばんたーん( •ㅈ• )
 
 
 
 
 
この写真ねめっちゃお気に入りなの
昨日発売のB.L.T.さんでのグラビアなんだけど...
今回は着物を着させて頂きました♡♡♡♡
 
 
 
 
この日の撮影は外だったので
めちゃくちゃ寒かったけど
皆で暖かい飲み物飲んだり
カイロ持ちながら寒さをしのいだよ（笑）
 
 
 
1番寒かったのは皆でイチョウを持って
投げたりもしたんだけどその時に
手が本気でカチコチになったの（笑）
 
 
 
 
めちゃくちゃラブリーな着物
着せてもらえて嬉しかったな( ˙˘˙ )
ぜひ、B.L.T.チェックしてみてね？
 
 
 
(※この写真も結構お気に入り☆
皆で覗き込んでるのめっちゃ可愛い（笑）
何か自分が小さくなった気分（笑）)

 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
そして先日のTHE 魂ゲストさんは
カラテカ 矢部さん☆
 
 
 
 
お話していてめちゃくちゃ
ホッコリしました〜( ´･ᴗ･` )
何か、ぽかぽかしてくる感じ
現在発売中の漫画「大家さんと僕」も
凄く気になりました、読んでみたいな
ぜひまた遊びに来てください〜☆
 
 
 
 
 
 
 
それでは明日のMステスーパーライブにも
出演しますので見てね（笑）
そして私にとっても嬉しいことが...♡
TWICEさんと初の共演♡
凄く愛らしくて可愛いこと
間違いなしなので素敵な時間になることを
楽しみにしています( ˊᵕˋ )
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
