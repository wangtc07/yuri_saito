2017/0830Wed斉藤優里冷たいお茶

 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
逃げ水の名古屋全国握手会が
ありました〜！！！！！！
そこでちゃんとフルではじめて
「アンダー」を披露させて頂きましたが
いかがだったでしょうか？
 
 
 
 
 
 
歌詞がなかなか胸にグッと
来るものだったりしますよね.....
アンダーを応援してくださってる方
なら尚更かな？
 
 
 
私、今回のシングルの中で
「女は一人じゃ眠れない」が
一番好きです♡♡♡♡♡♡
すっごい、かっこよくて
踊れはしないけどモニター見ながら
勝手に覚えようとしてました（笑）
 
 
 
 
 
これからの全握もまだあるので
そこで覚えきれたらいいな〜（笑）
きっと私ダンス覚える能力も
低めなんだけど...（笑）
 
 
 
 
 
 
 
 
そして今回のペアはいおりでした♡
 
 
 
 
いおりとは今までペアで
お仕事とかもあんまり
したことないけど、とっても
まったりのペースで楽しめました！！！
 
 
 
 
 
 
2人でレーン行く最中に
たわいもないお話をしたりするのが
何だか良かったな♡♡♡♡♡♡♡♡
 
 
乃木坂の曲はどんな曲携帯に入れてるか〜
とかポスターは部屋に貼るor貼らないか〜
とか盛り上がりました( ˙˘˙ )
 
 
 
 
 
またペアになった時はぜひ
よろしくお願いします♡♡♡
 
 
 
 
 
その日はまっちゅんの誕生日でも
あったので、まいちゅん発信から
犬メンでサプライズパーティーをしました
 
 
 
先にまっちゅんのお部屋に忍び込んで
皆で飾り付けをしました♡♡♡♡♡
 
 
 
 
風船を膨らますの凄く大変だったけど
皆で一気にフーフーしてる光景が
凄く面白かったな( ´›ω‹｀)
 
 
 
 
 
でね、ぎゅうぎゅうに洗面台の所に
隠れてまっちゅんが入ってきた瞬間
バースデーソングとケーキを持って登場っ！！！
本当にびっくりして喜んでくれてたから
大成功して嬉しかったな〜
 
 
 
その夜はUNOとか人狼をひたすら...
犬メンはその2つのゲームに
はまってるからそれがあれば幸せです（笑）
 
 
 
 
 
 
 
まっちゅん本当に本当に
お誕生日おめでとう
25歳幸せな1年を過ごしてね〜
 
 
 
まっちゅん、大好きよ♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )

皆映ってくれてありがとう！

(※誰かは忘れました（笑）ごめんちゃい)

うん、何か寄り目でよかった♡♡♡♡♡♡♡♡

写真なんて撮ることなかなかないから良かったな☆)

パーソナリティはペイジさん！！！！
 
 
もう、なんなら月曜日に
いつも来てくれるのは
私達2人が大好きってことで
大丈夫ですよねきっと〜（笑）
 
 
 
その日のゲストは「HAN－KUN」さん
 
 
 
とってもかっこよくて
お会い出来て凄く嬉しかったです
だんちゃんとも仲良しそうで
とっても羨ましかったな〜( ˙O˙ )
 
またぜひ、月曜日に遊びに来てください♡
 
 
 
 
 
 
名古屋公演が終わり木曜日の
プラスワンパーソナリティとして
お邪魔してきました〜
 
サリオちゃんペイジさん
お世話になりました
 
 
 
罰ゲームは本当に最悪でした！！！！
この一言に尽きる（笑）
あの匂いは嗅ぐもんじゃないよ...（笑）
 
 
 
 
 
とっても楽しかった？「魂フェス」
夏ということで皆さんとお祭り騒ぎ
が出来たことが何より楽しかった
そして嬉しかった\( ¨̮ )/
 
でも、やっぱりあの匂いだけは許せない（笑）
 
 
 
そんな「魂フェス」でした（笑）
