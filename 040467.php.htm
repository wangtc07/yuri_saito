2017/0830Wed斉藤優里冷たいお茶

 
 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
逃げ水の名古屋全国握手会が
ありました〜！！！！！！
そこでちゃんとフルではじめて
「アンダー」を披露させて頂きましたが
いかがだったでしょうか？
 
 
 
 
 
 
歌詞がなかなか胸にグッと
来るものだったりしますよね.....
アンダーを応援してくださってる方
なら尚更かな？
 
 
 
私、今回のシングルの中で
「女は一人じゃ眠れない」が
一番好きです♡♡♡♡♡♡
すっごい、かっこよくて
踊れはしないけどモニター見ながら
勝手に覚えようとしてました（笑）
 
 
 
 
 
これからの全握もまだあるので
そこで覚えきれたらいいな〜（笑）
きっと私ダンス覚える能力も
低めなんだけど...（笑）
 
 
 
 
 
 
 
 
そして今回のペアはいおりでした♡
 
 
 
 
いおりとは今までペアで
お仕事とかもあんまり
したことないけど、とっても
まったりのペースで楽しめました！！！
 
 
 
 
 
 
2人でレーン行く最中に
たわいもないお話をしたりするのが
何だか良かったな♡♡♡♡♡♡♡♡
 
 
乃木坂の曲はどんな曲携帯に入れてるか〜
とかポスターは部屋に貼るor貼らないか〜
とか盛り上がりました( ˙˘˙ )
 
 
 
 
 
またペアになった時はぜひ
よろしくお願いします♡♡♡
 
 
 
 
 
その日はまっちゅんの誕生日でも
あったので、まいちゅん発信から
犬メンでサプライズパーティーをしました
 
 
 
先にまっちゅんのお部屋に忍び込んで
皆で飾り付けをしました♡♡♡♡♡
 
 
 
 
風船を膨らますの凄く大変だったけど
皆で一気にフーフーしてる光景が
凄く面白かったな( ´›ω‹｀)
 
 
 
 
 
でね、ぎゅうぎゅうに洗面台の所に
隠れてまっちゅんが入ってきた瞬間
バースデーソングとケーキを持って登場っ！！！
本当にびっくりして喜んでくれてたから
大成功して嬉しかったな〜
 
 
 
その夜はUNOとか人狼をひたすら...
犬メンはその2つのゲームに
はまってるからそれがあれば幸せです（笑）
 
 
 
 
 
 
 
まっちゅん本当に本当に
お誕生日おめでとう
25歳幸せな1年を過ごしてね〜
 
 
 
まっちゅん、大好きよ♡♡♡♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
