2016/0528Sat斉藤優里卵ポンポン

 
 
 
 
皆がブログにＨappy Easterの
写真載せてたから私も載せたく
なっちゃた〜( ˙˘˙ )
 
 
 
 
 
 
てぃーちゃんのブログに
卵投げつけ合う行事かな？
みたいなこと書いてあったけど...
違うよね？（笑）（笑）
 
 
 
 
 
 
どっかのトマト祭りみたいな感じなの？
イースターの由来は分からないけど
めためた可愛い行事！！！すき！！
もう1回このメンバーで
ディズニー行きたいな
 
 
 
楽屋でね次は、
上海のディズニー行ってみたいね
って最近お話してて、動画見たりすると
もうダメだよね(´·_·｀)
行きたくて行きたくて我慢出来なくなる
 
 
 
 
 
 
カリブの海賊？的なものが
1番私的にヒットしてて
体感してみたいの♡♡♡
うう、行きたいな行きたいな〜
 
 
 
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そしてこの前のおに魂は
「それぞれの椅子｣に伴って
かりんちゃんとちまが
来てくれました〜
 
 
 
 
嬉しい嬉しいよ、メンバーが
来ると全然緊張しないんだもん
それに今回は大好きなちまと
かりんちゃんだったから
話が弾んだな\( ¨̮ )/♡
 
 
 
 
 
ちまのおに魂ネームも決めたんだよ
「ちまんちゅ」これからリスナーさんは
ちまのことは「ちまんちゅ｣って
呼んであげてね（笑）（笑）
 
 
 
 
また絶対遊びにきてね
私と鬼にゃんは待ってるよ〜
 
 
 
 
 
 
それといつも沢山のリスナーさんに
支えられてます( ´･ᴗ･` )
ありがとうございます！
 
 
↑こんなお便り沢山届いたよ
にゃんちゃんなの
 
 
 
 
 
 
 
 
嬉しいお仕事がまた出来そうです！！！
にやにや、早くふぁみたんに告知が
したいです( ´･ᴗ･` )
 
 
 
最後になっちゃたけど
なな、かりんちゃん
お誕生日おめでとう！！！
 
 
 
 
 
それではまた更新する時まで♡
 
 
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
 
 
 
 
 
 
 
 
 
 
 
 
 
 
_
