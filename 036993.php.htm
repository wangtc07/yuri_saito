2017/0219Sun斉藤優里私はピンク×白

 
 
 
 
 
 
 
 
こんばんたーーん( •ㅈ• )
 
 
 
 
 
 
 
ライブ前日になっちゃいましたね
何ですけど私の書きたかったこと
本当に呑気なんです
ごめんなさい（笑）
 
 
 
 
18日放送のズームイン！！サタデーさんの
「中高生に人気のポーズTTポーズとは？」
というコーナーがあったのですがそこで
私のTTポーズの写真が使われたんです♡♡♡
 
乃木坂のことではなくて
TWICEに少しでも形は違えど
こういう形で関われるなんて
本当に嬉しくて思わずブログに
書いちゃいました（笑）
 
 
 
 
 
 
 
次からは乃木坂のことでも沢山
取り上げてもらえるようにもっと
頑張らなきゃですよね（笑）
 
 
それでもやっぱりTWICE大好きだから
本当はTWICELANDのライブも
行きたかったの(TT)
カードキャプターさくらちゃんの
「Catch You Catch Me」という曲も
カバーしていたみたいで大好きなもの同士の
コラボってもう最高じゃないですか！！！！
 
 
あーーもう次ある時には絶対見に行かなきゃ！
これは絶対絶対♡（笑）
ライブグッズも手に入れたかった
やっぱグッズってあるのとないのとじゃ
だいぶ違いますよね、欲しかったな(TT)
 
 
 
 
 
TWICEの新曲がもうYouTubeで
見れるようになってました
「KNOCK KNOCK」
まいちゅんとれいかと一緒に
見ちゃいました◎
 
 
 
 
 
 
ツウィちゃんにいつか会えますように♡
私ももっと頑張ります（笑）
 
 
 
 
 
 
 
そんな私たちのライブのリハの合間には
肩を出した服を着ていたら落書き
されちゃいました（笑）
 
 
誰が書いたかわかりますか？
 
 
 
一番上が純奈 うさちゃんはてぃーちゃん
オレンジ君と三つ葉はかりんちゃん
 
何だか愛着が湧いちゃってその日
1日中落とさなくてお家帰ってから
お風呂に入って落としたよ（笑）
 
 
 
 
 
 
そんな感じで楽しくて毎日リハしてました
ななみんのラストのライブだから
本当に素敵なものにしたいし
まだ全然実感は湧かないけど
ファンの人にも悲しいけど
楽しんでもらいたいな( ´･ᴗ･` )
 
 
 
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
 
 
そしてちょっと前のおに魂の
「松岡卓弥」さんが遊びに
来てくださいました！！！
 
 
 
弟さんが乃木坂のことが大好きで
いてくれたみたいで本当に嬉しいです
推しは違いましたが（笑）
 
ぜひライブにも遊びに来て欲しいです♡
 
 
 
 
 
 
 
そしてこの前のおに魂は
「家入レオ」さんでした☆
 
 
 
凄く明るくて面白い雰囲気の方で
本当にお話していて楽しかったです
驚いたのはとってもダイエットに
ストイックな部分です
私もぜひ、見習いたいです（笑）
 
また遊びにぜひ来てください！
 
 
 
 
 
 
最後にお知らせなのですが
明日のおに魂は欠席させて頂きます、
せっかくの感謝祭ですが
本当にごめんなさい(TT)
 
そのかわりといってはあれですが
木曜日にお邪魔させて頂きますので
ぜひ聴いてください
サリオちゃんとペイジさんと
楽しい放送をお届けします♡♡♡
 
 
 
 
 
 
 
 
 
それでは明日のライブに向けて
皆さんも沢山睡眠とってきて
いっぱいいっぱい騒ぎましょうね♡
ライブ大好き\( ¨̮ )/♡
 
 
 
 
 
 
 
 
 
 
 
 
たんたんたん ゆったんたん( •ㅈ• )
