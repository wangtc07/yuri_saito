こんばんたーん( •ㅈ• )
 
 
 
 
 
 
 
まだまだ「じゃらん」のオフショットが
あったので載せちゃいたいと思います！
これはねスキューバーダイビングの時の（笑）
猫耳付けたままゴーグル？付けるの
めっちゃ大変だったよ(TT) 
 
 
 
 
 
 
昨日のテレ東音楽祭のオフショットで
隣で髪の毛をセットしてもらってる途中の
生ちゃんが面白かったので.....（笑）
 
 
 
 
しかも可愛すぎるだろう♡♡♡♡
なんなんだなんなんだ（笑）
生ちゃんスカイダイビング踊るのは
昨日がはじめてだったので緊張してたの
 
いざ！って時に「あ〜もうダメだ」
とか言ってたけどそれもまた愛嬌だな♡
 
 
 
 
 
ピースで星を作ろうとしてた所に
参加したんだけどせっかく上手く
作れたのにカメラの向きで残念ながら
見えませんでした(TT)
 
 
 
 
 
皆で爆笑よ〜結構苦労して作ったのにな（笑）
 
 
 
 
 
 
そんな楽しい日々を毎日
過ごしています(ˊᵕˋ)
今度は皆でアイス巡りしに行くんだ！
でも私1番食べたいのは火鍋（笑）
 
 
 
 
 
 
 
*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡*⃝̣◌⑅⃝◍♡◌*⃝̥◍♡
 
 
 
 
 
そしてこの前のラジオには「ライス」さん
が遊びに来て下さいました☆☆☆
 
 
 
 
 
お会い出来てとっても嬉しかったですし
ネタを目の前で見せてもらえる嬉しさ
といったら半端ないです(TT)
 
 
月曜のレギュラーは奪いに
来ちゃだめですがいつでも遊びに来て下さい！
 
 
 
 
 
 
 
 
 
 
それでは、もう間もなく神宮！
体力、グッズの準備はどうですか？
今回の真夏の全国ツアーの写真
そこまでブサイクではなかったよ（笑）
 
良かったらチェックしてみてね？

こんばんたーん( •ㅈ• )

出席してきました♡♡♡♡♡♡♡♡

したんですよ( ´･ᴗ･` )

記者会見で見たのですがとっても可愛い

全握がラストでした！！！！！！

だったのでとっても嬉しかったし
なんならめちゃめちゃはじけた（笑）

てぃーちゃんだよやばいだろう♡♡♡
 
 
はじけたっていう言い方はあれだけど
凄いテンション上がっちゃて
沢山の方に喋りまくっちゃいました！！
話したい会話が出来なかった方
もし居たらごめんね（笑）
 
 
 
 
それでは、またてぃーちゃんとなれること
楽しみにしてます( ´›ω‹｀)
ひなめろの皆さんもわちゃわちゃ
してくれて本当にありがとう
 
 
 
 
 
 
 
そしてそんな最後のお話は「テレ東音楽祭」に
インフルエンサーとスカイダイビングで
出演してきました〜！！！！！
 
インフルエンサーはヒム子さんと一緒に♡
ヒム子さんはTVで見てたことが
あったのでまさかのコラボが
出来るとは思ってなかったですし
凄い振りをしっかり覚えて下さって
本当に忙しい中ありがとうございました！！！
 
 
 
 
私達も覚えるの大変だったから
絶対やばいと思うの(TT)
筋肉痛になってないかな？
 
 
 
 
 
ヒム子さんのポーズ、ななとまいやんが
